# 活动效果评级规则 v10

## 核心指标

| 维度 | 计算方式 | 权重 | 说明 |
|------|----------|------|------|
| **变现力** | ARPU × 付费率 / 10 | 50% | 综合衡量单价和转化，避免只看ARPU忽视高转化活动 |
| **转化力** | 付费率 | 25% | 付费参与广度 |
| 鲸鱼依赖度 | 超R收入占比 | 15% | 50-70%最健康，过高风险大 |
| 分层清晰度 | 超R付费率/小R付费率 | 10% | R级梯度是否合理 |

## 评分标准

### 变现力（有效变现指数）
| 指数 | 得分 | 示例 |
|------|------|------|
| ≥15 | 100 | 异族大富翁(24.1) |
| 10-15 | 85 | 挖孔小游戏(13.1)、弹珠抽奖(14.9) |
| 6-10 | 70 | 砍价(6.2)、钓鱼(6.7) |
| 3-6 | 55 | MJ8(4.5) |
| 1-3 | 40 | 多条件连锁(2.5) |
| <1 | 20 | - |

### 转化力（付费率%）
| 付费率 | 得分 |
|--------|------|
| ≥15% | 100 |
| 10-15% | 80 |
| 7-10% | 60 |
| 4-7% | 40 |
| <4% | 20 |

### 鲸鱼依赖度（超R收入占比%）
| 占比 | 得分 | 说明 |
|------|------|------|
| 50-70% | 100 | 最健康，全R级变现 |
| 70-80% | 80 | 略偏超R |
| 40-50% | 60 | 偏小R |
| 80-90% | 40 | 高度依赖超R |
| 其他 | 20 | 极端情况 |

## 等级划分

| 等级 | 综合得分 | 含义 |
|------|----------|------|
| **S** | ≥85 | 标杆活动 |
| **A** | 70-84 | 优秀 |
| **B** | 55-69 | 合格 |
| **C** | 40-54 | 待优化 |
| **D** | <40 | 较差 |

## 同名活动合并规则

- 自动识别多期活动（如"挖孔小游戏-第1期"、"第2期"）
- 合并后：总付费额累加，ARPU/付费率取平均
- Excel中保留`上线次数`和`原始活动`字段可追溯

## 四象限定位

```
        高鲸鱼依赖(>70%)
             │
  超R专属型   │   超R主导型
  (低ARPU)   │   (高ARPU)
─────────────┼─────────────→ 高ARPU(>10)
  全R普惠型   │   全R理想型
  (低ARPU)   │   (高ARPU) ★最佳
             │
        低鲸鱼依赖(<70%)
```
