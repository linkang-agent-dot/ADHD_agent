## 〇、评分规则

| 维度 | 计算方式 | 权重 | 说明 |
|-----|---------|------|------|
| 变现力 | ARPU × 付费率 / 10 | 50% | 综合衡量单价和转化，避免只看ARPU忽视高转化活动 |
| 转化力 | 付费率 | 25% | 付费参与广度 |
| 鲸鱼依赖度 | 超R收入占比 | 15% | 50-70%最健康，过高风险大 |
| 分层清晰度 | 超R付费率 / 小R付费率 | 10% | R级梯度是否合理 |

## 二、活动机制模板

### 2.1 BP体系

BP体系分为三类：节日BP、雷达BP、签到BP

---

#### 2.1.1 节日BP

| 字段 | 数值 |
|-----|------|
| 活动类型 | 活跃+消耗 |
| 核心人群 | 小中R |
| 持续时长 | 120小时 |
| 排行榜 | 个人全服排行 |
| 付费档位 | FREE / Purchase1($3.99) / Purchase2($9.99) |
| BP暴击 | 1倍(60%) / 2倍(40%) / 3倍(15%) / 5倍(5%) |
| 升满需要 | 约411个BP道具 |
| 核心投放 | gacha道具、装饰、加速、资源、探测 |
| 排行榜奖励 | 行军皮肤、金头 |
| 复用节日 | 14个（春节、情人节、科技节、复活节、拓荒节、深海节、登月、周年庆、音乐节、万圣节、感恩节、圣诞节等） |

**设计目的**：大范围拉小额度付费

**主要行为**：玩家的活跃行为可以转化成付费，并且带多档位，价格递进，层次促进转化

**玩家反馈**：收入很稳定

**评语**：做不了大超R的付费深度，但整体是一个好形式

**数据评分**：通行证 73.2分 (B) / 情人节BP 51分 (C) / 复活节BP 56.5分 (B)

**数据回归**：
- 付费率：情人节7.36% / 圣诞节5.33% / 狂飙拉力赛12.36%
- 付费节点：76%玩家在满级(20/25级)后购买解锁礼包
- 付费时间：集中在活动开启第一天，第二天下跌53%
- 关键瓶颈：界面触达（45%玩家卡在进入活动界面这一步）
- ROI敏感：ROI调低玩家有感知，情人节ROI是狂飙的1/3，付费率也明显下降
- 优化方向：新增HUD/气泡提醒、礼包界面强化对比、50级后增加循环奖励

---

#### 2.1.2 雷达BP

| 字段 | 数值 |
|-----|------|
| 活动类型 | 付费活动 |
| 核心人群 | 全体 |
| 持续时长 | 168小时（7日循环） |
| 付费档位 | FREE / Purchase($9.99) |
| 核心机制 | 每日完成10个雷达任务升级 |
| 购买后福利 | 每日额外10次雷达任务机会 |
| 核心投放 | 雷达次数、养成线奖励 |
| ROI | 614.47% |

**设计目的**：跟单个任务模块绑定，以该模块的主要游玩人群为目标对象

**玩家反馈**：收入比较稳定，但受众池子比节日大BP小

**评语**：两个BP之间会有一定体验重复疲劳，不能堆积太多无特殊玩法的单模块BP形式

## 三、特色活动机制

### 3.1 AI讲价小游戏

**活动类型**：活跃活动  
**核心人群**：全体  
**持续时长**：72小时

**玩法**：玩家选择想要的物品，进行砍价，砍价成功后可低价购买

**数值思路**：砍价最低价根据神秘商店&CD消耗活动的最低价来定

**商品配置示例**：

| 道具 | 原价 | 最低价 | 折扣率 |

| 万能英雄碎片-橙色 | 5,000 | 1,500 | 30% |
| 英雄升星-橙色-小 | 750 | 200 | 27% |
| 12小时加速 | 1,100 | 330 | 30% |

### 3.2 骰子红包

**活动类型**：付费活动  
**核心人群**：超R

**定位**：付费玩家购买礼包的附赠品，红包占礼包部分ROI

**设计目的**：
- 体现发红包抢红包的氛围
- 给发红包的玩家更好的社交体验

**数值设计**：
- 单个红包可领20次
- 红包骰子ROI：0.4
- 随机规则：双倍随机法

### 7.1 节日BP

**数据结论**：
- 付费率趋势：登月节4.84% → 万圣节4.66% → 感恩节5.87% → 圣诞节6.80% → 春节8.28%，持续上升

- 付费玩家占比：春节18.21%（占活跃付费玩家），为历史最高
- 关键发现：累充+排行榜分组对付费深度有提升；免产增加对付费率有正向效果
- 付费节点：76%玩家在满级后购买解锁礼包；付费集中在活动首日，次日下跌53%
- 瓶颈：界面触达率仅45%，需优化HUD/气泡提醒

**评语**：稳定的小额付费收割器，付费率和深度持续优化中，但超R付费深度有限

### 7.6 七日活动

**数据结论**：
- 付费率下滑：登月节7.58% → 圣诞节4.64% → 情人节4.76%，整体呈下降趋势
- 任务完成率：完成所有任务仅6-11%，投放gacha券可提升至19%
- R级付费率：超R 42-63%、大R 31-47%、中R 11-21%、小R 0.5-1.5%
- 关键发现：付费情况与完成率关联不大，问题在付费体验而非活跃度
- 优化方向：开箱ROI分配调整，增加保底+小额抽奖形式，排除垃圾奖品

**评语**：活跃拉动效果好，但付费转化持续走低，需从付费体验角度优化
