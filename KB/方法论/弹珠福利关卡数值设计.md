# 弹珠福利关卡数值设计

## 零、设计目的

### 核心问题
弹珠活动上线3期后，**数据持续下跌，超R付费流失严重**

### 设计目标
| 目标 | 说明 |
|-----|------|
| 挽回超R | 给超R玩家提供短期内的明确目标，提升付费体验 |
| 新增玩法层 | 参考挖孔活动，加入福利关卡机制，丰富游玩体验 |
| 提升付费数据 | 通过高价值返场道具和养成道具，拉动弹珠活动整体付费 |

### 设计约束
- **不能砍普通关产出**：线上已跑多期，玩家有预期
- **总ROI控制在2.1以内**：不能影响其他活动形式
- **不能改概率配置**：技术和运营限制

## 一、玩家循环
```
普通弹珠50个 → 触发福利关卡 → 获得50福利弹珠 → 打出高级兑换券 → 高级商店兑换
```

### 各模块设计目的
| 模块 | 设计目的 |
|-----|---------|
| 普通弹珠50个 | 触发门槛，$30一次的成本让超R有明确的"再打一轮就能进"的目标感，中低频率保持稀缺性 |
| 触发福利关卡 | 阶段奖励逻辑，累计触发而非随机，给玩家确定性预期 |
| 获得50福利弹珠 | 仪式感设计，专属弹珠+视觉效果，让玩家感知"我在打一个特别的关卡" |
| 打出高级兑换券 | 价值载体，通过小件兑换（100券=10弹珠）建立价值锚点，玩家自然感知券的价值 |
| 高级商店兑换 | 核心奖励出口，三条路径满足不同需求：皮肤（情感消费）、养成道具（刚需）、普通币（兜底） |
| 里程碑宝箱 | 打破匀速攒券的无聊感，每个节点都是情绪峰值点，增加惊喜和期待 |

## 基础数值
| 参数 | 数值 |
|-----|------|
| 弹珠单价 | $0.6 |
| 进洞成本 | 50普通弹珠（约$30） |
| 福利弹珠数量 | 50个/次 |
| 高级券产出 | ~450券/次（期望） |
| 高级券单位价值 | $0.018/券 |
| 福利关ROI增量 | ~27% |
| 总ROI | ~2.07（普通关1.8 + 福利关0.27） |

## 高级商店设计

### 皮肤类（不计入ROI，独立价值）
| 档位 | 单价 | 道具本身价值 |
|-----|------|-------------|
| S级 | 15,000券 | $1,500 |
| A级 | 5,000券 | $500 |
| B级 | 2,000券 | $200 |

### 养成道具类（计入ROI）
| 道具 | 单价 | 价值 |
|-----|------|------|
| 红色英雄碎片 | 56券/个 | $1/个 |
| 红色英雄技能材料 | 278券/个 | $5/个 |

### 兜底兑换
- 普通代币兑换：1高级券 → 5普通券
- 阶段奖励自选箱：1,000券/个，限5个
- 小件兑换：100券 = 10普通弹珠（价值锚点）

## 返场规则
- 高级商店只返场1年以前的排行榜奖励
- 保留排行榜皮肤1年的独占窗口期
- 皮肤定价按道具本身价值，不计入ROI

## 设计要点总结

| 聚焦点 | 问题 | 设计方案 | 如何解决 |
|-------|------|---------|---------|
| 福利关卡定位 | 是额外福利还是价值转移？ | 额外福利，作为阶段奖励逻辑，不砍普通关产出 | 福利关本质是阶段奖励，玩家打普通关累计触发，不需要从普通关砍产出来平衡，避免老玩家感知到"变差了" |
| ROI控制 | 加了福利关后总ROI会超标 | 福利关ROI增量控制在~27%，总ROI约2.07 | 通过压缩高级券单位价值（$0.018/券），让福利关产出的道具价值可控，不影响整体经济体系 |
| 高级商店定价矛盾 | 皮肤价值$1500，但按ROI算玩家要花$5000才能换到 | 拆成两类：皮肤不计ROI（独立价值），养成道具计ROI | 皮肤是稀缺品/情感消费，玩家愿意额外付出；养成道具是刚需，必须保证ROI合理。两套逻辑独立，互不干扰 |
| 玩家价值感知 | 玩家算不清券值多少钱 | 用小件兑换（100券=10弹珠）建立价值锚点，让玩家自己悟 | 不直接告诉玩家"1券=$0.018"，而是通过小件兑换让玩家在游玩中自然建立价值感知，体验更顺滑 |
| 返场道具定价 | 排行榜皮肤竞争成本差异大（$2000 vs $5000） | 按道具本身品质定价，不按历史竞争成本 | 排行榜的溢价是"打败别人"的成本，不是道具本身价值。返场的是道具，不是竞争体验，所以按品质定价 |
| 体验节奏 | 攒券过程太平，缺少阶段性反馈 | 里程碑宝箱（6个节点），随机开出保底/中奖/大奖/终极 | 把里程碑变成"开箱时刻"，每个节点都是情绪峰值点，打破匀速攒券的无聊感，增加惊喜和期待 |
| 高级券消耗出口 | 不能只有皮肤一个出口 | 三条路：皮肤（额外福利）+ 养成道具（红色英雄）+ 兜底兑换（普通币） | 不同玩家有不同需求：追皮肤的换皮肤，追养成的换红色英雄，都不想要的兑普通币保底，券不会溢出浪费 |
