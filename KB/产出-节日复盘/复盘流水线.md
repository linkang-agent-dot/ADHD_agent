# 节日整体 Review 一键生成流水线

> 源文件：`_festival_review_pipeline.md`（最后更新 2026-04-07，科技节）

## 输入参数
- 日期范围（节日起止）
- `control_id`（活动组 ID）

## 流程链
```
1. Trino 图表 5 脚本 → 原始付费数据
2. 数据聚合 → _tech_fest_input.json
3. 字段映射 → 报告模板填充
4. 图表生成 → 3 张标准 PNG
5. Markdown/HTML 报告组装
6. 推送 Notion（可选）
```

## 标准图表
1. 营收趋势折线图
2. 模块营收结构图
3. 用户分层/增长图

## 关联的中间产物
- `_tmp_chart5_result.json` / `_tmp_all_charts_result.json` — 图表数据
- `_tmp_trend_section*.html` — 趋势片段
- `_tmp_rank_top20.html` / `_tmp_rank_rest.html` — 排名表
- `_tmp_subact_table_v*.html` — 子活动大表
- `_tmp_skin_slot_section.html` — 皮肤槽位对比

## 复用说明
改日期 + control_id 即可套用到下一个节日。
