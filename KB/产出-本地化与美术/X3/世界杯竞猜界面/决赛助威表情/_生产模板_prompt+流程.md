# 决赛助威动态表情 · 生产模板

决赛专属、**真动效**助威表情，决赛 2 队各 1 个。区别于小组赛 48 队静态版。
**动作定稿**：双拳振臂过顶 + 仰头大喊「GO!」（决赛级助威，不夹气、不庆祝/不举杯——还没夺冠）。
**角色**：Q版爱莉希雅啦啦队（沿用小组赛 FINAL 形象）。

## ★后续流程（决赛队伍定了照此跑，三步）

> 用户定的顺序：**先出球衣图 → 再出视频/逐帧 → 再切**。

### S1. 先出该队「球衣定妆图」(静态, 1张)
- 用小组赛同款脸彩+队色啦啦队链路（`_gen_*` / x3-media march_emoji），出决赛队的**静态定妆图**：队服+花球换该队队色、脸颊画该队国旗 face paint。
- 这张 = 后续动画的**角色锚**（锁定脸型/队色/脸彩），等同小组赛 `WC_FaceEmote_{码}.png`。
- 球衣配色查 `48队球衣参考表.md`。

### S2. 用定妆图出动态(4×4逐帧 sprite sheet)
- x3-media `dynamic_march_emoji`，reference_images=S1定妆图，prompt 用下方【PROMPT】（把"中性白底留白"换成"保持参考图的队色+脸彩不变"）。
- 输出 4×4=16格、每格256、透明1024、首尾可循环。

### S3. 切帧 → 透明循环 GIF → .bytes 进游戏
- 切帧脚本逻辑（本次已验，在 `决赛助威表情\` 对话留存）：
  ```
  sheet=Image.open(4x4).convert("RGBA"); cw,ch=W//4,H//4
  逐格crop→thumbnail(256)→16帧
  透明GIF: alpha阈值化(p>=128→255)二值, convert("P",colors=255), paste(255, 反mask), 
           save(save_all,duration~90,loop=0,transparency=255,disposal=2)
  → 同内容复制为 .bytes
  ```
- 落地同小组赛：Emoticons表(新ID 接348+) Res字段指 `DK_WC_Final_{码}`→`.bytes`动图 + icon + Item表情包道具 + DK注册 + i18n。

## 坑/注意
