一个界面 = prefab（积木拼装图，你拼） + 代码（已写完）。代码运行时按字段名找零件。

唯一翻车点：prefab 绑定的字段名跟代码对不上 → 找不到 → 编译报错/空指针。本指南每个零件都标了「绑定字段名」，一字不差。
保命心法：不从零新建，全程「复制现成的 → 改名 → 换图」。

② 渐变表现（X3 文字三件套）

X3 文字 = TFWText + BetterOutline(描边) + Gradient2(金渐变)，全游戏统一风格，默认整套保留、不删组件。

  • 深色底上的字（如弹窗框顶部 title/副标）→ 保模板金渐变，啥都不动。
  • ★米色/羊皮纸档位面板上的字（本弹窗两栏的 txt_name/txt_desc/txt_state 大多在米色 nourish_3 底上，金字会发糊）→ 把 Gradient2 → Effect Gradient 两端改深棕：
    左端(0%) 7A4F1D ／ 右端(100%) 5A3A12；描边(BetterOutline)保留。
  • 改法：选中文字节点 → Inspector 的 Gradient2 → Effect Gradient 色带两端色标填上面 Hex。

③ Best Fit / 描边 注意

  • X3 文字基本都勾 Best Fit（多语言自适应保护，别取消）→ 勾着时 Font Size 失效，字号填到 Max Size + 文本框 Width/Height 拉够装；Min Size 统一改 20。
  • 加粗：Font Style → Bold。
  • 描边：深色底白/金字保留描边；米色底改深棕渐变后描边也保留（别删组件，改 Gradient2 色值即可）。
心法（世界杯冻结版）：界面风格锚是游戏模板不是效果图 —— 保模板金字，不删组件、不改平色；唯一动作 = 米色面板上的字把 Gradient2 改深棕。其余照旧。

⚠ 翻车点（照搬世界杯踩过的）

1. 绑定字段名对不上：本指南第 6 步的字段名是和代码硬约定的，错一个字编译就挂。

2. Ctrl+D 复制带 (1) 后缀：复制 ColLuxury 后查内部 RewardList/UIBtnPurchase/txt_state 没被加 (1)。

3. 组件类型：填字要 TFWText、填图要 TFWImage → 全程复制现成节点不新建。

4. 加图节点别复制 Root/Bg：会带全屏缩放脚本(Rect Transform Scaler)，Scale 锁死/图全消失。复制干净 Image 节点当源。

5. Play 模式(▶)期间改 prefab = 退出全蒸发：动手前瞄一眼播放键。
