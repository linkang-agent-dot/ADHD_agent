# X3 通用累充图标（FES）美需

> 类型：UI 图标 · 静态资产
> 项目：X3
> 适用：全节日通用 · 累充类礼包入口

---

## 一、需求背景

X3 现有通用累充图标 `icon_global_integral.png`（金菱形 + 金色礼物盒）作为单一款式使用，无法在不同入口/不同档位中做视觉区分。本次新增一款**圆形红金徽章**作为平行款，与原图标并存：

- 原 `icon_global_integral.png`：金菱形，偏中性
- 新增 `icon_global_integral_round_FES.png`：圆形红金徽章 + FES 字样，识别度更高，强调"节日累充"语义

---

## 二、设计定位

| 维度 | 说明 |
|------|------|
| 心智 | 勋章感 / 奖章感，匹配"累充奖励" |
| 通用性 | 不绑任何节日符号，全年节日可复用 |
| 与原图关系 | 主体礼物盒造型/质感保持一致；外形/底色/标签差异化区分 |

---

## 三、视觉规格

| 项 | 描述 |
|---|---|
| **整体形状** | 圆形徽章，含厚重外描边（暗棕黑色描边围绕金环） |
| **外环** | 金色粗描边环，金属高光（顶部偏亮、底部偏暗，立体感） |
| **内圈** | 单层细金线，距外环略内缩，作层次分隔 |
| **底色** | 深红径向渐变（中心暖红 → 边缘暗红，营造内凹立体感） |
| **主体** | 金色立体礼物盒 + 金色蝴蝶结，沿用原 `icon_global_integral.png` 礼盒造型与质感 |
| **主体占比** | 礼包外接矩形约占内圈直径 60–65%，居中略偏上 |
| **FES 字母** | 大写、粗体、金属金色，立体凸起，带深色描边；置于礼盒正下方，悬挂在金环下沿（局部叠在金环上，呈"挂牌"效果） |
| **质感统一** | 外环金 / 礼包金 / FES 金 同色系（亮金 → 暗金渐变），保证视觉同源 |
| **整体风格** | 扁平卡通 3D · 厚重外描边 · 柔和高光（与原 integral 图标风格一致） |
