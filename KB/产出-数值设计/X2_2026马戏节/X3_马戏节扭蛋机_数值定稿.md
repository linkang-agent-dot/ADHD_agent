## §3 EV/ROI 推导

### 3.1 单抽成本
付费单抽 = 1208 抽奖券×1（设计锚 $0.25=125钻）+ 1207 扭蛋币×1（积分任务免费产出，计 0）→ **实付 125钻/抽**。

### 3.2 整活动模型（playbook 抽奖类口径）
锚玩家 = 拿满 14 档积分任务 **430 扭蛋币**，全部作付费抽陪扣（理性策略：付费池 EV≫免费池），即 **锚抽数=430**（阶段1 前400 + 阶段2 后30）：

```
全程实付 = 430 × 125 = 53,750钻（≈$107.5 的券）
全程产出 = 400×977.3 + 30×1050.7 = 422,432钻
整活动 ROI = 422,432 / 53,750 = 7.86x   （锚 8x，偏差 −1.8%）
```

积分任务 430 币的免费产出已内含在模型里（430 抽的币腿全部计 0 成本）。由于成本与 EV 均按抽线性，**该 ROI 对锚抽数不敏感**：若币另有产出渠道使付费抽数超 430，ROI 逐步趋近阶段2 的 8.41x，仍在 8x 邻域。

锚玩家期望产出结构（430抽）：罗盘 ≈52个（1.3万钻）、传奇技能书 ≈37本（3.7万）、万能史诗信物 ≈61个（6.1万）、万能传奇信物 ≈12个（5.9万）、强化石 ≈306个（3.1万）、冒险阅历 ≈370w点（11.1万）、驯养经验 ≈370w点（11.1万）。

### 3.3 与 8x 锚的偏差处理
7.86x 为**刻意压在锚下方**：排行奖（Top100 全服面：加速 321×2400=77.0万钻，含技能书 237×1000 后全榜内部口径≈100.7万钻/服，验收复算补全）与免费池产出（F2P 路径 430 免费抽）均未计入付费产出，留作上浮垫；抽奖类"越抽越赚"病灶已控（阶段2/阶段1 EV 比 = 1.08x，远低于深海格子升级 3-6x 的放大器）。

## §8 假设与告警清单

1. **单抽成本模型**：付费池 action_use=[1207,1208] 按"两者同扣"理解（任务口径）。若程序实现为"任一可用"（OR），付费抽数不再受 430 币上限——但成本与 EV 均按抽线性，**per-抽 ROI 结论不变**，仅锚抽数叙述需改（趋近阶段2 的 8.41x）。落表前找程序确认扣费语义。
2. **加速价值口径**：30m=400钻/3h=2400钻 为内部线性口径（13.33钻/min），公认偏高。付费池不含加速槽，**8x 结论不受此污染**；免费池 EV(444/434钻) 与排行档价值仅作相对参考，不可当真实成本汇报。
