# 教学包 · 英雄养成手册出图流程

> 给同事讲「X3 活动界面从想法到落地，怎么产出可用 UI 图/组件」的全套材料。
> 案例 = 英雄养成手册双版本（ActvType=27 加 $49.99 豪华档·二选一互斥）。

## 讲课顺序（照编号走）

### 1. 先讲流程 —— `00_先讲_流程总览+产出分类.html`
5 阶段主线（出方案 → 出界面 → 定交互 → 写代码 → 翻译/QA）+ Leader 中途设计转向 + 全部产出文件分类（定稿/中间稿/废弃）。让同事先有全局。

### 2. 重点讲出图 —— `01_重点_可用UI图与组件_出图方法.html`
**本次教学的核心**。最大卡点 = 怎么搞出能落地的 UI 图/组件。
- 开篇「反面教材」：先看没走流程、纯 AI 直出的垃圾图（为什么会烂）
- 可用组件 = 去 prefab 抠真 sprite（GUID 反查），别画
- 可用 UI 图 = 真组件拼装锁布局 → AI reskin 换皮（五步法）
- ★血泪教训：拼装糙没关系，第 4 步全 ERASE 只取布局，别死磕拼装精度
