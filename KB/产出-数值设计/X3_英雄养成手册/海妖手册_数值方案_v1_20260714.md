## 3. 口径A（设计侧，推荐）

### 3.1 先证明：纯XP方案不可行（这是秘币入池的根本理由）

手册若只发XP，两个拍板要求互斥：
- 要求①（用户已拍板）：豪华ROI = 周卡实发 → 豪华XP = 11.15×49.99/0.00025 = 2,229,554 XP → XP性价比 = 2,229,554/49.99 = **44,600 XP/$ = 周卡的111%** → 周卡"全游戏第一经验性价比"（V2优化案的立案目标）被自家新品击穿。
- 要求②（周卡V2定位）：手册XP/$ < 40,040 → 豪华XP < 2,001,600 → 豪华ROI < 2,001,600×0.00025/49.99 = **10.01x**，比锚11.15x低10%，"与周卡一致"不成立。

两个纯XP参考版（供对照，不推荐）：

| 版本 | 低价日XP | 高价日XP | 基础30天XP | 基础ROI | 豪华30天XP | 豪华ROI | 豪华XP/$ | 问题 |
|---|---|---|---|---|---|---|---|---|

**现状**（全部已验证事实）：Item_1143，官方价1,000钻/枚；**100%付费获取、无免费渠道**；唯一入口=2014001-5直购弹窗（$4.99~99.99：秘币3/5/10/25/50+等面额钻+VIP点）；近30天全服仅106人获得1,148枚。用途=海妖养成的稀缺付费材料。

**入池战略意义**：
1. **首个长周期获取入口**：现在秘币只有"即时弹窗"一种脉冲式供给，手册使其变成30天日发的订阅式供给——这是养成线材料体系从"买断"到"月供"的层级补全（对标周卡之于XP）。
2. **手册的差异化身份**：§3.1已证纯XP手册无处安放；秘币让手册="秘币唯一月供入口+XP大总量"，与周卡"XP性价比之王"彻底分工，互不替代 → 支撑双买。
3. **价格锚自洽**：官方价1,000钻/枚与弹窗隐含边际价一致（$49.99=25枚→$2/枚），入池计价无编造。

**风险（都要监控/拍板）**：
1. **供给通胀**：豪华48枚/人/月。参照现英雄手册516买家/60天的体量，若海妖手册月售100~300豪华 → 月增4,800~14,400枚 = 现全服供给（1,148枚/30天）的**4~13倍**。秘币"稀缺付费材料"的稀缺性会被重新定价；秘币的消耗端容量（sink够不够深）本方案无数据，**上线前需数值侧补验**。
2. **蚕食2014弹窗**：手册豪华48枚/月接近弹窗$99.99档（50枚）的量，且还附带185万XP。缓解因素=弹窗是即时到手（鲸鱼推进度时的补给速度溢价），手册是30天慢发；若仍担心，切§3.2保守版（豪华24枚/月=弹窗$49.99档一单量）。
3. 秘币在总价值中占比：基础24,000/139,500 = 17.2%（豪华同）。占比不高，即便秘币实际感知价值低于官方价，ROI失真上限约±2x，不致方案翻车。

| 非XP | 钻5,700/周（月24,429钻） | 秘币24（唯一月供入口） | 秘币48 |
| 承诺结构 | 每周决策、7天日领 | 单次决策、30天日领 | 同左 |

**会不会杀死周卡V2的定位？——不会，且是刻意设计的结果**：手册XP/$全档位压在周卡之下（92.3%/77%），周卡"全游戏第一经验性价比"无损；手册赢在总量（豪华月XP≈周卡月供108%）、买断便利、秘币独占。两者是"性价比 vs 总量+稀缺材料"的分工，不是同位替代。
**协同建议**：
1. 面向重度玩家推"双买"组合：周卡+豪华手册 = $92.80/月 → 356万XP + 秘币48 + 钻24,429，覆盖重度月耗且拿满秘币——商城陈列/推荐位可按组合叙事。
2. 手册上线时间应**晚于或同步**周卡V2上线：手册锚定的是V2（11.15x）；若V2延期而手册先上，手册豪华11.16x对着线上V1（整体口径 (320,000×0.125+5,700)/4,995 = 9.15x）就高出22%，反向击穿。此依赖关系列待拍板§8-5。
3. 双开玩家XP溢出问题见§7，靠84%消耗集中在重度玩家 + 碎片选项泄压兜底。

## 7. sink校验：会不会普遍溢出

供需对照（XP/月，消耗数据=成熟服近30天，2,664名消耗者，总耗67.5亿，消化率84%）：

| 情形 | 月供给 | vs 人均月耗253万 |
|---|---|---|
| 免费玩家（情报任务82%+机甲转盘10%等） | 4.7万/日×30 = 141万 | 56% |
| 免费+手册基础 | 141+92.4 = 233.4万 | 92% ✓ 无溢出 |
| 免费+手册豪华 | 141+184.8 = 325.8万 | 129%，超28.8% |
| 免费+豪华+周卡V2 | 325.8+171.4 = 497.2万 | 196%，明显溢出 |

**判断：不会普遍溢出，但豪华+周卡双开的中度玩家会阶段性溢出**：
- 253万是**均值**；84%的消耗量发生在20级后，手册买家画像=重度海妖付费者，其个体月耗应显著高于均值 → 豪华单开（129%）对目标客群可消化。
- 坑深侧：母表口径养成坑深1025（碎片口径）、转盘养满6.7周、P2超R对照8周拉满——单海妖坑有限但海妖有三只（提亚马特/阿斯忒里亚/海瑶），XP总坑口径母表未给绝对值，**双开客群的真实坑深余量需上线前用等级分布数据补验**（待拍板§8-5附带项）。
- 泄压阀：①高价日二选一的碎片选项（XP毕业玩家改拿碎片）；②手册30天日发天然平滑，不产生脉冲式浪费；③等级区间14,99挡住未解锁玩家误购。
