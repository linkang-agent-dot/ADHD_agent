## 一、口径讲解（从头讲清：道具怎么估值 → ROI 是什么 / 怎么不投爆 → 投放怎么定）

> 你对数值不熟没关系，这一段用本活动的具体例子把三件事讲透。看完你能自己核验下面每张表。

### 1.1 一个道具值多少钱 —— 全部换算成「钻石」

游戏里发的每样东西（券、加速、碎片、皮肤…）都能折算成一个钱数。X3 的尺子是官方充值比例：

```
1 美元(USD) = 500 钻石          ← 一切的尺子
1 钻石 = $0.002
```

`道具表` 里每个道具的价值就是用**钻石**标的（钻石自己=1）。本活动会用到的几个锚（都是从 `道具表` 现查的真值）：

| 道具 | X3 钻石价值 | 折算美元 | 出处 |
|---|---|---|---|
| 抽奖券（通用锚） | 125 钻/张 | $0.25/张 | foundation 统一锚（夏日/尼罗/世界杯一致）|
| 1 小时通用加速 | 800 钻 | $1.6 | 道具表 11004 |
| 3 小时通用加速 | 2,400 钻 | $4.8 | 道具表 11005 |
| 8 小时通用加速 | 6,400 钻 | $12.8 | 道具表 11006 |
| 英雄美酒 | 3,000 钻 | $6.0 | 道具表 7001 |
| 幸运宝箱 | 8,000 钻 | $16.0 | 道具表 70003 |
| 豪华宝箱 | 18,000 钻 | $36.0 | 道具表 70004 |
| 金币 | 0.04 钻 | ≈0 | 道具表 1001（当填充物）|

> ⚠️ **`道具表` 大片是空的**——英雄碎片、成长自选宝箱、军备图纸、升星/天赋材料这些**养成道具，X3 大多没标钻石价值**。查不到的**绝不自己猜一个填进去**（这是准确率从 90% 掉到 50% 的头号原因）。查不到时的处理见 §1.4「相对守恒法」和 §6「待要价」。

**举个具体例子**：钻石线档 2 想装「3 小时通用加速 × 2」。它值多少？= 2,400 钻 × 2 = **4,800 钻 = $9.6**。这就是把道具翻译成钱。

### 1.2 ROI = 「玩家拿到的价值 ÷ 玩家花的钱」

```
ROI = 礼包总价值 ÷ 售价
```

例：直购线档 1 卖 $9.99，玩家拿到值 $69.93（=34,965 钻）的东西 → ROI = 69.93 ÷ 9.99 ≈ **7.0**。
意思是：玩家花 1 块，拿回 7 块的东西。

### 1.3 为什么不能「多给点」—— 不投爆 ROI 是铁律

直觉说「送多点玩家开心」。错。三个原因：
1. **直接亏钱**：ROI 越高，每卖一份送得越多。
2. **破坏付费结构**：如果便宜档 ROI 特别高、贵档反而低，所有人只买便宜档，贵档没人要，总收入塌。所以**同一条线里，每档 ROI 必须一致**（$9.99 给 X，$19.99 就给 2X，线性不破）。
3. **养坏预期**：这期投爆，下期回正常玩家觉得「变抠了」。

**ROI 没有「全局标准值」，只有「同类基准」**：节日主力链式礼包 ≈2.25x、周卡 ≈2.75x、晋升皮肤 17-19x、抽奖活动 ≈8x。判断投没投爆的唯一靠谱办法 = **跟同类比**。

**本活动为什么钻石线 2 / 直购线 7 是合理的**（用户已拍定的硬约束，这里解释「为什么守得住」）：
- **钻石线 ROI 2 = 引流**：花的是玩家**攒的钻石**（不是新掏钱），2 倍价值把玩家「勾进活动」+ 消耗钻石经济，广度优先。ROI 2 比链式 2.25x 还低一点，克制、纯引流，不会冲击真金付费。
- **直购线 ROI 7 = 冲付费**：ROI 7 看着高（介于周卡 2.75x 和皮肤 17x 之间），但它被**限时抢购的两把锁**死死压住——① **全服限量**（抢完即无）② **单人限次**（每期每包只能买 1 次）。玩家没法靠高 ROI 反复薅，稀缺性 = 高 ROI 的合理对价。这也对齐 X2 唱片版 ROI 5.80（X3 略提到 7，因 X3 节日免付比更高、且限量更强）。

### 1.4 投放怎么定 —— 「盘子锁死 + 相对守恒法」

两步：
1. **先锁盘子**：`礼包总价值 = 售价(USD) × 500 × ROI`。这一步纯算术，锁死每个礼包「该装多少钻的东西」。
2. **再填内容**：盘子里投什么道具。
   - 能查到价值的（加速/券/资源/宝箱）→ 直接算，凑到盘子。
   - **查不到价值的养成道具** → 用**相对守恒法**：盘子是固定的，养成道具按「补足盘子剩余钻数」的量投放，**量随价格倍数同比缩放 → ROI 严格不变**。不需要知道每个养成道具值多少钱，就能保证「没投爆、跟目标 ROI 对齐」。绝对数量待策划按 X3 养成线钻石价值要价后填满（护栏①③）。

**本活动的落地策略**（诚实交代能算到什么程度）：
- **钻石线**（盘子小，1,000–20,000 钻）：全部用**已估值道具（加速+资源）**填满 → ROI 可精确核验为 2.00。见 §3.1。
- **直购线**（盘子大，35k–350k 钻，且本质是养成线）：**皮肤宝箱钩子 + 少量已估值打底（券/英雄美酒/宝箱/加速）+ 养成大头（待策划要价）**。盘子锁死、ROI 由构造守住；养成道具具体数量待策划要价校准。见 §3.2。

### 2.2 X2 → X3 重构依据（为什么变成「两线各 4 档」）

X2 的轴是「**R 级（人群深度）**」，9 档同套按钻石/唱片两货币并存。
用户为 X3 重定的轴是「**支付方式**」：钻石线（引流）vs 直购线（冲付费）。轴变了，所以重构档位：

| 重构动作 | 依据 |
|---|---|
| 9 档 → **8 档**（4+4） | 代码方案明确「一期货架 8 个礼包」；砍掉 X2 冗余的第 3 个 $99.99（X2 有 3×$99.99），直购线保留单一 $99.99 顶档 |
| 钻石线承接 X2 的 **RO + R1 低价广度位** | 花攒的钻石、低门槛、引流；对齐 X2 钻石版的低 R 定位 |
| 直购线承接 X2 的 **R1–R4 真金深度位** | IAP 真金、稀缺高价值、冲付费；对齐 X2 唱片版的高 ROI 变现主轴 |
| ROI 3.13/5.80 → **2 / 7** | 用户拍定的硬约束（引流克制 / 冲付费高价值+限量对价），不是我调的 |

## 四、抽奖池期望值（达标免费抽，非付费抽）

> 定位：这是「分享→被助力达标→买过礼包 → **免费抽 1 次**」的破冰奖励，不是付费抽。所以看的是**单抽期望价值**（免费送出去多少），不是 ROI。抽中的礼包免费到手、不占限量/限次（代码方案 §1.4）。
