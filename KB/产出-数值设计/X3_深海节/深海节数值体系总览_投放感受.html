⑤ 投放感受诊断 我标出的可能要调的点 — 你拍板

ROI 口径不统一，跨模块对比会"忽厚忽薄"高
累充 50%(gacha口径) / 成就 800% / 存钱罐 1000% / 每日 1000% / 装饰标称 3200–5000%(其实是固定产出兑换比例标签、非真溢价)。玩家不会区分口径，只会感觉"累充好抠、装饰虚高"。
→ 建议：统一对外口径，或明确分两类——福利型(每日/存钱罐/装饰，高倍直观)vs 变现型(累充/转盘，材料抽奖口径)，装饰别再标3200%误导。
价位带断层：中高端空、顶端三连跳高
SKU 全堆在 $4.99 / $19.99 / $99.99；$29.99 / $59.99 全空；$99.99 → $200 → $2000 跨度过大，$100–$2000 之间只有累充一根线接。中 R 想再加码、够不到大 R 时没有合适台阶。
→ 建议：补 1 个 $29.99 或 $59.99 档（大富翁成就中段 / 一个深海大礼包），让 $100→$2000 之间有过渡。
重复变现集中在大富翁后6档中
单模块 $99.99×6 = $600（跑60/70/80/90/100/200圈，内容完全相同）。ROI 稳是好事，但 6 个一模一样的 $99.99 摆出来，whale 容易疲劳、感知"重复刷钱"。
→ 建议：后段档拉点差异（递增纪念卡/限定外显/递增罗盘），或砍到 4 档，把强度并进更稀缺的高档。
"罗盘券/抽奖券"投放点过多，可能稀释转盘付费中
每日礼包、装饰、周卡、头像框、累充顶档都在发罗盘券/藏宝图。转盘是主变现核之一，免费券给太足会压低券包真金购买欲。
→ 建议：盘一下全节日免费券总投放量 vs 转盘消耗，确认"白嫖能抽多少"，别让转盘付费入口形同虚设。
主体仍 D1 齐开，错峰刚起步（10天排期）中·改善中
已把拜访 + 周卡错峰到后7天(D4–D10)，后段有了补给点 👍。但 D1 仍同时点亮 8 个（大富翁/转盘/累充/BP/许愿池/每日/装饰/头像框）+酒馆 → 开局选择仍偏过载、付费焦点散。
→ 可继续：再错峰 1–2 个（如存钱罐随大富翁但延后触发、某档礼包中段上），或 HUD 做每日推荐位轮换让玩家分天聚焦，别一进来 9 个入口糊脸。
外显钩子分散，缺统一招牌皮观察
核心外显散在 5 处（转盘潜艇皮 / 兑换月心珍珠主城皮$300 / 拜访门头 / 印记头像框 / 成就纪念卡），没有一个"全节日招牌外显"统一引流——D1 全开后这个问题更突出（一上来五个皮各喊各的）。
→ 建议：选 1 个当节日 KV 主皮(潜艇皮或月心珍珠)集中曝光，其余做配套。
大量模块数值仍是草稿/占位前置
累充10档值待复盘、BP进阶轨是占位值、转盘SSR权重/券价未定、装饰是否换3件不同家具未定。投放感受的最终手感，很大程度取决于这几处定稿。
→ 建议：先把累充档位 + BP进阶轨 + 转盘权重/券价这三处定了，再整体回看一遍曲线。
