# 深海转盘「全服(跨服)排行榜」程序需求单

> 提交：林康 ｜ 2026-06-30 ｜ 目标分支：dev_festival
> 一句话：深海罗盘转盘(AO 101025)要加一个**全服(跨服)榜**，前20名发深海猎手永久行军船皮。配置已配好，**程序只需补一行白名单**。

## 一、要改的（就 1 行）

`client/Assets/Scripts/CSShared/Common/Const/ActivityConst.cs` 的 `CenterHostedSingleActivityTypes`：

```csharp
public readonly static HashSet<int> CenterHostedSingleActivityTypes = new HashSet<int>()
{
    TRIGGER_TYPE_COIN_PUSHER,
    TRIGGER_TYPE_LUCKY_WHEEL,   // ← 新增：转盘支持中心托管单服活动 → 可挂跨服榜

```

CSShared 客户端/服务端共用，一处覆盖两端。

## 二、为什么这一行就够（其余代码已写好）

转盘的跨服双榜链路**早已实现**，唯一缺口是转盘没进中心托管白名单 → 跑活动时不会被开成跨服实例 → 跨服主榜(RankType=12)写分无处落。补上白名单后即复用现成链路：

| 环节 | 现状 | 文件 |
|---|---|---|
| 写分(主榜+本服副榜) | ✅已写 `if(cfg.MainRankID>0) UpdateActivityRankScore(...)` + `if(ServerRank>0) ...` | `server/.../ActivityMeta.LuckyWheel.cs:142-150` |
| 建榜/删榜/发奖(含 IsGlobal 跨服分支) | ✅已写 `TRIGGER_TYPE_LUCKY_WHEEL` 分支 | `server/.../ServerActivityRankMeta.cs:47,186,257,325` |
| 跨服榜路由到中心服 | ✅已写 `IsLocalRank` 按 RankType 分流，跨服归 CenterServer | 同上:108 |
| 客户端主榜+本服榜双tab | ✅已写 `mActvRank` + `mActvLocalRank`(同RankID自动去重) | `client/.../UIActvLuckyWheel.cs:160-170` |
| 中心托管单服活动编排 | ✅已写，注释原话"**后续同类需求加白名单即可复用**" | `server/.../ActivityMgr.CenterHostedSingleActv.cs` |

**现成参照**：推币机单服版(`TRIGGER_TYPE_COIN_PUSHER`)是该白名单当前唯一消费者，跨服版靠 `CrossServerRank=true` 归跨服——转盘完全照此模式。

## 三、配置侧（已配好，dev_festival，无需程序动）

- `Rank__RankCfg` 新增 **2001**（RankType=12 跨服 / 归属97 / 人数100 / 上榜需求2500 / 结算邮件1000018）= 全服榜。
- `Rank__RankRewardSlotCfg` 新增 100431-100437（7 档：1/2/3/4-5/6-10/11-20/21-50）。
- `Reward` 新增组 30930-30936（前20名发**深海猎手永久船皮 Item 15065** + 3h三加速 + 前4档铭牌82005）。
- `ActvOnline` **101025**：`CrossServerRank(col19)=1`、`RankCfg(col21)=2001`（主榜=全服榜）。
- `ActvLuckyWheel` CID1025 `ServerRank=2000` 不变（=本服副榜，本服榜大奖已换深海纪念卡180080）。
- 客户端逻辑：主榜2001(跨服)≠本服榜2000 → 自动渲染两个 tab。

## 四、联调要确认的 3 点

1. 中心服能把转盘开成跨服实例（arkServerIds），各服抽奖次数聚合到 2001 这张跨服榜。
2. **深海投放的是 D50+ 老服 → 这些服必须在同一跨服集群里**，否则"全服"无从聚合（榜会是空的或只本服）。这点请程序/QA 确认深海部署服的跨服分组。
3. 活动结束发奖走 CenterServer 的跨服结算（前20名皮肤），本服副榜走本服结算。

## 五、上线顺序（重要）

⚠️ 配置已带 `CrossServerRank=1`。**程序白名单上线前，不要把转盘当跨服部署**——否则跨服主榜写分无处落、全服榜不工作。**配置 + 程序白名单必须同批上**。深海节计划 7/10 全服 D1，留足联调窗口。
