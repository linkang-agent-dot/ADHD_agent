# X3 深海节 · 活动策划案（草稿 v1）

> 基于夏日恋语策划案格式 + 深海节题材改造。核心路线：**提高付费深度** —— 新增第二周核心循环，把单周单循环升级为双周双循环，第二周专承载深度付费点；继续新增可制作的节日类投放付费点（头像框、铭牌）。
> 数据口径：1-88服，拥有率/R级来自数仓（2026-06 快照）。

## 一、付费深度提升思路

夏日恋语验证：**breadth 成功**（中小R 节日渗透 ×6.5/×2.1），但 **depth 没起来**（高价款偏弱、成熟服节日流水反跑输情人节）。深海节核心 = **双周双核心循环**，第二周专攻深度 + 继续铺可制作的节日付费点。

### 1、双周核心循环（本期最大改动）

| 周 | 核心循环 | 主付费点 |
|---|---|---|
| **第一周（D1–D7）** | **英雄皮肤转盘抽奖 + 排行榜** | 转盘/兑换抽英雄皮肤 + 排行榜冲榜铭牌 |
| **第二周（D8–D14）** | **主城皮肤大富翁 + 航海通行证** | 大富翁跑圈 + 成就礼包 + 通行证 |

**第一周核心循环：英雄皮肤转盘抽奖 + 排行榜**
- 兑换商店兑换 + 转盘**有概率直接抽到「海风旅者·霍普金斯」**（复用老皮肤配置，全服仅 **41 人持有**/超R 5、大R 10、中R 21）。
- 排行榜投放「热浪尤物·赛米拉」（全服仅 **17 人持有**/超R 3、大R 4、中R 5），**排行榜新增传说品质铭牌（带 3% 战斗属性，约主城皮肤增益的 1/3）** —— 刺激已有皮肤的玩家也来冲榜（给老皮肤玩家一个新的冲榜动机）。

**第二周核心循环：主城皮肤大富翁 + 航海通行证**
- **大富翁**（复用航海之路 Voyage 框架）：**换美术资源、换岛屿布局**，每个岛屿**突出对应的奖励**，不做纯装饰感的包装（让玩家一眼看清这格能拿什么）。
- **新增成就礼包**【🔧 占位·待定】：跑满 X 圈后即可购买（**参考 X2 代码**）。*需先改大富翁数值后再定 X 值与定价档位，本期先占位。*
- **BINGO 任务改造**：改为**累计地格任务升级模块**，强化阶段反馈；**50% 活跃玩家可玩付费以外的全部 bingo 地格任务**。
- **主城皮肤**定价跟**原航海之路一致**。
- **航海通行证**：9.99 + **史诗品质铭牌（X3 头衔）**，原有逻辑直接加设计，**不带任何 BUFF**。

## 五、核心循环详细设计

### 【第一周核心循环】英雄皮肤转盘抽奖 + 排行榜

**设计目的**：第一周用"老皮肤返场 + 排行榜新铭牌"双驱动——既给没拿到海风旅者/赛米拉的玩家一个获取通道（转盘/兑换），又用排行榜传说铭牌给**已有皮肤的玩家**一个新的冲榜理由（解决"老玩家皮肤已满、无冲榜动机"的问题）。

**活动详情**：
- 转盘/兑换：复用夏日开箱框架；奖池加入海风旅者·霍普金斯（直接抽到）+ 兑换商店可兑换。
- 排行榜：A+B 线消费冲榜，Top 段奖励除赛米拉皮肤外，**新增深海主题传说铭牌（带一定属性）**。
- 排行榜门槛提示沿用夏日新增的"距离上榜还有 {0} 分"。

**铭牌（头衔系统）说明**：
- **传说品质 + 3% 战斗属性**（约主城皮肤增益的 1/3）→ 走 X3 头衔系统，重复获取按头衔 Regained 转钻石补偿（参 Title 系统）。

### 【第二周核心循环】主城皮肤大富翁 + 航海通行证

**设计目的**：第二周是本期深度增量核心。大富翁（航海之路 Voyage）承载主城皮肤产出 + 跑圈成就礼包，通行证承载史诗铭牌，构成"边跑边攒边付费"的深度循环。

**1）大富翁改动（复用 ActvVoyage 框架）**：
- **换美术资源**：24 地块岛图（`ActvVoyageEvent.DKImg`，按事件组×等级，约 4 套基础岛图：起始/幸运/钻石/神秘）换深海主题；角色模型 `ActvVoyage.DKRoleMedel` 换深海角色。棋盘 24 格位置 prefab 写死，**布局动画零改**。
- **换岛屿布局**：调整 `ActvVoyageIsland` 的地块类型分布。
- **每个岛屿突出对应奖励**：地块美术/文案直接体现该格奖励（不做纯装饰包装），强化"看得见的目标感"。

**2）新增成就礼包（跑 X 圈）**【🔧 占位·待定】：
- 玩家累计跑满 X 圈解锁可购买的成就礼包（**参考 X2 代码实现**）。
- 作用：把大富翁的进度转成付费点，拉深度。
- ⚠️ **需先改大富翁数值后再定 X 圈数 + 礼包定价**，本期先占位，数值定稿后回填。

**3）BINGO 改造（累计地格任务升级模块）**：
- 改为累计地格任务**升级**模块，强化阶段反馈（每完成阶段有明确反馈/奖励）。
- **50% 活跃玩家可玩付费以外的全部 bingo 地格任务**（提高免费参与覆盖，扩大漏斗口）。

**4）主城皮肤**：定价 = 原航海之路一致（复用月心珍珠定价盘，30 人持有、人均终身付费 $22k 的超R向）。

**5）航海通行证**：9.99 + 史诗品质铭牌（头衔，无 BUFF）。

## 六、深海节题材包装方向（供美术参考）

> 题材池参考 P2 深海节：脱离世界观，借"深海/海洋冒险"符号。情绪词：蔚蓝·探险·神秘·奇幻·漂流·寻宝·清凉。

- **主色系**：深蓝/浅蓝/青绿为主，辅金黄(宝藏)、珍珠白(贝壳)、墨绿紫(幽灵/海怪)。
- **大富翁岛屿布局换皮方向**（每格突出奖励）：起始岛=灯塔/码头、宝藏岛=沉船金币、钻石岛=珊瑚宝石、幸运岛=海星/贝壳、神秘岛=漩涡/海怪触手。
- **避坑**：海洋生物卡通化（不要克苏鲁写实恐怖）；装饰底部加水（不要直接落草地）；与已用过的航海之路/夏日海滨做形态区分。
- **可探索方向**：海底城市/海盗沉船宝藏/深潜仿生机械/热带珊瑚礁/美人鱼王国。
