## 〇、包装层（深海节）
- **活动名称**：航海之路（深海远航 / 寻宝航线）
- **主标题**：启航！沿着航线探索每一座神秘岛屿
- **角色**：阿米娜（纸片人 spine，`ActvVoyage.DKRoleMedel`）；深海节皮肤可用「海风旅者·霍普金斯」(兑换皮 5303401)，最终以换皮方案为准 🔧
- **主色**：深蓝/浅蓝/青绿 + 金黄(宝藏)/珍珠白(贝壳)/墨绿紫(幽灵海怪)
- **每格突出奖励命名**：起始岛=启航港 / 宝藏岛=沉船宝藏 / 钻石岛=珊瑚秘境 / 幸运岛=海风礁 / 神秘岛=迷雾漩涡
- **避坑**：海洋生物卡通化（非克苏鲁写实）；与原版航海之路/夏日海滨做形态区分

## 一、设计目的
1. 丰富抽奖活动形式，在抽奖活动中加入新付费点。
2. 付费点：海神罗盘（指定骰子）、抽奖代币礼包、链式礼包、阶段/进度奖励。
3. 最终大奖：主城皮肤「月心珍珠」(≈300刀，月心珍珠 30人持有全付费、人均终身付费$22,149，超R向)。

---

## 二、玩法说明（核心循环）
以大富翁为原型，首尾相连 24 格棋盘（包装成岛屿）。阿米娜从起点出发，每次掷 1 颗骰子（包装成航海罗盘），前进对应格数，每次**停留**获得所在岛的奖励或触发事件。部分岛可升级（停留累计经验，满则自动升级，奖励更好）。神秘宝藏随机刷在岛上，到达额外获得。付费道具「海神罗盘」可自选下一次前进格数。【新增】沿途棋盘常驻珍珠贝，停留拾取填满屏幕上方独立进度条领进度奖励。

## 三、规则说明

### 3.0 基础设定
- **活动类型**：28 = 航海之路（全表唯一专属类型）
- **主活动**：ContentID **2801**；ActvOnline **102801**
- **开启排期**：海域开放(TriggerType=4) → TimeCycle **2702**「海域开放第13天开、持续5天」→ ActvOnline 102801(Type28) 经 TimeController 绑 TC2702。子活动：兑换101332 / 拼图1018091 / BP1022091。
- **自动探索解锁**(V0.2)：累计探索 `ActvVoyageAutoUnlockCondition`(=30) 次后解锁；未解锁点击飘字 `Text_Actv_Btn_Auto_tip`。

### 3.1 岛屿类型（24格）
| 类型 | 数量 | 到达效果 | 可升级 |
|---|---|---|---|
| 起始岛 | 1 | 到达/路过随机获得 2 个岛的奖励（不含宝藏/神秘岛） | 否 |
| 宝藏岛 | 4 | 随机数量活动代币（海盗金币/远洋金币1060） | 否 |
| 神秘岛 | 3 | 触发随机事件（获指定奖励 / 后退N格 / 下个岛经验+2） | 否 |
| 钻石岛 | 4 | 随机数量钻石 | 是 |
| 幸运岛 | 12 | 随机奖励 | 是 |

### 3.2 岛屿升级 【改·本期③】
- 钻石岛/幸运岛可升级，最高 5 级；每停留 +1 经验，满级自动升级、奖励提升、经验清空。
- 升级所需经验(`ActvVoyageLevel`)：lv1→5 = 2/3/7/15。
- **【改】弱化升级奖励反馈，突出岛屿等级的美术反馈**：
  - 升级时**弱化/去掉原升级特效**（原 `GridItem\EffectRoot` 特效，验收反馈中本就未播）。
  - 改为**岛屿按等级变色**表达成长：lv1→5 用不同配色的岛屿美术。
  - 实现：地块图本就按 `(EventGroupID × Level)` 取 `ActvVoyageEvent.DKImg`（`GridItem.cs:92-94`，渲染链现成），美术出 lv1~5 分级配色图填入即可，逻辑零改。

### 3.3 起始岛
- 路过/停留随机获得 2 个岛奖励（不含宝藏/神秘岛）。
- 表现：获奖岛播获奖特效 + 【新增①】海盗金币弹脸（见 3.10）。

### 3.4 宝藏岛
- 到达获随机数量活动代币（海盗金币 1060）。
- 表现：弹活动提示弹窗(见界面 4.4) → 【新增①】海盗金币弹脸 → 飞入惊喜奖励按钮。

### 3.5 神秘岛
触发随机事件并弹提示弹窗：
1. 后退 N 格：关弹窗后阿米娜后退 N 格并获停留岛奖励。
2. 获得随机奖励(读配置)：奖励1=活动代币 / 奖励2=海神罗盘。
3. 下一次到达岛屿额外 +2 经验（若下个岛不可升级，效果保留至可升级岛）。

### 3.6 海神罗盘（付费道具 1058）
- 付费道具，活跃玩家可少量投放。使用可指定下一次前进的岛屿数（交互见 4.3）。

### 3.7 神秘宝藏（1059）
- 活动开始随机刷在一岛（除起始/神秘岛）；到达额外获宝藏奖励；拿走后在其他岛重刷；同时只 1 个；各岛刷新权重读 `ActvVoyageIsland.BoxWeight`。

### 3.8 阶段奖励（按抽奖次数 · 原有保留）
- 达指定抽奖次数(`ActvVoyageOtherReward.NeedTime` vs `totalCount`)领阶段奖励；点任一档一次性领所有可领。
- 8 档(数值表)：抽 20/50/100/150/250/350/500/700。

### 3.9 进度道具（珍珠贝）+ 独立进度条 【新增②】
- **珍珠贝** = 棋盘拾取的进度道具，**机制复用神秘宝藏**（刷格子→停留领→补刷）。
- **刷新**：棋盘上**始终常驻 4 个**（整张地图同时 4 个）；可刷新岛**排除起始/神秘岛**，位置走权重(沿用 `BoxWeight` 思路)。
- **领取**：**仅"停留"在珍珠贝格子才领**（路过不领）→ **进度 +1** → 立刻在别处**补刷 1 个**，维持场上 4 个。
- **进度条**：**独立进度条，置于屏幕上方**，与 3.8 抽奖次数阶段奖励**并存、互不影响**；进度单位=珍珠贝个数；点档位一次性领所有可领。
- **档位阈值 / 各档进度奖励**：🔧占位·待定（随大富翁数值定盘校准；产出速率=常驻4个×跑圈量）。

### 3.10 海盗金币弹脸特效 【新增①】
- **触发**：停在掉海盗金币的格子（宝藏岛/起始岛分奖/神秘宝藏等）、结算获得代币那一刻。
- **表现**：金币从落点**爆开一次**（参考 Coin Master 金币炸开，但**少粒子、无烟花光条、约1.5s 干脆爆一下**）→ 数量跳字 → 汇流飞入【惊喜奖励】按钮（衔接现有飞入+按钮抖动）。叠加在现有飞入流程**前面**，不替换。
- **资源**：金币爆开特效（序列帧/粒子，黑底加色叠加）；金币用海盗金币(`Item 1060`→`DK_icon_Monopoly_currency`)。详见美术特效需求。

### 3.11 成就礼包 🔧占位·待定 【新增⑤】
- 跑满 X 圈后可购买的成就礼包（参考 X2 代码）。**卡数值**：先定跑圈节奏/代币产出，再定 X 值 + 定价。本期占位。

### 3.12 阿米娜移动表现
- 纸片人 spine，岛间跳跃移动；前进>1格连续跳；连抽 5 次时每次终点岛停顿 1s 再继续；到达终点播奖励飞飞动画 + 获取特效。

### 3.13 道具回收
- 活动结束未使用的罗盘/海神罗盘回收，兑换指定道具邮件发放。`ActvItemRecycleMail`=邮件模板 **101111**；`ActvOnline` 新增「道具回收」字段。

### 3.14 红点提示
- 阶段奖励可领：活动按钮 / 页签 / 阶段奖励处 显示感叹号红点。
- 抽奖券>0：活动按钮 / 页签 / 抽奖按钮 显示红点。
- 【新增②】珍珠贝进度奖励可领：屏幕上方进度条处红点。

### 3.15 活动邀请函
- 活动期间首次登陆弹邀请函（优先级最低），点信封展开(MailOpen)，「接受邀请」前往活动界面。
