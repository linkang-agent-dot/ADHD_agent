一句话方案
开箱本体（连锁+锚点）保持不动，按付费段各补一个主优化：【$100 前】新增 X2 式随机礼包（$19.99/$49.99/$99.99 三档每日限购，ROI 对齐现行连锁）——用礼包形式提高付费深度；【$100 后】累充 $500 档直接投放老行军皮肤自选宝箱（MVP·已拍板）——花到必得、零新美术，先试试"$500 独占外显节点"的效果（皮肤照旧一节一款不做降级；验证后二期候选=$700/$1,000 档纯券换外显）；【外显货架】头像框 $9.99 / 聊天表情 $19.99 直售（通用模块零开发）。三组都是加法——回归已证开箱本体稳定（接近夏日、未被挤压），优化只补它缺的中段台阶、头部梯子和外显货架。

模块 A · 随机礼包（X2 式双轨）——$100 前段主优化
一、设计目的
  • 核心目的：填补 $100+ 付费深度（现状=买光墙封顶）；与固定连锁构成 固定+随机双轨（P2 三板斧②，X2 占星已验证）。
  • 付费点：随机包可每日复购 → 深度玩家每天有新的可买项；随机性吸鲸（P2 随机包顶档人均 14.7 单先例）。
  • 目标人群：$50+ 段既有买家（WC 开箱 $100+ 占比 11.6%、约 45 人）+ 被 $585 墙拒收的头部。
二、档位与限购（用户定）
档位价格限购内容形式ROI（期望口径）
随机档 1$19.99每日 1随机池抽取（券+外显碎片+养成，权重公示）对齐现行连锁礼包实配（用户口径：连锁 ROI 至少 6.0 起步）
具体倍数由数值环节按与现行连锁同一张道具价值表反算（深海总览的 2.0x 为另一价值口径，各口径差异大、以同表对齐为准）；期望值口径、随机上下浮动；随机包可略优于固定档（P2 惯例，随机性溢价）
随机档 2$49.99每日 1同池加量+稀有权重上调
随机档 3$99.99每日 1同池加量+超级大奖位（外显）
三、深度容量测算（本方案的核心论证）
口径现状（WC 开箱）加随机礼包后
单人单日可买上限连锁买光即止（一次性）连锁 + $169.97/日（三档各 1）
单人 14 天节日容量连锁+锚点买光 ≈$707（深海转盘同型墙 $585）买光墙 + $2,380 = ~$3,000
对应缺口正好铺满 p99（$460）→ 塔尖（$1,415）→ P2 圣诞上榜线（$1,444）→中位（$2,529）之间的空白台阶
四、奖池设计原则（数值明细待方案过审后按 x3-numerical-design 流程细化）
  • 期望 ROI 锚定现行连锁实配（同一价值表反算，≥6.0 口径起步），随机浮动区间控制（参照随机礼包期望值调整原则：调期望不动天花板）。
  • 池内必含外显位（皮肤碎片/框/表情自选券），呼应"外显要进玩法"结论；养成大头对齐成熟服付费意愿排名（传奇技能书类）。
  • 保底机制：N 次内必出稀有位（防口碑事故）。
落地路线（已拍板 2026-07-16）：直接走 X2 的 GACHA 模块——随机礼包能力随「马戏节扭蛋机 X2→X3 搬运」一起进来，该搬运合入后本模块直接调用，不另做系统。排期上模块 A 与扭蛋机搬运绑定（先后顺序=搬运合入 → 配随机礼包）；每日限购按 X2 模块原生能力配置。

模块 C · 皮肤三层梯子（$200 → $500 → 排行榜）——$100 后段主优化
一、设计目的与卡点机理
  • $500 只剩 3 人的机制性原因（用户判定）：走到那里的人已经拿到核心奖励（开箱皮肤大奖），没有继续付费的动力——累充虽有 $400/$700 档但发的是纯券（有档无钩）。修法不是加返利，是让"拿到皮肤"变成中间站而不是终点。
  • 三层结构（P2 圣诞同构：本体皮中位 $517 → 上榜线 $1,444 → Top3 染色中位 $2,529）：
节点奖励落地作用
$500（累充加档）老行军皮肤自选宝箱（存量行军皮 N 选 1，零新美术）MVP·已拍板累充新增 5000 积分档直发；自选宝箱纯配置（WC 4强自选宝箱 1153/1154 现成先例）——白名单口径=真实付费额，免费券污染不进来花到必得 · 先试试效果：验证"$500 放独占外显"能否把人拉过卡点（观察 $300→$500 斜率是否从 -85% 回到 -40% 常数带）
$700（既有档改造·二期）皮肤同主题配套外显（行军特效 / 专属表情）Reward 组 59806 纯券 600 张 → 换配套外显必得中间站：$500 之后确定性奖励不断档
$1,000（既有档改造·二期）配套外显 II（专属头像框 / 染色部件）Reward 组 59807 纯券 → 换外显向榜的最后一级铺垫
皮肤体系照旧：一节一款、现行投放（开箱大奖+跨服榜）不动，不做降级/双版处理（用户 07-17 定）—保持现行皮肤生产节奏，本方案不加美术负担
定稿（用户 07-17）：一期只做一件事——累充加 $500 档、投老行军皮肤自选宝箱，先试试效果；呈现用累充原生界面（不做跨界面进度条）；皮肤照旧一节一款、不做降级处理。试点验证节点有效（$300→$500 斜率回归 -40% 常数带）后，二期候选=把 $700/$1,000 档的纯券也换成外显（配套行军特效/头像框等，仍用存量或低成本外显）。
三、数据依据
  • 头衔/跨服榜已证明"排行外显竞争有付费力"（塔尖 $1,415/$1,330），缺的就是向下的档位——$200 档正好落在本届 $100+ 买家（11.6%、约 45 人/代）的能力圈上沿。
  • 本服榜现状（回归 kx6）：每服 top1 中位仅 $60——榜在、竞争不在，因为没值得争的东西；第二档皮肤就是"有效投放"本体。
  • 零 BUFF+静态 = 低成本可规模化投放，为"拓展皮肤投放量、提升品质分层"（结论④）铺量产路径。
四、数值校准要点（待数值环节）
  • 累充档位积分：5000 分 = $500（$1=10 积分，PackPrice.Points 实证）；$700/$1,000 既有档（7000/10000 分）只换奖励内容不动阈值。
  • 预期人数（按 -30~40%/级 自然衰减常数）：$500 档 ~25 人起步（累充口径实测）→ 中间站修好后 $700/$1,000 预期 15/10 人（现状 6/2 人，翻 2-5 倍）。

模块 B · 外显直售——头像框 $9.99 / 聊天表情 $19.99
一、设计目的：承接已被竞猜档验证的外显付费需求（框档 76 人/表情档 54 人），把外显从"竞猜搭售"升级为独立货架；用通用礼包模块直接做，零程序开发。 二、SKU 定义
SKU价格内容限购落地方式
头像框礼包$9.99本期新款头像框（+钻石/VIP 按惯例比例）每期 1通用 Pack 直售 + RecommendPack 触发弹窗/活动页售卖位（现有模块，纯配置）
聊天表情礼包$19.99本期新款表情组（+钻石/VIP）每期 1
三、成功条件（规避深海 211019 零销售）
  • 注直售形式本届无实测先例（深海印记框 211019 未上架、改进了 BP）；竞猜侧 $9.99 框档 76 人买证明的是"外显在场景里卖得动"——直售要自己造场景。
  • 直售必须满足至少两条：①新款且本期唯一获取渠道（不进兑换/不进奖池）②强展示场景（上架期内聊天/头像高频曝光，配跑马灯）③可选做成自选宝箱（新款+往期款自选，WC 4强自选宝箱形式已验证）。
  • 品质：按用户定稿结论④方向，皮肤/外显品质提档，为后续"双层排行榜（高低档）"铺素材梯度。
