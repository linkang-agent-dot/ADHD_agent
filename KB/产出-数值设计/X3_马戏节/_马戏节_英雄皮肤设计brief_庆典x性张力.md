# X3 马戏节 · 开箱大奖英雄皮肤设计 Brief（庆典×性张力）

> 2026-07-10 合成自两份全网研究（尘白禁区性感化转型手法 / 碧蓝航线庆典系皮肤设计公式），原始报告要点已收编本文。
> X3 语境校准：X3 无 Live2D 互动系统，皮肤演出走**视频化管线**（reference_x3_hero_skin_video_production）——L2D 手法在此译成"立绘设计+视频分镜"。X3 主发海外，尺度可对标国际服，但见 §五 红线。

## 一、核心公式（两游戏共识，直接当美术需求写法）

1. **场景正当性**：性感必须有"她为什么穿成这样"的叙事理由——马戏节天然赠送：**兔女郎摊主/魔术师助手/驯兽师/空中飞人/领舞团长**全是"职业正当性"外壳（碧蓝：游乐园所以兔女郎；尘白：教练所以紧身衣）。
2. **一皮一强符号**：性感元素不堆砌，全皮肤只立 1-2 个记忆点（碧蓝天狼星=奶盖、腓特烈=红绸、柴郡=轮盘+礼帽）。马戏可选强符号：**红绸吊环/驯兽鞭+礼帽/巨型扑克/魔术幕布**。
3. **重点"框选"而非全裸露**：用剪裁开口/透视纱/丝袜上缘刻意框出大腿根、臀线、背、肩四区之一二（尘白被通报的正是这四处=他们的设计重心，也是红线所在——选 1-2 区，别全开）。
4. **丝袜=腿部导视线**：网袜/黑丝上缘+吊带把视线引向大腿；兔女郎标准件=兔耳+高衩连体衣+网袜+袖口领结+高跟（碧蓝的"袜子系统"是其最强付费说服点）。
5. **萌属性降敏**：兔耳/猫耳/Q 版反差稀释纯性感的攻击性（碧蓝信浓变小信浓、尘白运动兔），也翻倍传播素材。
6. **暗示>直露**：动作清白+联想留白（尘白揉面团/涂防晒乳；碧蓝香槟杯/台球杆）——视频演出里"borderline 单帧全部可过审、含义靠联想"。

## 二、服装设计需求模板（给美术/出图 prompt 用）

- 主体：`{英雄}` 马戏 `{职业外壳}` 装——高衩连体衣或高开衩演出裙（露腿线）+ 背部/腰侧镂空（二选一）+ 过肘手套 + choker
- 腿部：网袜或黑丝（上缘+吊带明确画出）+ 高跟
- 强符号道具（选一）：红绸 / 驯兽鞭+高礼帽 / 巨型扑克牌 / 魔术杖+幕布
- 头饰：兔耳或迷你礼帽（歪戴）
- 配色：**红黑金**（马戏帐篷红+演出黑丝绒+金色滚边灯泡），区别于活动 UI 的红白金——皮肤走"夜场演出"色板拉性张力
- 禁忌：不三点式、不全裸背+全开腿同时出现（四区选二）、无低俗手势

## 三、视频演出分镜手法（L2D 手法 → 视频化翻译）

1. 开场：幕布拉开/聚光灯亮（碧蓝"UI 融入演出"→视频版=灯光当转场）
2. 中段强符号演出（选一）：红绸缠绕旋转下落 / 鞭子抽响指挥火圈 / 扑克牌雨中回眸
3. 性张力镜头语法：**俯拍躺姿/侧卧+道具支点**（碧蓝柴郡卧轮盘）、**足部或手部特写**（尘白涂防晒/揉面的"清白动作性感化"）、慢镜身体曲线转身
4. 收尾：飞吻/wink+礼帽遮面，落版节日 logo
5. 声音：ASMR 贴耳语音+递进音效（尘白验证听觉暗示强于视觉直露）
