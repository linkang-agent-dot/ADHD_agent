# Dashboard 看板

## 概述
Datain 数据平台的 Dashboard/看板/报表/图表管理技能。

## 核心能力
- **Dashboard 管理** — 创建/复制/克隆/修改
- **SQL 图表** — 基于 SQL 查询结果生成图表
- **Cube 图表** — 基于多维模型的图表
- **Filter SQL** — 筛选器管理
- **权限分享** — 控制看板访问权限

## 支持图表类型
折线图、饼图、柱状图、仪表盘等。

## 触发词
`Dashboard` `看板` `报表` `图表` `可视化` `折线图` `饼图` `柱状图` `仪表盘` `复制Dashboard` `Fork` `筛选器` `Filter SQL` `分享` `权限` `Cube` `维度` `指标` `留存` `DAU` `DNU`
