# 数据分析与可视化

## 通用数据分析（data-analysis）
**路径**：`.agents/skills/data-analysis/`

用 pandas/SQL 等工具做数据探索、清洗、统计与可视化并输出结构化报告。
- 支持 CSV、JSON、SQL 查询结果
- pandas 数据处理
- 统计分析
- 可视化报表生成

**触发**：探索数据、创建可视化、统计分析

## 图表可视化（chart-visualization）
**路径**：`~/.agents/skills/chart-visualization/`

智能选择图表类型（约 26 种可选），提取参数并用 JS 脚本生成图表图片。

**工作流**：
1. 分析数据特征 → 选择最佳图表类型
2. 提取图表参数（标题、轴、系列等）
3. 调用 JS 脚本生成图片

**触发**：可视化数据、需要图表
