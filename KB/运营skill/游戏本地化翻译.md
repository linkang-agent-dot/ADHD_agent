# 游戏本地化翻译

## 概述
游戏本地化翻译全流程工具。从 UX 截图或原始文本提取 UI 文本，自动查重、复用翻译记忆、生成 18 语言翻译，支持写入暂存区 review 或直接写入目标页签。

## 工作流
1. **文本提取** — 从 UX 截图识别 UI 文本，或直接接收原始文本
2. **查重** — 检查是否已有相同/相似翻译
3. **翻译记忆（TM）** — 复用已有翻译
4. **18 语言翻译** — 生成所有目标语言版本
5. **写入** — 暂存区 review 或直接写入目标页签

## 支持 18 种语言
全覆盖游戏多语言配置需求。

## 触发场景
`游戏多语言配置` `本地化翻译表` `i18n配置` `UI文本提取` `多语言扩散翻译` `LC_Key生成` `翻译存入表格` `UX截图翻译` `翻译LC` `直出i18n`
