# 换皮工具集

## 一、装饰粉刷（art-reskin-brush）
**路径**：`.cursor/skills/art-reskin-brush/`

### 三类流程

#### 1) 装饰粉刷
给游戏图标右下角叠加蓝紫柄橙毛刷子，标识可换皮资产。
- GRFal Gemini 图生图
- PIL 洪水填充去白底
- 256×256 透明 PNG

#### 2) 头像框生成
AI 生成节日头像框完整闭环：
- GRFal 生成 → 规格校验（350×350 RGBA，内径 54-67%，环宽 26-59px）
- SVN 上传 → DK 写入 GSheet

#### 3) 铭牌生成
AI 生成节日聊天铭牌完整闭环：
- GRFal 生成 → 抠图 → 规格校验（128×128 RGBA）
- SVN 上传 → 颜色方案示意图 → 1173 配置表填表

### 触发词
`加刷子` `装饰粉刷` `换皮标记` `头像框` `铭牌` `nameplate` `chat skin`

## 二、UI Sprite 换皮（ui-sprite-reskin）
**路径**：`.cursor/skills/ui-sprite-reskin/`

### 核心能力
AI 生成替换图片 + 保持原始 alpha mask 形状，无拉伸。
- AI 按正确宽高比生成 → cover-crop → alpha mask 叠加
- 支持单文件和批量文件夹
- 颜色/主题变体生成
- 目录海报卡片合成（Figma 风格边框 + 内描边）

### 触发词
`换皮` `reskin` `替换贴图` `批量替换图片` `蒙版` `alpha mask` `catalog poster`

## 三、SVN 上传（svn-upload）
**路径**：`.cursor/skills/svn-upload/`

P2 美术资产通过 TortoiseProc.exe（本机无命令行 svn）完成 Add + Commit。
- 按指定 ID 为文件名复制到 SVN 工作目录
- 支持单文件和批量上传
- 内置常用目标路径（装饰物、头像框等）

### 触发词
`传SVN` `上传SVN` `svn提交` `传图到SVN`
