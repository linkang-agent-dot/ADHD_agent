# X2 统一媒体入口

## 概述
X2 项目所有图片、视频、3D 及媒体处理请求的唯一入口。GRFal 为主后端，art-skills 为自动 fallback，双后端互备。

## 8 种专属类型
1. **技能图标** — 游戏技能图标生成
2. **集卡册卡片** — CardGallary 卡片
3. **行军表情** — 行军表情包
4. **成就徽章** — 成就系统徽章
5. **游戏视频** — 视频制作与集成
6. **UI 素材提取** — UI 拆解与元素提取
7. **活动图标** — 节日活动 ICON
8. **通用生图/修图/生视频** — 万能媒体处理

## 17+ 种工具能力
抠图、超分、换背景、扩图、3D、虚拟试穿、LoRA、文生图、图生图等。

## 触发词
**中文**：`生图` `画图` `做图` `文生图` `图生图` `抠图` `超分` `换背景` `扩图` `活动图标` `切图` `拆UI`
**英文**：`generate image` `generate video` `remove background` `upscale` `grfal` `UI extract`
