# 兑换商店分析

## 概述
节日兑换商店数据深度分析（Notion + Wiki 双版本）。

## 五维分析
1. **饱和度排名** — 各商品兑换进度排名
2. **代币消耗结构** — 代币去向分布
3. **人次 vs 消耗气泡图** — 参与度与消耗量关系
4. **四象限定位** — 高/低频 × 高/低消耗
5. **皮肤专项** — 皮肤类商品专项分析

## 自动能力
- 自动分类道具
- 运营优化建议

## 数据输入方式
1. 文本粘贴
2. Excel 文件
3. JSON 直接输入

## 输出
- 5 张高清图表
- 分类统计
- 运营优化建议
