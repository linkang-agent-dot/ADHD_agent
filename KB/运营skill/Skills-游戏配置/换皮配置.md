# 换皮配置

## 概述
按模块分析 P2/X2 换皮需要修改的表和字段，生成配置行思路。上传流程走 [[P2配置上传]]。

## 工作流
1. 识别活动模块类型
2. 分析该模块涉及的所有表
3. 逐表列出需换/不换的字段
4. 生成配置行（Tab 分隔，可粘贴到 GSheet）

## 参考文档
- `modules-p2.md` — P2 模块换皮字段索引
- `modules-x2.md` — X2 模块换皮字段索引

## 相关知识
- [[换皮规则详解]] — 完整的列类型处理规则与追踪链
- [[配置案例集]] — 实际换皮案例

## 重要规则
- `base-ID` 不换
- `A_STR_constant` 必须新增不能复用
- 每层引用都要向下追踪
- 输出时 `""` 必须写成 `'""`（GSheet 粘贴规则）
