# P2 配置上传

## 概述
从 Google Sheet 下载配置表并提交到 gdconfig 仓库。支持全表下载和行筛选两种模式。内置 898 张表的编号→页签映射。

## 重要限制
> **表 1011（i18n）暂不由 Agent 自动上传**（Skill 正文有禁令）

## 两种模式
1. **全表下载** — 下载整张表的所有数据
2. **行筛选** — 按 ID 筛选特定行下载

## 工作流
1. 用户提供表号 + 分支名
2. 查表号→页签映射
3. 从 GSheet 下载数据
4. 格式校验 + ID 重复检测
5. 提交到 gdconfig 仓库对应分支

## 关键文件
- `scripts/i18n_dl_utf8.py` — i18n 下载脚本
- `scripts/tab_history.py` — 页签历史
- `references/cases/` — 案例参考
