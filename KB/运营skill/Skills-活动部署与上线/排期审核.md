# 排期审核

## 概述
对比排期审核表（xlsx）与正式上线表（csv），进行三维审核并生成 markdown 报告。

## 三维审核
1. **活动条目匹配** — 排期表中的活动是否都在上线表中
2. **上线时间对比** — 时间是否一致
3. **服务器配置对比** — 服务器分组是否匹配

## 输入
- 排期审核表（`.xlsx` 格式）
- 正式上线表（`.csv` 格式）

## 输出
- Markdown 格式审核总结报告

## 关键文件
- `scripts/` — 审核脚本
- `business_rules.md` — 业务规则说明
