## 一、活动背景

1. X2 首次上线大富翁形式（换皮自 P2 普通版大富翁）；UI从横屏改为竖屏
2. 活动模块：大富翁玩法 + **成就礼包（13档）** + 存钱罐 + **锚点礼包（5档）** + **帮派红包（2档）** + 兑换商店
3. 当前仅 2 个新服（1006902、1006802）上线了大富翁模块，各约 660 用户
4. 同期老服没有大富翁模块，作为对照组
5. 免费骰子设计投放约 470 个（累充300 + BP50 + 扭蛋30 + 掉转30 + 7日30 + 抢购30），设计目标到 30 圈

---

## 二、结论

1. **大富翁是节日新模块中收入贡献最大的**：新服均值 $94.92，占 3 个新模块增量的 **59.4%**
2. **收入高度集中于超R**：3 名超R贡献了 $179.88（占大富翁总收入 **94.8%**），中R及以下只买了 $1.99 的入门礼包
3. **13档成就礼包，仅5档有购买，8档零购买**：设计了从5圈到200圈共13档成就礼包，但40圈及以上全部无人购买，说明绝大多数玩家进度未超过30圈
4. **未蚕食共有模块**：加入大富翁后，共有模块收入仍持平（+7.6%），属纯增量叠加
5. **核心问题不是礼包档数不够，而是玩家进度推不动**：免费骰子覆盖~30圈，之后玩家缺乏动力继续

## 五、后续优化建议

基于设计方案 vs 实际数据的差距，以及 P2 历史迭代经验，提出以下优化方向：

### 5.1 成就礼包价格梯度修正（优先级：高）

**现状问题**：13档设计了，但价格梯度不对。第1档$1.99 → 第2档直接跳到$99.99 → 第3档反而$49.99，不符合玩家逐步升级的付费心理。P2 的梯度是严格递增的。

**建议**：
- 重排前5档价格为递增序列：$1.99 → $4.99 → $9.99 → $19.99 → $49.99，对应 5\~30 圈
- 30圈后的后8档适当调整为 $4.99\~$99.99 逐级递增
- 参考 P2 做法：低价高频在前，高价在后；前期转化，后期吸金

**作者一句话评价** ：前期版可以做，按逐级递增降低玩家付费门槛

### 5.2 深度进度驱动力不足（优先级：高）

**现状问题**：40圈+成就礼包全部0购买，玩家进度集中在30圈以内。

**待验证**：当前缺少全量玩家和付费玩家的停留关卡分布曲线，无法准确判断断崖原因。免费骰子投放170个到30圈只是巧合对应，付费玩家通过购买礼包获得额外骰子（成就礼包2自带300个），理论上可以推进远超30圈。需要用 counter 或活动参与数据拉出两条曲线：
- **全量玩家进度分布**：看大盘停在哪里
- **付费玩家进度分布**：看付费玩家是否也停了，还是继续在推

**建议**：
- **加深度驱动**：30圈后的阶段奖励需要更有吸引力（当前阶段奖励ROI仅0.37，远低于成就礼包2.42）
- **投放装饰物：在30圈设置一个小目标装饰物，50\~100圈设置升级版装饰物，创造持续追求动力

### 5.3 红包礼包（优先级：低）

**锚点说明**：锚点礼包本身的设计目的是**价格锚定**（让成就礼包显得更划算），购买率低是正常的，不作为优化项。

**红包问题**：2档全部0购买（$49.99/$99.99），门槛偏高，且新服社交生态不成熟。后续可在社交功能更完善时降低起始价位再观察。

**作者一句话评价** ：本身基于做联盟社交，X2单服人数较少，这块需要注意社交、人数相关的礼包设计都要降低门槛。

### 5.4 存钱罐维持现状

付费率 37.5%（3/8），$2.99 表现正常。
