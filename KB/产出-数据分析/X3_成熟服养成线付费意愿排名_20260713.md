# X3 成熟服养成线付费意愿分析（2026-05-14 ~ 2026-07-12，60天）

> 目的：为「高级皮肤挂属性/技能/养成内容」方案提供选品依据——成熟服付费玩家最愿意为哪条养成线掏钱。

## 一、口径与 SQL 要点

- 数据源：Datain TRINO_HF，`v1090.ods_user_order`（`pay_status=1`）join `v1090.dim_iap`
- USD 金额：`CASE WHEN currency_type='usd' THEN actual_charge ELSE pay_price END`（规避 TOKEN actual_charge=USD×100 坑）
- 成熟服段：`TRY_CAST(server_id AS INTEGER) BETWEEN 1000 AND 1870`（1-88服）
- 买家 = `user_id||'_'||server_id` distinct；R级 join `dl_user_rlevel_all_info`（game_cd=1090, partition_date='2026-07-12'，按 user_id）
- 礼包→养成线归类：dim_iap 三级类目（iap_type/type2/type3 + note）为主；对 SLG 城建大杂烩/节日包等 60+ 个高收入礼包用 `ods_user_asset WHERE reason_id='buy_gift' AND reason_sub_id=礼包id` 实际发放内容定性（join dim_asset 译名）
- 总盘：**$471,729 / 2,108 买家 / 35,195 单 / 1,093 个 iap_id**（top500 礼包覆盖 94.6% 收入）
- 中间结果：本目录 s1_iap_agg.json（按包聚合）、s4_pack_contents.json（包内容）、s5_lines.json（按线聚合）、s7_rlevel_lines.json（线×R级）

## 五、常驻 vs 节日

- 节日活动包合计 $59.5k = 总盘 12.6%（节日抽奖包 $31.5k 全额 + 战令线里 64% 是节日护航令/集结令 $28k）。
- 其余 87% 是常驻售卖。**养成内容本身谁最能卖的答案在常驻盘**：英雄养成($91k) > 战令 > 船只装备 ≈ 资源加速 > 每日特惠 > 海妖 ≈ 活动抽奖。
- 节日包的作用是"复购结构最好的售卖形式"（抽奖券 6.2 单/人），不是"卖了别的东西"——节日包里发的仍是抽奖券+钻+VIP点。

## 六、结论：高级皮肤挂什么、奖池放什么

1. **付费号召力排名（意愿分）**：船只装备/改装 > 英雄养成 > 抽奖代币类 > 英雄技能 ≈ 战令 > 资源加速 > 海妖 > 纪念卡 > **纯外显（垫底组）**。
2. **皮肤附加价值选型**：
   - 首选挂**英雄养成向**：专属技能/技能书 + 信物碎片。理由：英雄养成是量价双优第一大线（19.3% 收入、52% 付费玩家参与）；每日特惠证明"技能书+信物"是全场复购最广的硬通货；海风旅者(带专属技能"商会互惠条约")已有先例。
   - 顶配版可叠**裸属性加成**：220002 证明 $99.99 纯属性包有 39 人买单、63% 收入来自顶档——皮肤挂属性能直接锚顶档价。
   - 鲸鱼定向可绑**船只改装材料/纪念卡入场券**：船只装备是超R+大R 出钱占比最高的战力线(55%)；纪念卡是超R集中度最高的情感线(50%)。
3. **抽奖奖池选品（按已验证付费意愿排序）**：信物/英雄碎片 > 传奇技能书 > 船只改装材料(神秘金属/高级木板/航海屏障碎片) > 渊海秘币(海妖) > 招募美酒 > 钻石+VIP点数(保底填充)。避免纯外显小件当主奖投放（复购 2.1 单/人，拉不动循环）。
4. **形式建议**：参照 210921 夏日柔情海湾（皮肤+抽奖券 $99.99×30人）与 220001 英雄养成手册（BP 形式 $23.5k 单品）——"高级皮肤当抽奖头奖 + 奖池装英雄养成硬通货 + 冲榜不封顶"与既有付费结构完全吻合。

### Caveats
- 1000-1160 段部分已合服零流水，属正常；买家跨服同人未合并（口径=user_id×server_id）。
- 战令/BP 类不走 buy_gift，按名称+类目归线，内容定性粒度弱于礼包类。
- 每日特惠为混合线单列，若按内容拆分英雄向占比会进一步抬高英雄养成线。
- R级表以 2026-07-12 快照为准，未分级=$11.8k(2.5%)。
