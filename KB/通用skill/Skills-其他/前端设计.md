# 前端设计

## 概述
创建独特的、生产级前端界面，具备高设计品质，避免泛化 AI 审美。

## 适用场景
- 网站 / 落地页
- Dashboard
- React 组件
- HTML/CSS 布局
- 海报式 artifact
- 任意 Web UI 美化

## 特点
- 强排版意识
- 动效设计
- 主题一致性
- 非模板化风格
