# 飞书异步通知

## 概述
异步多 Agent 协作通知系统，覆盖三大能力。

## 三大能力

### 1. 完成通知
Agent 完成任务后通过飞书推送"已完成"。
```powershell
python notify_done.py --task "任务描述" --result "结果摘要" --agent "Cursor"
```

### 2. 确认请求
需要用户决策时发飞书阻塞等待回复。
```powershell
python notify_confirm.py --task "任务名" --question "决策问题" --agent "Cursor" --timeout 600
```
返回 `{"decision": "yes"/"no"/"timeout"}`

### 3. 全自动后台执行
`cursor_daemon` 常驻轮询任务队列：
- 发现任务 → 调用 Cursor Agent CLI 执行
- 写结果 → 推飞书
- 无需 Cursor 窗口保持打开
- 支持模型/模式/沙箱按任务配置

## 脚本路径
`C:\ADHD_agent\.cursor\skills\async-notify\scripts\`

- `submit_task.py` — 提交任务
- `cursor_daemon.py` — 后台守护进程
- `cursor_watchdog.py` — 监控狗
- `daily_report.py` — 日报
- `feishu_helper.py` — 飞书 API 封装

## 强制规则
> 每次完成涉及 >2 个工具调用的任务，最后一步**必须**发飞书通知。无例外。
