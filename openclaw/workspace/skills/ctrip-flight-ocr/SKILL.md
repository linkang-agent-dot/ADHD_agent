## 触发条件
用户发来携程机票截图，并提到"机票"、"航班"、"价格"相关需求。

## 工作流程

### Step 1: 用 image 工具识别截图
对截图使用 `image` 工具，prompt 如下：
```
识别图中所有航班信息，输出格式：
- 航线：出发地 → 目的地
- 航班号 + 航空公司
- 日期
- 起飞时间 - 到达时间
- 价格（元）
- 是否直飞
```

### Step 2: 分析整理
- 过滤直飞航班
- 按价格排序
- 标注最低价

### Step 3: 回复用户
- 整理好的表格形式输出
- 指出最优惠的航班
- 简洁直接，不要废话

## 示例输出格式
```
🛫 携程机票筛选结果

✅ 直飞 | 4.30-5.4 | 上海✈️三亚
| 航班 | 航司 | 起飞-到达 | 价格 |
| CA1234 | 东航 | 08:00-11:30 | ¥580 |
| MU5678 | 吉祥 | 14:20-17:45 | ¥420 ← 最低价 |

💰 最低价：吉祥航空 MU5678，¥420
```

## 注意事项
- 用户发图时直接调用 image 工具识别，不需要先问用户
- 如果图不清晰，标注"信息可能存在误差"
- 重点标出直飞和最低价
