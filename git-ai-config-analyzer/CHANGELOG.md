## [1.0.0] - 2026-02-02

### 新增
- ✨ 首次发布
- 🎯 Git 提交记录检查功能
- 📊 配置文件差异内容提取
- 🤖 AI 智能配置分析（支持 GPT-4 和 GPT-3.5）
- 🏷️ 自动识别功能新增配置
- 📝 多格式报告生成（Console、Markdown、JSON）
- ⚙️ 灵活的命令行参数配置
- 🚫 无 AI 模式支持（不需要 API Key）
- 📋 批处理脚本快速使用
- ✅ 基础单元测试

### 功能特性
- 支持自定义仓库路径和目标目录
- 支持指定基准分支和比较分支
- 支持自定义提交范围
- 智能分类：功能新增、参数调整、Bug修复、结构重构、废弃移除
- 影响评估和优先级判断
- 配置同步建议
- 相关系统识别
