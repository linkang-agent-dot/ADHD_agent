## 🎯 核心知识速查

### 进攻三板斧
- **护** → 稳住球权
- **带** → 创造空间
- **传** → 组织进攻

### 防守三要素
- **跟** → 正确站位
- **贴** → 干扰出球
- **拉** → 压缩空间

### 属性阈值
- 以"5"为分界：135→140→145→150→155
- 突破阈值比堆数值更重要

### 选人优先级
- 数值 > 模型 > 特性 > 逆足
- 前场双五优先级：中锋 > 边锋 > 前腰 > 中场
