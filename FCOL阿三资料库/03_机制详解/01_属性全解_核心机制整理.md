## 前言

球员属性是FC OL的核心机制，理解属性的作用和阈值，是建队和挑选球员的基础。

本篇系统整理所有核心属性的作用、阈值和实战影响。

---

## 一、属性阈值机制

### 什么是阈值？

游戏中球员属性并不是线性增长，而是存在**阈值档位**。

阈值档位以"5"为分界：
- 135 → 140 → 145 → 150 → 155

每突破一个阈值，球员该属性的实际表现会有明显提升。

### 阈值的意义

- 144和145差1点，但实际差距巨大
- 145和149差4点，但实际差距很小

**所以：突破阈值 > 单纯堆数值**

## 二、核心属性分类

### 速度类属性

| 属性 | 作用 |
|------|------|
| 速度 | 持球状态下的最高速度 |
| 加速 | 从静止到最高速的时间 |

**关键阈值**：145、155

- 145：具备跑锋能力
- 155：顶尖跑锋水准

### 射门类属性

| 属性 | 作用 |
|------|------|
| 射术 | 射门的准确度 |
| 射门力量 | 射门的球速 |
| 远射 | 禁区外射门质量 |
| 凌空抽射 | 凌空/半高球射门 |
| 弧线 | 弧线球质量（搓射、传中） |

**关键阈值**：145、155

### 传球类属性

| 属性 | 作用 |
|------|------|
| 短传 | 短距离传球准确度和球速 |
| 长传 | 长距离传球准确度 |
| 视野 | 传球时的AI辅助瞄准 |
| 传中 | 边路传中质量 |

### 盘带类属性（四带）

| 属性 | 作用 |
|------|------|
| 盘带 | 带球时的触球频率和稳定性 |
| 控球 | 停球质量和护球能力 |
| 灵活 | 转身和变向速度 |
| 平衡 | 对抗后的稳定性 |

**四带是影响手感的核心属性！**

### 身体类属性

| 属性 | 作用 |
|------|------|
| 反应 | 对突发情况的响应速度 |
| 体力 | 全场持续输出能力 |
| 强壮 | 身体对抗能力 |
| 对抗 | 护球和肉搏能力 |
| 弹跳 | 争顶高度 |

### 防守类属性（三防）

| 属性 | 作用 |
|------|------|
| 人盯人 | 盯人和卡位能力 |
| 拦截 | 拦截传球路线 |
| 抢断/铲断 | 主动抢球能力 |

## 三、位置属性优先级

### 中锋

**最重要**：速度、射术、射门力量、四带
**次要**：头球、对抗、反应

### 边锋

**最重要**：速度、加速、四带、传中
**次要**：远射、弧线

### 前腰/中场

**最重要**：短传、视野、远射、四带
**次要**：速度、防守属性

### 后腰

**最重要**：三防、体力、对抗
**次要**：速度、短传

### 中后卫

**最重要**：速度、三防、强壮、头球
**次要**：反应、短传

### 边后卫

**最重要**：速度、加速、三防、传中
**次要**：体力、强壮

---

## 四、隐藏属性

### 逆足

影响非惯用脚的所有操作质量。

等级：1-5星
- 5星：双脚无差别
- 4星：逆足有一定影响
- 3星及以下：逆足严重影响表现

### 花式

影响可使用的花式动作数量和质量。

### 体型

- 瘦弱：灵活好，对抗差
- 普通：平衡
- 健硕：对抗强，灵活差

---

## 五、属性搭配原则

### 1. 突破阈值优先

宁可放弃一些不重要的属性，也要让核心属性突破阈值。

### 2. 四带协调

盘带、控球、灵活、平衡四项要协调，不能有明显短板。

### 3. 三防协调

人盯人、拦截、抢断三项同样要协调。

### 4. 体力不能太低

再强的属性，体力不够70分钟后都是摆设。

---

## 六、总结

属性的核心要点：

1. **阈值机制**——以"5"为分界，突破阈值是关键
2. **位置匹配**——不同位置关注不同属性
3. **四带决定手感**——盘带、控球、灵活、平衡
4. **三防决定防守**——人盯人、拦截、抢断
5. **逆足很重要**——尤其是前场攻击球员
