## 3.2 积分经济

**积分来源**

| 来源 | 单次积分范围 | 说明 |
|---|---|---|
| 矿石进阶宝箱（联盟币矿）| 200 ~ 1,400 | 随矿种等级递增，共15矿种 |
| 矿石完成宝箱 | 200 ~ 1,000 | 挖完矿石时触发 |
| 子关卡完成宝箱 | 1,000 ~ 5,000 | 每关1次，单次最大来源 |
| 免费宝箱（3小时/次）| 200 ~ 600 | 4天共约32次，轻度玩家主要来源 |

> 基准：1小时效率 ≈ **2,000 积分**（无英雄BUFF）

**玩家4天总积分预估**

| 玩家类型 | 4天总积分 | 可到关卡 |
|---|---|---|
| 轻度（无付费）| 4,000 ~ 6,000 | 第2关 |
| 中度（无付费）| 10,000 ~ 14,000 | 第3关 |
| 中R（$20~50 药水）| 20,000 ~ 32,000 | 第3~4关 |
| 超R（$100+ 药水）| 60,000 ~ 100,000 | 第4关深层 |
