# 模块二：搬运修改内容

---

## 1.1 规则改动

| 改动点 | P2 原版 | X2 节日版 | 改动理由 |
|---|---|---|---|
| **开启条件** | 主线关卡4完成 + 基地5级 | **直接开放，无前置条件** | 触达率差距主要来自门槛；节日挖矿资源独立，开放不破坏经济 |
| **英雄挖矿 BUFF** | 英雄提升挖矿效率，绑定当期售卖英雄 | **删除** | 砍掉英雄依赖，降低参与门槛，避免无英雄玩家极低效率 |
| **排名奖励** | 德里克金头 + SLG 道具 | **通用养成材料**（具体道具见模块三） | 通用养成对各段位玩家均有价值 |
| **奖励层级** | 仅排名奖励 | **排名奖励 + 节日卡册（双层）** | 卡册兜底中低付费用户获得感，排名驱动超R冲榜 |
| **付费加速** | 活动专属药水礼包 | **保留**（具体礼包见模块三） | 积分付费加速是超R拉开差距的核心入口 |
| **工人变身** | P2 未实现 | **不做** | P2 未落地，X2 同样不做 |
| **怪物矿** | P2 未实现 | **不做** | 同上 |
| **活动周期** | 预热1天 + 开放4天 + 展示1天 | **直接复用** | 节奏合理，无需调整 |

## 1.2 界面改动

| 模块 | 改动幅度 | 说明 |
|---|---|---|
| 关卡地图场景 | **轻改** | 直接搬 P2 场景，加节日主题包装；不重新设计 |
| 矿石外观 | **中改** | 特殊矿石（积分矿等）需出 X2 风格新图 |
| 节日卡册界面 | **轻改** | X2 已有卡册系统，替换配置内容即可 |
| 排名奖励界面 | **轻改** | 结构不变，替换道具图标 + 文案 |
| 活动入口图标 | **轻改** | 换一张图标 |
| 科技界面图标 | **轻改** | 结构复用，替换 icon |
| 英雄展示区 | **删除** | 随英雄 BUFF 一并去掉，无需保留 |

## 1.3 节日卡册复用规则

X2 已有卡册系统，本次**直接复用框架**，仅替换配置内容。

| 项目 | 复用/改动 | 说明 |
|---|---|---|
| UI 框架 | **复用** | 卡册界面、进度条、格子解锁动效全部不动 |
| 进度记录逻辑 | **复用** | 积分累积→格子解锁的触发逻辑不变 |
| 解锁机制 | **复用** | 达到门槛自动解锁，无需手动领取逻辑不变 |
| 总格数 | **改** | 本次设定 20 格（与 X2 现有卡册格数对齐，如不一致需配置调整）|
| 积分门槛 | **改** | 按模块三 3.5 节重新配置（500 → 100,000）|
| 每格奖励道具 | **改** | 替换为挖矿活动专属道具（见模块三 3.5）|
| 卡册名称/文案 | **改** | 替换为节日挖矿主题文案 |
