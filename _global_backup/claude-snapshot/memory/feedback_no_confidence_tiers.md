2026-07-10 对照「影子迁移包」（微信收到的 Cursor AI 影子知识库，逻辑全景见 KB\方法论\影子迁移包_逻辑全景_20260710.html）时用户裁决：**本体系不引入置信度三级标注制度**。

**Why**：影子包的置信度是为"AI 观察人"设计的（画像天然半猜测）。本体系 memory 全部来自真实案例落地（feedback=用户明说、workflow=跑通过、reference=实测、坑=真实翻车），写入即高置信，分级无区分度只剩仪式成本。真正会失真的轴是**时效性**（如导表迁 TSV 让旧知识一夜失效），已有对策：recalled memory 用前验证 + 发现错就删改 + 周五归纳清单。

**How to apply**：
- 不要再提议给 memory 加置信度符号/月度置信度裁决制度。
- 唯一保留的习惯：写入**推断而非实证**的结论（如根因未实锤的"可能是XX"）时，行内显式标「未验证/推测」，验证后回写删标（现有 ⚠️ 用法延续）。
