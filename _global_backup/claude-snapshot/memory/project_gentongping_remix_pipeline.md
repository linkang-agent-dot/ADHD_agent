**方法论教训（拆分精度）**：
- ⚠️ **整片转写会整段漏人声**：小蕊-3 的 11~30s 有 19 秒人声被 whisper 整片跑直接跳过（medium/small 都漏）→ 切点附近必须 ±6s **窗口化重转写**（vad_filter=False）才可信；"空窗"必须抽帧看画面再定归属
- ⚠️ **寻点必须锚定下一段开头词**（"认准/千万/这是/现在"），只在该语义边界测间隙；自由"找最近的宽间隙"会滑进邻句内部（初版实测踩过）
- 间隙分级：≥0.3s 优 / 0.12~0.3 中置信 / <0.12 低置信人工听（口播语速快，0.3 一刀切太苛刻）
