### 🔑 夏日恋歌回归核心结论（第二轮优化的地基，来自 2026-06-01 会话）
口径：成熟服（D0已过D35），付费玩家付费率 = 节日付费/总付费。详见 [[reference_x3_festival_report_template]]。

1. **breadth 成功，验证有效** —— 付费率提升 **90% 是真实档内渗透**（不是结构假象）。中小R 渗透暴涨：小R 5.1%→32.9%(×6.5)、中R 29.5%→61.6%(×2.1)。高档接近饱和（超R 91%、大R 77%）。
2. **驱动款 = 两个 $4.99 低价款**：① 许愿池 $4.99（可重复抽，订单 121→321）；② 特惠连锁 $4.99 入门款（210702）。把"本来会付费但不买节日包"的中小R 大量转化进来。
3. **❌ 深度仍然没起来（第二轮最大缺口）**：高价档(≥$50)买家夏日 ~39人 vs 情人节 ~84人（情人节有 210716皮肤$49.99×55、210717家具$99.99×16）；**成熟服上夏日节日流水反而跑输情人节（$3.7k vs $6.5k）**，全88服打平靠新服红利。增量几乎全在 $4.99–$9.99 breadth，**深度款偏弱**。
4. **数据卫生债**：夏日特惠连锁链文案仍是情人节"纯白花嫁礼包"（TimeCycle/Pack 复用残留，没换皮），玩家看到的不是夏日主题——下次换皮要扫这类残留。

→ **第二轮优先级明确**：breadth 已验证（保持 $4.99 可重复/阶梯入门），**重点转向「付费深度」**——把模块1从"待启动"提到第一优先。
