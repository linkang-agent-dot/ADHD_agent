## 三份文档分工

| 文档 | 角色 | 核心问题 | 更新频率 |
|------|------|---------|---------|
| `p2-numerical-design/养成线深度手册.md` | 数值侧落档 | 每条养成线值多少美元、消耗多少材料 | 配表大改后更新 |

引用关系：制作人梳理 → 引用 → 深度手册 + 数据现状档案；深度手册反向引用制作人梳理。

## How to apply

- **新礼包定价 / 活动掉落估值** → 先查 `养成线深度手册.md` §一~§六 的材料单价
- **判断中R扩线方向** → 查 `P2养成线与养成线梳理.md` §六（中R扩线可行性矩阵）
- **看当前运营数据/趋势** → 查 `P2数据现状档案.md`（每月最新一期月报合入）
