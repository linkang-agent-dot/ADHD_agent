X3 评估"节日有没有把基本盘做起来"时的标准月环比方法。配合 [[reference_x3_festival_performance]] / [[reference_x3_festival_report_template]] 用。

## 同生命周期月环比方法（评基线收入/ARPU 是否提升）
问题形态："本期节日 vs 上个月，整体/非节日基线收入和 ARPU 提升了吗，同生命周期服"。
- **窗口对齐**：节日窗口（如夏日 5.29–6.7，10天）↔ 上月同日历窗口整月平移（4.29–5.7，等长）。注意 D0 是 **5.29**（不是 5.30）。
- **生命周期对齐 = 各窗口各取「当时已过 D35 的成熟服批次」**（物理服不同、服数不同），不是同一批物理服（用户明确否过"同一批服"）。
  - 服筛：`dim_open_server` `CAST(open_time AS DATE) <= 窗口起始 - 35天`。基线@4.29→open≤03-25；节日@5.29→open≤04-24。
  - ⚠️ 服数不同（基线42 vs 节日50活跃服）→ **只比率(ARPU/人均/单服)，绝对流水不可比**。
- **收入口径**：同日报 `REV = CASE WHEN currency_type='usd' THEN actual_charge ELSE pay_price END`，`pay_status=1`。
- **⚠️ ARPU 分母口径分两种，判"衰减/回升"必须用 DAU 口径，别用付费人数**：
  - `ARPU = 流水 / 登录去重人数(DAU)` ← 真 ARPU，判趋势/活动效果用这个（DAU 取 `v1090.ods_user_login` distinct user_id，按 server_id IN 成熟服集）。
  - `ARPPU = 流水 / 付费人数` ← 日报里写的"ARPU=流水/付费人数"其实是这个（见 [[feedback_x3_festival_arpu_denominator]]），它会把"付费人数涨"消掉、让活动效果看不见，**判衰减/回升会得反结论**。
  - **必须三件套一起看**：ARPU(/DAU) = 付费率(付费/DAU) × ARPPU(流水/付费)。拆开才知道动的是渗透还是深度。
- **节日/非节日拆分**：节日包 = ActvOnline 累充白名单（夏日 100595 的 col49，17包：210702/704/706/708/710/712/713/714/715/717/921/917/918/919 + 130020/130021/1002001）；非节日 = 总 − 节日。
- **基线窗口必须验"无藏活动"**：查 Top 礼包，若全是日常款（每日特惠/月卡/养成手册）= 干净基线；若有集中爆款 = 实为活动 vs 活动，对比失真。

## ⚠️ 合服踩坑：配置服数 ≠ 活跃出单服数
- 1000-1880 配置上是 **89 服**（dim_open_server），但节日窗口**只有约50个服 id 有付费订单，最小 id=1170**——低 id 老服已合服，订单归到存活服 id 下。
- 影响：**按 id 区间聚合流水/人数不丢数**（合并服的钱记在存活服 id）；但**按"服数"算单服指标要用活跃出单服数**（JOIN dim_open_server 后 count distinct server_id），不能拿配置的 89。

## 夏日节结论（2026-06，同生命周期 D35+ 成熟服；DAU 口径=权威）
**用 DAU 口径(流水/登录人数)看，节前 5.11-5.28 vs 节日 5.29-6.7（固定服 open≤02-25,1000-1880）：**
- **真 ARPU(/DAU) +28.5%**（$1.45→$1.86）← 节日确实把 ARPU 拉回升了
- **付费率 +32.2%**（4.70%→6.22%）← 回升全靠渗透
- **ARPPU 持平 −2.8%**（$30.81→$29.95）← 深度/客单没动
- DAU −5.9%、付费人数 +24.4%、总流水 +20.9%；节日 ARPU 首销 D0 冲高($3.12)3天回落但仍高于趋势线。
- **无节日基线衰减的本质 = 付费渗透率崩**：基线真ARPU(/DAU)≈−35%，但 DAU 只−14%、付费人数−45%（人还登录、越来越多不付费），ARPPU 一直稳$30。**夏日节对症了渗透衰减，但没做深度。**
- ⚠️ **（旧版用 ARPPU 口径得过反结论"ARPU降11%/没回升"，是分母错）**：非节日流水/服 −17%、ARPPU −2.8% 仍成立（节日蚕食日常款预算、人均未提），但"ARPU 是否回升"要看 /DAU。
- **下一步**：渗透已被节日证明能拉回，缺的是深度款（高客单/可重复养成/订阅把 ARPPU 抬上去）。
- **上月基线 ARPU 构成**：无主力包，524 SKU 长尾，最大单包仅 5.9%、Top20 才 35.6%；靠常青地基款=英雄养成手册(220001 $30)/每日特惠(60001 $5,214买家)/月卡周卡(30001/30006)/转盘/登录礼/储蓄罐。要做基本盘应盯这些高买家数地基款的人均渗透，不是再加节日外显宽度款。
- 与既有结论一致（[[reference_x3_festival_performance]]：夏日深度没起来、增量全在 $4.99-9.99 breadth）。
