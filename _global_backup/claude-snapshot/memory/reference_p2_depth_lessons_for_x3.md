为 X3 第二轮节日优化（[[X3 下期节日优化待办清单]]：breadth成功但depth没起来）从 P2 偷的经验。
X3 是 P2 换皮，P2 变现更成熟、深度梯度更完整，是最近的参照系。
源文档：`C:\ADHD_agent\.cursor\p2-numerical-design\养成线深度手册.md`、`.cursor\p2-producer\P2养成线与养成线梳理.md`、`P2数据现状档案.md`（数据截至 2026-03）。

## 一、P2 收藏品线诊断 = X3 第二轮的镜子（最值钱的一段）
P2 收藏品线半年超R ARPPU $148→$84(-43%)，诊断结论**不是饱和、是"没东西刺激花钱"**（DAU只跌7%、人还在、渗透率从79%→68%）。根因结构问题，X3 几乎照搬同款病：

1. **价格断层（X3 同病）**：P2 破冰 $4.99×6(人均$14) → 核心盲盒 $78/件起，中间无 $9.99–$29.99 过渡 → 中R破冰后无下一步去处。
   → X3 现状一模一样：增量全在 $4.99 breadth，高价深度款偏弱，中间断层。**第二轮要补 $9.99/$19.99「进阶」衔接款**。
2. **高价值品类无低门槛获取路径**：P2 红色收藏品迭代到第8代但自选只覆盖前3代、抽奖池只有橙紫无红 → 中R除高价盲盒外拿不到新代红色。
   → 对 X3：高价值外显（皮肤/岛屿）只有 $99.99 直购，中R无阶梯/保底/抽奖等低门槛获取路径。
3. **订阅制是抗衰退最强形态**：P2「BP&月卡」半年仅跌28%(全线最稳)、中R≈大R≈超R分布最均衡；收藏品「保养系统」属性非永久(50%/80%/120%到期失效)= 订阅制，造出超R周期性消费。
   → 对 X3：考虑节日订阅/月卡 + 「需维护才保效果」的周期性消费机制。
4. **中R ARPPU 纹丝不动($32–35半年没变)**：中R愿意花的钱固定，**扩品类/降门槛能拿增量，提价不行**（呼应 X3 已定的"$99.99→$300保底是劝退、要保价扩坑"）。

## 二、P2 怎么把深度做出来（X3 缺的梯度）
- **队列深度梯度极陡**：中R $13,578 / 大R $27,522(2.0×) / 超R $46,228(3.4×)。X3 靠 $4.99–$99.99 太浅，缺中段和终局坑。
- **「理论无限消耗」终局机制是超R深度命根**：战装重铸(随机词条+锁定，30次/件~$5,528，完美$18,000+)、收藏品保养(订阅)。**搬养成线最容易漏搬重铸/保养这类无底洞 → 超R深度断崖**（搬运陷阱，见手册§9.3）。
- **全R覆盖深度线 = 中R扩线首选**：P2 战装寄生英雄系统、无额外解锁门槛、反馈直观、$105→$9,500 覆盖全R段，是中R扩线最佳。对 X3 选扩线方向时优先找这种"低门槛入口+长梯度+绑已有系统"的线。
- **红色套娃系统**：红英雄不是更强的橙，是 7星/同频技能/同频斗士/天赋同频/专属技能整套，超R完成率仅8.4% → 付费空间靠"系统未展开"撑着。

## 三、P2 当前大盘风险（搬经验时别只学好的）
- DAU 断崖（EDAU 4月跌21%）、小R生态崩塌、超R依赖度58%+ 又抬头 → P2 也在"量跌质升"，靠超R+活动续命。X3 学深度机制，但别复制"过度依赖超R+活动断档就崩"的结构。
- 养成线进度：英雄/战装/收藏品三线还在涨，机甲/机师/战车/兵种技能饱和或停滞 → 投放预算要压在"还有完成率空间"的线上。

## 跨项目搬运纪律（手册§9）
搬的是**比例结构 + 深度梯度，不是绝对数字**。流程：定目标队列深度→按P2各线占比分预算→定新项目经济倍率(锚点)→反推消耗量→反向校验(偏差≥30%要说明)。
P2 中R队列各线占比：机甲35.5%>装备25.9%>战车15.4%>英雄14.1%>机师9.1%（X3 按自身玩法定位调整，别照抄）。

## 四、P2 节日变现结构经验（节日活动复盘 wiki 版提炼）
源：`C:\ADHD_agent\KB\产出-数据分析\` 下 2025圣诞/2026情人节/2026春节/推币机 复盘 wiki。

1. **P2 节日 = 三层结构：外显类 + 小游戏 + 混合/养成**。小游戏是**纯增量收入、不挤压原有模块**（圣诞 30%/$276K、情人节 40%/$340K）。
   → X3 节日目前只有礼包链（特惠连锁/拜访皮肤/装饰/通行证/许愿池/抽奖），**缺"小游戏"这种独立增量玩法层** —— 第二轮可考虑引入（挖孔/大富翁/推币机式）。
2. **GACHA/随机礼包 = 深度引擎，远胜固定礼包**：推币机 GACHA 人均 $52 是固定礼包 $26 的 2 倍，随机性驱动超额复购（$19.99 档实际人均 $35=1.76次）；机甲 GACHA ARPPU $60.84 全场最高、单SKU效率最高。
   → X3 高价深度款偏弱，正缺 GACHA 驱动的深度复购。**但**：高价位段（$99.99）固定礼包人均 $303 反而高于 GACHA $174 —— 顶端大R对确定性高价包有重复购买需求，GACHA 随机性在高价位反而抑制付费。深度款要分层：中段用 GACHA 拉复购、顶端保留确定性高价包。
3. **「宽口浅底」是 X3 第二轮的精确病名**：推币机付费率全场最高(11%)但 ARPPU 最低($27.5)，中额→中高额衰减 21.8%(恶化13.6pp)是漏斗最严重环节。X3 现状（breadth成功/depth没起来）就是同款"宽口浅底"。**重点修中额→中高额的转化断点**。
4. **节日 4 阶段排期模板（错开大付费点，直接喂 [[X3 下期节日优化待办清单]] 模块2投放循环）**：
   - 阶段1(D1-3) BP/GACHA/预购/大富翁 → 阶段2(D4-10) 小游戏1 → 阶段3(D11-17) 皮肤/小游戏2 → 阶段4(D18-24) 冲榜/小游戏/优惠礼包补付费。
5. **皮肤单抽付费力 < 机甲/带养成的 GACHA**：圣诞斗士皮肤付费占比 45% vs 机甲 70%，100刀后转化弱于不放回抽奖；行军/斗士纯外显衰减明显。外显尽量绑养成价值或做不放回抽奖。
6. **情感向高价外显对高R极有效**：情人节送花礼包 ARPPU $165.93（情感消费标杆）。节日主题强情感款是高R深度抓手。
7. **诊断口径对齐**：ARPPU 偏低 = 高价款吸引力/付费深度不足（春节、推币机反复命中此结论）；低价礼包(≤$30)占人数 80%+ 是正常引流，但鸡肋礼包(人少+收入低)要砍；新增内容通常对全 R 级 ARPU 同比都拉（圣诞超R+61%/中R+75%）。
