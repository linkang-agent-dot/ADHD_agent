# X2 两节日开局窗口收入对比方法论（2026-06-22 占星vs拓荒沉淀）

做「为啥这期节日比上期少收入 / 差在哪几个模块 / 还能不能追平」这类对比时套这套。数据源 Datain `v1089.dl_user_order o` LEFT JOIN `v1089.dim_iap i`，用 `C:\ADHD_agent\skills\ai-to-sql\scripts\_datain_api.py::execute_sql(sql,'TRINO_HF')`。

## 口径对齐三铁律（不对齐=结论全错）
1. **服务器必须可比**：先 `count(distinct o.server_id)` 数两期活跃出单服。只有服数一致才公平（本次两期都=30服才敢直接比）。不一致要么限定同一批服、要么明说规模因素。
2. **窗口对齐用 D0–D9 完整10天**：最新节日的 D10=今天当天数据不全，剔除；两边都取 D0–D9。`Dn=(partition_date-D0).days`。
3. **模块归类靠名字关键词、不靠 id 段也不靠"谁家名字"**：⚠️X2 节日 BP/GACHA/七日/抽奖券/掉落转付费**复用旧 id**，dim_iap 名字常是旧节日的（拓荒 BP/GACHA 名字写"2025占星节"也有）→**按模块语义归类，只剔真·往期残留**（名字含`夏日节/万圣节/2024`的才是别的活动尾巴）。

## 节日收入判定口径（FEST，本次实测全覆盖）
```sql
FEST = (i.iap_type IN ('混合-节日活动','活动礼包')   -- 抓复用旧id的BP/GACHA/大富翁/七日
   OR substr(o.iap_id,1,7)='2013920'                 -- 当期节日新段
   OR substr(o.iap_id,1,8)='20135001'                 -- 强消耗抽奖券复用槽
   OR substr(o.iap_id,1,8)='20132302')                -- 挖矿关卡scene礼包
```

## 模块 classify（名字关键词，剔往期残留优先）
往期残留(夏日节/万圣节/2024)→剔。挖矿(名字含挖矿 or id 20132302)→有"翻倍/特权"=挖矿特权,否则挖矿关卡礼包。其余:GACHA / 大富翁(大富翁|骰子|存钱罐) / BP通行证(通行证|节日活动BP|bp礼包|BP集结) / 七日活动 / 自选周卡 / 每日礼包 / 掉落转付费 / 强消耗抽奖券 / 限时抢购 / 行军表情 / 巨猿砸蛋(砸蛋) / 装饰礼包(装饰) / 连锁&节日礼包(自选卡包|连锁|礼包) / 其他。

## 缺口拆解 = 广度 × 深度（核心框架）
**模块收入 = 付费人数(广度) × ARPPU(深度)**，每个缺口模块判是哪个掉的：
- 模块付费人数必须 SQL 层 `count(distinct user_id)` 按模块去重（拉 user×iap 明细 python 归类后对模块求 distinct，**不能按 iap 的 buyers 相加**=重复计人）。
- ARPPU=模块收入/模块去重付费人数；渗透率=模块付费人数/**节日总付费人数**(不是DAC)。
- 判据：付费人数掉>25%=广度↓(进场人少)；ARPPU掉>35%且大于广度降幅=深度↓(档位配低)。

## 付费率分母坑
DAC(活跃)做分母两期**不可比**：`ods_user_daily` 全服/节日服累计活跃都因**期间合服+老服衰减**大幅变化(本次占星节日服10天累计活跃74k→拓荒32k,纯口径噪音)。**用「模块付费人数/节日总付费人数」作渗透率**最干净。

## 追平预测算法
缺口÷剩余天数=**所需日均**;对比**实际近期日均**(取尾段D7-D9均值,已进衰减区,最代表"今天水平")。节日尾段只衰减不反弹→所需日均若>实际1.5x基本判"追不平"。**还要看逐日付费人数**:塌方=结构性枯竭,延长天数补不回。

## 本次结论（拓荒 vs 占星 D0–D9，30服，2026-06-22）
占星 $25,733/233付费人 vs 拓荒 $17,288/193付费人，缺口 **$8.4k**，78%集中在:大富翁−$3.1k(广度崩 渗透57%→37%,骰子产出优化疑未生效)+GACHA−$2.8k(老皮肤复购乏力 广度−31%)+连锁礼包−$1.1k(深度腰斩 ARPPU$36→$17)。**几乎全是广度(付费人数)掉,不是变抠**(ARPPU普遍持平)。
**根因=付费人数塌方**:拓荒逐日付费人 D0 76→D5 31→D9 20→D10 17,开局D0就差40%($6964 vs $4185)。节日尾段自然枯竭,时间补不回。再跑4天(实际日均已掉~$850-1000且在跌)约+$3.6k→~$20.9k,**追不平**(需日均$2111),仍差~$4.8k。延长只能把-$8.4k收窄到~-$4.8k,翻盘只能靠下期做起开局拉新+大富翁/GACHA参与广度。
