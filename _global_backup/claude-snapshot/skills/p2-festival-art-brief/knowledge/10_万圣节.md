> **重要节点：** 2021-2024 统一叫"祖灵节（万圣节）"——早期用"祖灵"作为规避宗教敏感的外层包装（对应文案背景："追悼祖先、祭祀祖灵"），西方万圣元素（南瓜/幽灵）作为视觉实现。**2025 起正式改名为"万圣节"**，手札内部道具描述仍保留"祖灵手札 / 祖灵积分 / 祖灵节装饰组成部件"的字样，新旧命名并存。

## 二、节日定位

- **核心概念**: 西方万圣节氛围做 P2 本地化——南瓜/幽灵/女巫/墓碑具象堆叠，但控制恐怖尺度（卡通抽象化，不要写实、不要吓人）。
- **情绪关键词**: 神秘 · 诡趣 · 恶作剧 · 暗夜氛围 · 糖果惊喜 · 亡灵戏谑
- **节日身份**: **手札循环模板节日**（2025 年版本标注定型）——手札装饰为主产出模块，走"主装饰 + 子模块 1-5"的收集拼装路径。
- **与其它循环节日的锚定关系**: 万圣节（手札循环） · 科技节（套装循环，主城 LV1-6） · 春节（行军特效套装循环） · 情人节（机甲皮肤循环） · 圣诞节（斗士皮肤循环）。**万圣节独立走"手札循环"这条线**，不与主城 LV1-6 套装争赛道。

## 四、模块拓扑（2025 手札循环模板定型）

### A. 主城皮肤（传说 + 超凡双档）

| 档次 | 功能定位 | 美需侧重 |
|------|---------|----------|
| 主城传说皮肤 | 基础形象（低级版）| 主体造型，可简单留升级空间 |
| 主城超凡皮肤 | 升级版（高级版）| 在低级版基础上换色/加特效/拟人化/动态演出 |

**低→高进化公式**（2024-2025 稳定）:
- 2024 "恶作剧先生 → 南瓜绅士"：静态南瓜 → 伸出机械腿踏步+礼帽跳动+张嘴掉糖
- 2025 "怪物小屋 → 怪物之家"：卡通怪物房子 → 整体换色+加发光机械结构+嘴巴张合+伸缩长舌+左右晃动

**双方向备选惯例（2025 建立）**: 每个皮肤给 ≥2 方向，示例："方案一：怪物房子带大嘴爪子 / 方案二：尖帽子卡通小幽灵 + 底座托起"。

### B. 主城特效（围绕主城底周的光效）

- 常态：暗绿色漩涡+鬼火环绕 / 紫黑色光雾+蝙蝠群环绕 / 扫把+女巫帽环绕
- 动态：地面冒出卡通鬼手/墓碑，做抓握动作后缩回
- **关键避坑**：(1) 不滥用南瓜（要兼容其它皮肤） (2) 不与行军特效混淆 (3) 考虑把主城底部"托起略浮空"以兼容所有皮肤

### C. 手札装饰系统（核心产出模块）★

**这是万圣节区别于其它节日的核心结构**。分两层：

#### C.1 主装饰 LV1-3（进化级别）

| 等级 | 名称样本（2025）| 美需演进 |
|------|----------------|---------|
| 装饰升级-1 | 尖叫树 | 树+树下 2 个南瓜+糖果篮子；树的嘴巴和眼睛发光 |
| 装饰升级-2 | (同上) | 新增组件 4：树上挂彩灯（灯带或挂灯，华丽些）|
| 装饰升级-3 | (同上) | 新增组件 5：树枝上卡通小幽灵（大而显眼，避免吓人）|

> **LV1-3 本质是主装饰逐级加挂件**，每升一级叠加 1-2 个子模块可见物；与科技节"三级换材质+特效"不同，万圣节走**"拼装可视化"路线**。

#### C.2 子模块 1-5（可独立收集的拼装件）

| 编号 | 道具名（2025）| 美需 | 描述 |
|------|--------------|------|------|
| 子模块 1 | 仿真枯树 | 枯树 | 祖灵节装饰的组成部件之 1 |
| 子模块 2 | 怪笑南瓜 | 南瓜 | 祖灵节装饰的组成部件之 2 |
| 子模块 3 | 寂寞墓碑 | 墓碑 | 祖灵节装饰的组成部件之 3 |
| 子模块 4 | 鬼脸图案 | 鬼脸 | 祖灵节装饰的组成部件之 4 |
| 子模块 5 | 彩灯挂饰 | 彩灯 | 祖灵节装饰的组成部件之 5 |

**子模块命名公式**: `[形容词/修饰] + [具象元素]`，如：仿真枯树 / 怪笑南瓜 / 寂寞墓碑 / 鬼脸图案 / 彩灯挂饰。

**手札成就道具**: `祖灵手札` (UI 名) · `祖灵积分` (道具名)，即使 2025 改名万圣节，内部道具仍沿用"祖灵"。

## 七、美需描述范式（从真实 2024-2025 抽提）

**结构**: `[核心主体] + [形态细节] + [底座/支撑判定] + [动态效果] + [配色] + [≥1 方向备选] + [避坑提示（控制恐怖尺度）]`

**示范 · 主城低级（2024 恶作剧先生）**:
```
1.机械主题、南瓜形状的近圆形建筑，做一定拟人化处理；
2.建筑可以是纯金属材质，也可在橙色涂漆南瓜的基础上添加机械瓜蒂（可用金属管道）、
  机械眼眶、机械鼻子（螺帽或压力表等）、机械门（嘴巴）等金属元素，
  可辅以齿轮、压力表、发条等蒸汽朋克元素表现；
3.底部有个简易台子或台阶即可，区别高级皮肤的机械腿。
```

**示范 · 主城高级动态（2024 南瓜绅士）**:
```
高级版：
1.在初级南瓜建筑基础上，底部增加较细的机械腿（看实际的视觉协调性决定是否添加机械手臂），
  可做动作伸腿撑起南瓜身体，原地踏步几下跳跳舞，腿再缩回去；
2.可添加更多拟人元素，如礼帽、眼镜、胡子等，做动作时礼帽可在头上跳动；
3.如需增加更多复杂度，可选（非必要）：张开嘴巴，腔内显露出一堆糖果，有少量糖果掉出来。
```

**示范 · 主城特效多方向（2024 暗影之主）**:
```
1.围绕主城建筑底周的一圈的动态特效，可呈旋转动态，也可是有节奏的放射性特效；
  特效主要在地表生成，特效圈内部边缘感清晰，不和主城皮肤自带特效冲突。
2.考虑到祖灵节（万圣节）投放，本次以相关元素为主题进行制作，
  比如：被紫黑色光雾笼罩的蝙蝠群、扫把和女巫帽子环绕主城等。
3.注意与行军特效需要有足够区分；
4.与万圣节主城皮肤本身协调；
5.考虑到和其它主城皮肤尽可能兼容，不滥用南瓜元素；
6.考虑蝙蝠群特效把主城底部托起来，略微浮空的表现效果，兼容所有皮肤。
```

**示范 · 行军双档对比（2025）**:
```
低级版（糖果魔法）：
底部流光：橙色+黑色流光，尾部有缓慢转动的魔法阵图案
  （或魔法阵不转动，做若隐若现效果，注重装饰性，避免像技能特效）。
拖尾装饰：几只小幽灵从前向后飘，钻入魔法阵中消失，魔法阵里飞出糖果。

高级版（南瓜恶作剧）：
底部流光：橙色+黑色流光，尾部有缓慢转动的魔法阵图案。
拖尾装饰：小幽灵从前向后飘，钻入魔法阵中，魔法阵上冒出南瓜虚影，
  然后炸开、飞出糖果。
前部光环：环绕的蝙蝠，身后有黑色或暗紫色烟雾轨迹（颜色可根据特效效果调整）。
```

**示范 · 手札主装饰 LV1-3 递进（2025 尖叫树）**:
```
LV1 组件 1-3：树 + 树下 2 个南瓜 + 糖果篮子；树的嘴巴和眼睛可以做发光特效。
LV2 组件 4：树上挂的彩灯装饰（灯带或挂灯，需要华丽一些）。
LV3 组件 5：树枝上的小幽灵（大而显眼，偏抽象卡通风格，避免吓人）。
```

**必备要素**:
1. 特征陈列 ≥2 点（编号或分号）
2. 底座/支撑物处理说明（拟人化主城必填）
3. 动态/特效描述（高级主城/主城特效/行军高级必填）
4. 多方向备选（2024 起成为惯例：南瓜门 vs 南瓜塔、暗色雾气黑袍 vs 南瓜灯滚动、怪物房子 vs 小幽灵）
5. **恐怖尺度控制**（万圣节特有必填）：卡通风格 / 不要写实 / 不要吓人 / 控制恐怖尺度
6. 兼容性说明（主城特效类必填：不滥用南瓜、兼容其它皮肤）
