> **命名约定**: 大部分年份对外叫"圣诞节（冬日节）"，"冬日节"作为去宗教化/温和化的备用名；2025 正式回归直接使用"圣诞节"。
>
> 从 2025 起圣诞节定义为 P2 的**第四个大循环节日**（"**斗士皮肤循环模板**"）—— 主产出从主城皮肤切换到斗士皮肤（玩家战场角色的皮肤/机甲皮肤斗士款）。

## 二、节日定位

- **核心概念**: 全球最强国民节日情绪锚点 —— 温暖 / 团聚 / 仪式感 / 送礼。猿族世界观吸收圣诞符号，但人→猴替换（圣诞老猴/姜饼猴/猴天使）。
- **情绪关键词**: 温暖 · 雪夜 · 铃铛 · 驯鹿 · 姜饼 · 童话 · 礼物 · 壁炉 · 红绿配
- **节日身份**（2025 起）: **斗士皮肤循环节日** —— 斗士（玩家战场角色）皮肤是主产出，配套抽奖活动（满月/新月许愿池）走斗士皮肤 gacha
- **与其它节日的锚定关系**:
  - 主城皮肤循环 → 科技节（3 月）
  - 行军特效套装循环 → 春节（2 月）
  - 机甲皮肤循环 → 情人节（2 月）
  - **斗士皮肤循环 → 圣诞节（12 月）** ← 本节日

| 材质池 | 翻糖 / 饼干 / 巧克力 / 姜饼 / 棉绒 / 磨砂玻璃雪景球 / 彩色冰晶 / 金属铃铛 |

## 七、美需描述范式（从真实 2025 抽提）

**结构**: `[品质级别] + [核心主体] + [结构细节分点] + [动态效果] + [配色] + [≥1 方向备选] + [避坑提示]`

**示范 · 2 级高级主城（童话树屋）**:
```
品质：2级
大型圣诞树屋，可增加复杂的房屋结构，
树屋外有一只麋鹿拉着雪橇转圈。

注意：需要凸显麋鹿，体型可适当放大。

体积：大 / 装饰：华丽 / 颜色：原色（绿色）
底部环绕：麋鹿+空雪橇
```

**示范 · 3 级璀璨染色（冰雪不夜城，动态编排）**:
```
品质：3级
1、彩色冰晶质感的圣诞树。
2、树的中段或底部镂空，里面有圣诞风格内容物（可考虑旋转）。
3、一只炫彩麋鹿拉着雪橇从树底绕圈飞上树顶，
   雪橇里坐着圣诞猴，身后有类似极光的拖尾效果。
4、麋鹿飞上树顶后消失，落下很多礼物（特效），
   间隔一定时间再次出现在树底（考虑做成动画，循环播放）。

体积：大，且比2级皮肤更高
装饰：华丽 / 颜色：染色（炫彩）
飞空环绕：麋鹿+载有圣诞猴的雪橇
```

**示范 · 斗士皮肤（多方向）**:
```
方向一：圣诞老人风格服装，可以有麋鹿角头饰
方向二：圣诞锡兵风格服装

斗士载具：
方向一：雪地车，比较适配游戏风格，考虑改成红绿配色，增加圣诞元素装饰
方向二：传统圣诞雪橇，比较经典，有差异感
```

**示范 · 行军特效（冰翼+拖尾）**:
```
前部：行军部队上方悬浮的冰翼，在开始行军时展开，
     行军过程中有拍打翅膀的动态。

拖尾：行军走过的地面冰封，地面上绽开朵朵冰花/冰晶簇，
     然后出现碎冰效果（慢速版）。

注意：冰翼的材质尽量半透明，不要太凝实，
     避免遮挡行军部队。
```

**示范 · 手札装饰（材质导向）**:
```
组件1：糖果轮子
组件2：饼干车厢
组件3：车厢四角的拐棍糖饰
组件4：车厢里装的圣诞礼物盒和糖果
组件5：车厢上坐的姜饼人（考虑做成猴子形）

（材质偏饼干一点）
```

**必备要素**:
1. 特征陈列 ≥2 点（编号或分号）
2. 底座/支撑物/环绕物处理说明（主城、装饰物类必填）
3. 动态/特效描述（2 级及以上必填）
4. 多方向备选 + 标注"运营倾向"（斗士/大富翁类必给 2 方向）
5. 材质强调（手札糖果类必须标注材质倾向）
6. 避坑提示（列明不要做什么 / 不要太像真食物 / 不要遮挡部队 / 不要与去年圣诞小屋雷同）
