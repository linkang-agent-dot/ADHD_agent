### 题材换挡编年

6 月档期是 P2 节日里**题材跳跃最剧烈**的一个档口，经历了两次明显的换挡：

1. **2022 幼猴节（废弃线）**：主题是"童趣大冒险"，奖励叫"梦幻乐园""童心永存"。只有文案骨架、美需列几乎全空，连主城皮肤的视觉方向都未落到字。可以理解为早期摸索期的 placeholder 年份。
2. **2023–2024 龙舟节（中国端午传统）**：两年锁定端午节文化母题——龙舟、粽子、鼓槌、船桨、粽叶。2023 还很克制（石质池塘+小船），2024 直接拉满——主城皮肤做成"**龙舟驼着粽子形状的房子**"、手札装饰是**龙舟鼓+巨型假粽子**、行军皮肤"粽情高歌"。龙舟鼓、粽叶、龙头龙尾是核心元素库。与春节"龙年皮肤"、去年端午皮肤需要主动做形态区分（2024 明确写入避坑）。
3. **2025 深海探险节（第一次海洋化）**：彻底脱离中式传统，锁定"鲸鱼潜艇"为主城核心。LV1–LV6 全是海洋仿生机械——鲸鱼潜艇→刺豚浮球→鳐鱼发电机→水龙卷/海浪→深海漫游者→长鲸出海。手札装饰改成"礁石底座+海星+珊瑚丛+贝壳+海马"五件套。挖矿场景则是西部元素（矿车、酒桶、油灯、牛仔帽）——和深海主线并存但路线分化。
4. **2026 深海节（海怪/沉船/大富翁融合）**：延续深海世界观但玩法更杂揉——林康的主城皮肤是**海盗船+幽灵船+海怪触手拖船**；龚亮线的主城皮肤是**大富翁主题**（猿猴穿西装、积木城堡、大富翁棋盘底座）。机甲染色是"海滩悍将"（蓝黄+沙滩涂鸦）。大富翁装饰物是戴海盗帽的章鱼玩偶+沙堡 LV1-3。跨题材并行度明显上升。

> **档期记忆要点**：龙舟/粽子元素（2023–2024）已连续两年饱和使用，目前暂别；2025 起确立"海洋/深海"为 6 月长期方向；2026 在深海基础上引入"海盗幽灵船+大富翁沙滩"两条子线。

## 二、节日定位

- **核心概念**: 脱离猿族世界观，借助真实世界"深海/海洋冒险"符号，营造打破次元壁的探索感。2026 进一步引入**海盗/幽灵/沉船**的奇幻色彩，叠加**沙滩度假+大富翁**的轻趣味副线。
- **情绪关键词**: 蔚蓝 · 探险 · 神秘 · 奇幻 · 漂流 · 潜入 · 寻宝 · 清凉
- **节日身份**: **题材换挡节日**（2025 才从端午龙舟切到深海探险，尚未定型为循环节日）。当前处于"建立深海符号库"阶段，美需倾向多方向试错（幽灵船 vs 大富翁 vs 沙滩度假）。
- **与其它节日的锚定关系**:
  - 与 **7 月沙滩节（仲夏节）** 主题相邻（同属夏日水系），需做元素错位——沙滩节主打沙滩/冲浪/椰子树，深海节主打潜入/海怪/沉船。
  - 与春节龙年皮肤、2024 端午龙舟皮肤需做**形态区分**（2024 已有避坑记录）。

## 四、模块拓扑（以 2025/2026 深海方向为准）

### A. 主城套装 LV1-6（2025 建立的深海仿生体系）

| 等级 | 功能定位 | 2025 深海样本 |
|------|---------|---------------|
| LV1 | 低级城堡（基础形象）| 蔚蓝鲸歌：鲸鱼造型潜水艇，仿生风格、机械感弱 |
| LV2 | 组件 1 | 刺豚浮球：卡通鱼形鱼雷/浮球贴在潜艇两侧 |
| LV3 | 组件 2 | 鳐鱼发电机：仿深海生物飞行器 |
| LV4 | 光环/高级特效 | 白浪鱼影：方案一水龙卷鱼群；方案二翻涌海浪+海星贝壳热带鱼+海鸥 |

### B. 主城皮肤（2026 双线并行）

- **gongliang 线（大富翁主题）**：传说（低级，像素方块/乐高大富翁）→ 超凡（高级，西装猿+积木城堡+大富翁棋盘底座，动态摇钱袋+灯闪烁+骰子跳动）→ 璀璨（玩偶猩猩搭积木、推倒、礼花喷射）
- **林康线（海盗幽灵船主题）**：传说（海盗船+猿猴旗+水面微动）→ 超凡"幽灵航船"（海盗船+海怪触手拖船，动态下沉/浮起循环）
- **主城特效（刘思毅）**：幽灵鲨鱼环绕船+水龙卷+炸开

### C. 手札装饰（2025 深海童话五件套）

- 组件 1：礁石底座
- 组件 2：围成一圈的海星（强调卡通装饰感，避免像活物）
- 组件 3：珊瑚丛
- 组件 4：珊瑚丛上张开的贝壳（卡通风格）
- 组件 5：贝壳中托着的海洋生物（海马 / 小丑鱼 / 刺豚 / 海星任选）
- 命名公式走"礁石底座 / 奇趣海星 / 珊瑚丛 / 珠光贝壳 / 黄金海马"

### D. 行军皮肤/特效组

- **2025 行军皮肤**（"海洋之梦" / "波光巡礼"）：拖尾用深蓝/浅蓝/紫色水流流光，尾部海豚跳跃+浪花；高级版加前部光环+小鱼/海星装饰
- **2026 行军皮肤组件**（虚空鲸歌）：两套方案并列——方案一鲸/鲲虚影半透明跃起+星空海洋流光+波纹光点；方案二幽灵鲨鱼前冲撕咬+墨绿/浅绿海浪+水藻破舵轮浮木
- **2026 低级行军特效**：蓝+黄水流流光，向后漂小黄鸭、游泳圈
- **前部破风**：两边排开海浪的破浪效果（方案一和二都保留）

### E. 机甲染色（跨年度延续）

- 2024 端午发生在"惊喜涂装"活动下的机甲染色（废土金属撬棍道具）
- 2025 螳螂机甲皮肤（参文档）
- 2026 金刚机甲染色"海滩悍将"：蓝黄渐变+沙滩海洋涂鸦（小鱼/海浪/椰子树图案）

### F. 节日装饰物 / 大富翁装饰物

- 2026 大富翁装饰物-1：戴海盗帽的章鱼玩偶，卡通化，底部加小滩水避免落草地
- 2026 装饰物-2 LV1-3（海边贩卖店主题）：沙堡 LV1（带小旗）→ LV2（加海星贝壳）→ LV3（加泳圈/阳伞/小桶小铲子）

### G. 行军表情（跨节日专用）

- 2026 深海节表情：猴子生气破防，举手机（或电子仪器替代），将手机掰断，可加嘴巴喷火——与深海主题关联度低，属于跨节日复用的情绪表情池。

## 七、美需描述范式（从真实 2025/2026 抽提）

**结构**: `[核心主体] + [形态细节] + [底座/支撑判定] + [动态效果] + [配色/材质] + [≥1 方向备选] + [避坑提示]`

**示范 · LV6 终极形态（动态编排，2025 长鲸出海）**:
```
外显动态特效：
1. 鲸鱼潜艇沉在水下，只露出上层舱室。
2. 水面冒泡，从舱室里伸出潜望镜观察四周。
3. 水花涌动，潜艇浮出水面，摇晃，排水管喷出水汽（模拟鲸鱼）。

增加动态：翅膀挥动动态，可伴随水波、浪花效果。
增加特效：配合鲸鱼潜艇上浮、喷水动作，出现彩虹，并有光点洒落。
```

**示范 · 主城皮肤（动态多阶段，2026 幽灵航船）**:
```
更华丽的海盗船或幽灵船，船身上缠绕着几根海怪触手。

动态：触手伸出水面，将一半船身拖入水中，海盗船剧烈晃动，下沉再浮起，间隔一定时间循环。

（倾向幽灵船）
```

**示范 · 行军皮肤（多方向备选，2026 虚空鲸歌）**:
```
方案一：
1. 主体：巨大的鲲/鲸鱼虚影从行军特效底部流光中缓慢跃起，在部队上方挥舞翅鳍停留数秒，化为星光/水花/碎片消失。整体为半透明质感。
2. 流光：类似星空海洋的梦幻感流光，行军过程中泛开波纹、飘出光点。
3. 前部破风：向两边排开海浪的破浪效果。

方案二：
主体：巨大的幽灵鲨鱼出现在行军部队上方，猛地前冲并张大嘴做撕咬动作，游动片刻，消失。
底部流光：墨绿+浅绿色的海浪，向后漂过水藻、破舵轮、浮木等海洋/航海元素。
前部破风：向两边排开海浪的破浪效果。
```

**示范 · 装饰物（底座处理 + 避坑，2026 大富翁-1）**:
```
戴着海盗帽子的章鱼玩偶，需要卡通化处理，不要太写实。
（底部可以加一小滩水，避免直接放在草地上）
```

**示范 · LV4 光环（多方向备选，2025 白浪鱼影）**:
```
方案一：
水流和鱼群旋转、环绕向上，形成水龙卷，再炸成水花散开。

方案二：
一层层向外翻涌、拍击的海浪，从内层不断飞出海星、贝壳、热带鱼、小螃蟹等元素，几只海鸥在海浪上方飞行环绕。
```

**必备要素**:
1. 特征陈列 ≥2 点（编号或分号）
2. 底座/支撑物处理说明（装饰物类必填——沙滩装饰底部加水）
3. 动态/特效描述（LV4–6 必填，深海节尤其重水流/冒泡/上浮等动效）
4. 多方向备选 + 标注"运营倾向"（例：倾向幽灵船、倾向方案二）
5. 避坑提示（海洋生物卡通化、不要写实恐怖、不要与龙年/端午撞形）

- **美人鱼王国**: 珍珠王座+海螺乐器+水草长发+半身鱼尾+贝壳宫殿

### 新美需写作 Checklist
- [ ] 主题与 2025 鲸鱼潜艇体系差异 ≥60%（题材刚换，不要重复鲸鱼）
- [ ] 与 2024 龙舟 / 春节龙年做形态区分（官方已有避坑记录）
- [ ] 命名主词 4 字、描述 10-14 字
- [ ] LV1-6 命名有进阶感（低级基础 → 高级华丽 → 终极动态）
- [ ] 每个装饰/模型给出底座方案（沙滩类必加水）
- [ ] LV4-6 提供动态描述（冒泡/上浮/摇晃/喷水/彩虹等深海专属动效）
- [ ] 至少一处给 ≥2 方向备选（标注"倾向 xx"）
- [ ] 明确避坑项（卡通化、非写实、海洋生物不像活物）
- [ ] 配色明确（主蓝+辅金/珍珠白/墨绿紫）
