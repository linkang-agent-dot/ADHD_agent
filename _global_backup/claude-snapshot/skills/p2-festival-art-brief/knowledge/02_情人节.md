### 名称演变（2023→2024 关键节点）

- **2022-2023 = 情猿节**：沿用"情猿"这个猿族自嘲词（"情猿"谐音情缘，延续猿族世界观），文案里出现"情猿魅力""有情猿""情猿门"。
- **2024 起 = 情人节**：改回真实世界节日命名，品牌对齐现代玩家认知；但 2024 文案里仍有"情猿节特别矿藏""情猿花语"作为过渡，2025/2026 彻底换成"情人节"。
- 演变逻辑：与科技节"跳出猿族世界观 · 打破次元壁"的定位一致——情人节也从猿族内部梗走向"玩家现实生活节"。

> **2026 定型**：情人节正式定义为"**机甲皮肤循环节日**"——主城皮肤每年换一套，走"石质+部分机械材质"的机甲风主城（区别于科技节纯机甲坦克/战舰）。主城皮肤是本节日最高产出优先级。

## 二、节日定位

- **核心概念**：爱情 · 浪漫 · 双人 · 仪式感；2026 起叠加"机甲主城皮肤"——用机甲材质去承载情人节符号（天使/魔王/爱心），做出"硬核浪漫"。
- **情绪关键词**：甜蜜 · 心动 · 告白 · 永恒 · 缘分 · 羁绊 · 浪漫 · 仪式
- **节日身份**：机甲皮肤循环节日（主城皮肤每年一套，2025 起稳定投放在皮肤 gacha 日排行榜），同时带装饰物循环、行军特效/皮肤循环。
- **与其它节日的锚定关系**：情人节与科技节（套装循环）、春节（行军特效套装循环）并列为三大"**循环节日**"。
- **差异化底线**：
  - 情人节的机甲 ≠ 科技节的机甲：情人节用机甲材质包装情人节符号（天使、恶魔、爱心、锁钥、天鹅），氛围华丽+浪漫；科技节是裸机甲+坦克+战舰+都市，氛围冷硬+未来。
  - 主城皮肤与春节、科技节的主城皮肤必须有可辨识的视觉断档（配色/符号/形态）。

## 四、模块拓扑（当前标准，2026 定型）

### A. 主城皮肤（机甲皮肤循环主产出）

2025 起稳定 4-5 款一组（普通/升级/特殊男/特殊女/合并特效）；2026 扩展为"羁绊 + 传说 + 超凡"三档结构，加一个羁绊光环。

| 功能位 | 模型特征 | 美需侧重 |
|--------|---------|----------|
| 普通皮肤（低级）| 2.5 级模型，基础形象 | 投放在皮肤 gacha 日排行榜，主体造型干净、辨识度高 |
| 升级版皮肤（高级）| 基于低级做材质/特效升级 | 华丽度提升，自带动态效果 |
| 特殊男/女版 | 男女成对，对称镜像造型 | 共 2 个 SKU，主体一致镜面翻转，颜色差异化 |
| 合并特效 | 男女主城靠近时触发 | 两主城相互发射特效在中间汇合（爱心相撞/烟花/能量对冲）|
| 超凡羁绊皮肤（2026 新）| 双皮肤成对，触发羁绊时有光环特效 | 2.5 级，光明/暗黑对偶结构 |
| 传说主城皮肤（2026 新）| 1 级低配版 | 方案一有天使像，方案二仅结构装饰 |
| 超凡主城皮肤（2026 新）| 2 级 | 传说款的华丽升级版 |

### B. 主城特效（累充高级产出）

2025 起固定配：底部光环 + 显隐效果（蝴蝶/漩涡/光束）。

### C. 行军特效（BP/付费产出）

- 行军特效1（高级品质）：BP 高档位 / 付费（2026 命名"审判之箭"）
- 行军特效2（普通品质）：BP 低档位（2026 命名"惩戒流光"，去掉箭本体仅保留拖尾）
- 2024-2025 叫"行军皮肤"（2024 玫瑰誓约 / 2025 世纪婚礼），2026 改叫"行军特效"（审判之箭/惩戒流光）

### D. 装饰物升级 1-3（三级进化）

三级进化，每级在前一级基础上 (1) 增加结构复杂度 (2) 增加材质/装饰元素 (3) 加水流/动态特效。
- 2024：玫瑰花门（从花门→加路引→加气球 LOVE 字样）
- 2025：告白气球（爱心气球→大热气球包裹→爱心形大热气球+毛绒熊）
- 2026：流水花庭 / 水池（希腊水榭/小水池→立柱+水流动态→丘比特猴倾倒陶罐）

### E. 头像框

- 2025 起固定配"成对头像框"：`A 名 × B 名`，描述成对仗句（比翼连心 / 永结同心；曦光之羽 / 暮光血翼）。
- 2026 主产出加一个动态头像框（玫瑰心锁）。

### F. 铭牌 & 聊天框

- 铭牌：`情猿花语`（2024）→ `甜蜜心情`（2025）→ `真爱羽翼`（2026）
- 聊天气泡：`情猿魅力`（2022-2023 老资源，可复用）

### G. 活动层

- **主 gacha 活动**（抽主城皮肤；2025 起改成"大小转盘"形式）——通常复用"心愿水晶"
- **节日 BP**（2024 起）——通常复用"甜蜜之约"+ 付费 BP
- **BINGO 活跃活动**（2024 起）——复用"巧克力集锦"
- **组队 BP**（2026 起脱离情人节风，走"光与暗/天使魔鬼/冰与火"魔幻路线）
- **送花活动**（2025 起，AI 可包装）：玩家赠送鲜花累计积分/排行榜。命名：`花语传情`（2025）/ `繁花盛典`（2026）。道具：玫瑰花束/红玫瑰/蓝玫瑰+心意值积分。
- **行军贬值活动-一番赏**（2026 新）：`行军幸运抽 / 幸运环游店`
- **沙滩寻宝换皮**（2026 复用伙伴节玩法：`云上探宝`/云上探测仪/流星碎片）
- **签到 / 掉落 / 活动道具 A/B/C**（固定配置）

### 命名公式

- **主城皮肤名**：`[形容词/意象] + [建筑/器物主体]`，四字为主。
  - 意象池：情锁 / 真爱 / 爱情 / 光明 / 光辉 / 暗炎 / 天使 / 神殿 / 魔城
  - 主体池：心扉 / 密钥 / 灵药 / 魔咒 / 神殿 / 圣城 / 圣殿 / 魔城 / 工厂 / 小屋
- **主城皮肤描述**：2 句结构，第一句画面（"以守护之名…" / "天使降临之处…" / "光辉自羽翼上洒落…"），第二句情绪/哲理（"筑造信仰之阶" / "即是永恒的圣所" / "荡涤一切尘埃"）。
- **行军特效**：`[动作/武器意象] + [主体名]`（审判之箭 / 惩戒流光 / 玫瑰誓约 / 世纪婚礼）。
- **头像框成对**：对仗两词（比翼连心 × 永结同心；曦光之羽 × 暮光血翼），A/B 两边字数一致、意象对立但情绪统一。
- **装饰系列**：`[材质/意象] + [空间/器物]`（流水花庭 / 告白气球 / 情猿门）。
- **情绪词高频池**（各模块均适用）：甜蜜 / 心动 / 告白 / 永恒 / 缘分 / 羁绊 / 浪漫 / 真爱 / 比翼 / 同心 / 相印 / 誓约 / 圆满 / 相伴 / 共舞。

## 七、美需描述范式（从真实 2026 抽提）

**结构**：`[风格定位/等级] + [主体造型] + [局部构件] + [材质判定] + [动态效果] + [≥1 方向备选] + [男女/光暗差异化] + [避坑提示]`

**示范 · 主城皮肤（2026 暗炎魔城 · 超凡羁绊-1）**：
```
2.5级
1.魔法塔/魔王城风格的建筑，黑红色为主
2.上方有黑暗天使/恶魔像，偏石质+部分机械材质

动态：
恶魔做出施法动作，整座建筑燃烧起来
```

**示范 · 主城皮肤升级款（2025 真爱密钥）**：
```
主城外观：
1.在低级皮肤的基础上，心锁的锁把顶部增加闪亮的心形红宝石，使之变成戒指（如图3，锁把——>宝石戒指）。
2.钥匙不再挂在心锁旁，而是插在锁孔里（如图2）。整体华丽度提升。

自带效果：插在锁孔里的钥匙转动，窗户和锁孔内亮起粉红光效，房门打开，
飞出大量情书、巧克力和花瓣（可以做成气流，向上盘旋消失）。

注意：整体颜色要和特殊主城颜色有区别。
```

**示范 · 男女情侣款差异化（2025 爱情灵药/爱情魔咒）**：
```
男女版本区别：
1.主城下部分爱心建筑镜面翻转，箭头朝右/左。
2.主城整体颜色不同（在传统的男蓝女粉外，也可以考虑选取稍微有高级感的配色，
  比如男版金、红、黑，女版金、粉、白，分别用黑白天鹅）。
3.主城自带效果颜色不同。
```

**示范 · 合并特效（2025 心心相印）**：
```
分别从男/女版主城射出带拖尾的爱心，两颗爱心在两主城中间点的空中相撞，
变成心形烟花（多个烟花，有相继炸开的效果，如图3、4）。间隔一定时间循环播放。
```

**示范 · 羁绊光环（2026 光与暗之颂）**：
```
两座主城上的能量石（或其他特殊装饰）发亮，顶部分别出现光/暗漩涡，
从中射出白金/黑红色能量束，汇聚/碰撞在一起，炸开成黑洞/时空裂缝，持续数秒后消失。

（能量石不参与普通动态循环，只在触发羁绊时变化）
```

**示范 · 装饰物三级进化（2026 流水花庭）**：
```
LV1：方向1：古典希腊风格的水榭，两侧有米白色立柱，水面上有花朵、花瓣装饰。
     方向2：小水池，出水口是希腊风格的陶罐。（静态水池）
LV2：方向1：水榭顶部落下水流，水流有流动的动态效果。
     方向2：出水口改为更精致的茶壶、茶杯造型，玫瑰花藤装饰，水面浮花瓣。
LV3：方向1：水榭顶部增加水池，大量花草点缀。
     方向2：出水口换为丘比特猴雕塑，举着陶罐向池中倒水，水流动态。
```

**示范 · 行军特效（2026 审判之箭）**：
```
光明风格的特效，整体形状类似弓箭（需要突出弓箭感），
前端的弓类似细长虚幻的天使羽翼，两侧可有漂浮的羽毛小箭装饰，
后方有白金色流光拖尾（不用太长，视觉上像穿过弓的长箭），整体华丽度高。
```

**必备要素**：
1. 等级标注（主城皮肤必填 1 级 / 2 级 / 2.5 级）
2. 主体造型 + ≥2 局部特征（编号分项）
3. 材质判定（石质 / 机械 / 水晶 / 布料 / 金属）
4. 动态/自带效果（主城皮肤 ≥1 段动态；合并特效独立段）
5. 男女/光暗差异化说明（2025 双版起必填）
6. 多方向备选（装饰三级尤其建议给 ≥2 方向）
7. 避坑提示（颜色、材质、与其它主城的区分）
