# 备用节日 · 美需包装手册（泼水节 / 摘果节 / 篝火节）

> P2 节日美需包装知识库 · 备用节日合集 · 更新于 2026-04

## 总览

本文档汇总三个 P2 "备用节日"——它们不在主排期内，但配置骨架已经预埋好，用作**临时插档 / 实验新题材 / 档期填充**。三者共用同一套 BP + 消耗子活 + 掉落子活 + 活动道具 A + 城内装饰建筑的最小骨架，区别完全在美术主题上。

| 节日 | 主题定位 | 典型启用场景 | 可能档期 |
|------|---------|--------------|----------|
| 泼水节 | 东南亚（泰国 / 傣族）水文化、夏日狂欢 | 4 月中（宋干节）或 7-8 月盛夏补档 | 春夏之交、盛夏 |
| 摘果节 | 田园收获、热带水果丰收 | 9-10 月秋收窗口、或南半球初夏 | 秋季、丰收季 |
| 篝火节 | 部落篝火、夜晚围火歌舞 | 跨年 / 仲冬夜、或秋末露营季 | 深秋、冬季、跨年 |

> 三节都走"**轻量节日**"形态——没有主城 LV1-6 套装循环，没有机甲套系，只保留活动层 + 1 件节日活动道具 + 1 座节日城内装饰，美需量级约为科技节的 10-15%。适合快速投放或做题材测试田。

## 泼水节

### 定位
- **核心概念**：东南亚泼水节 / 中国傣族泼水节的嫁接，把"水 + 炎热 + 狂欢宣泄"做成一场夏日水仗。
- **情绪关键词**：清凉 · 欢闹 · 水花 · 色彩缤纷 · 童趣 · 宣泄
- **节日身份**：夏日题材备用节日，首选 4 月（宋干节真实档期）或 7-8 月补档。
- **相邻联想**：可与海岛 / 沙滩 / 泳池类主题拼接，若未来做"夏日主节日"可把泼水节升级做大盘。

### 视觉调性
| 维度 | 特征 |
|------|------|
| 主色系 | 高饱和撞色：水蓝 + 柠黄 + 粉红 + 薄荷绿（泰式彩粉感）|
| 风格关键词 | 清凉湿润 / 糖果色 / 透明水体 / 动感飞溅 |
| 表现手法 | 水花粒子 · 透明水弹 · 飞溅轨迹 · 湿润高光 · 彩虹折射 |
| 避坑 | ❌ 偏沉的海洋风（深蓝 / 严肃）· ❌ 宗教化佛寺符号 · ❌ 与泳池派对混淆 |

### 模块拓扑（现有骨架）
| 模块 | 子项 | 文案锚点 |
|------|------|----------|
| 活跃集合子活动 | 夏日狂欢 | "完成每日狂欢任务，赢取丰厚奖励！" |
| 消耗子活动 | 清凉加速 | 消耗加速道具换活动道具 A |
| 掉落子活动 | 盛夏大作战 | pve/gve/采集触发掉落 |
| BP 主活动 | （待命名） | "使用清凉的彩色水弹参加盛夏大作战" |
| 活动道具 A | 清凉水弹 | "五颜六色的水弹，让夏日更加欢乐！" |
| 城内装饰建筑 | 活力冰泉 | 喷泉 + 雕塑 |
| 剧情引导 | BP 首进 | 艾文 / 斯科特 / 罗杰 / 艾文（全员兴奋立绘）|

### 命名风格样本
- 活动层：**夏日狂欢 · 清凉加速 · 盛夏大作战**（形容词 = 清凉 / 盛夏 / 夏日，动词 = 狂欢 / 加速 / 大作战）
- 道具：**清凉水弹**（2 字定性 + 2 字实体）
- 装饰：**活力冰泉**（2 字情绪 + 2 字场景）
- 命名公式：`[清凉/盛夏/活力/欢乐] + [实体名词]`，四字为主，回避沉稳词。

### 核心元素库（从现有配置抽提）
| 类别 | 元素 |
|------|------|
| 主元素 | 彩色水弹、水弹枪、喷泉（顶端猴子雕塑举水箱）|
| 辅元素 | 水花、水柱、透明水珠、彩色粉末（可选）|
| 角色互动 | 美妮发射彩色水弹、乔伊扔彩色水弹 |
| 规避 | 真实水枪过于写实、宗教符号、纯单色水 |

### 美需范式（真实样本）

**BP 氛围图 / 斗士互动**：
```
氛围图方案：众多斗士拿上水弹枪，蓄势待发，准备开始打水仗。
斗士动画交互：用美妮发射彩色水弹，或者用乔伊扔彩色水弹。
```

**活动道具 A**：
```
一组不同颜色的彩色水弹
```

**城内装饰 · 活力冰泉**：
```
一个顶端立着雕塑的喷泉，雕塑可以做成正拿着水箱准备攻击的小猴子
```

**美需写作要点**：
1. 装饰物必带**底座**（喷泉池底 / 水面）
2. 道具优先**多色**（水弹的颜色差异是识别点）
3. 角色联动动画要指定**猴子角色名**（美妮 / 乔伊）
4. BP 氛围图强调"蓄势待发"的**集体群像**

### 可探索方向
- **傣族水灯节**：水上漂浮花灯 + 夜晚水面倒影（日夜两套氛围）
- **热带岛屿**：椰林 + 沙滩水仗 + 冲浪板
- **水系精灵**：水之图腾 / 水灵兽 + 透明水材质角色皮肤
- **水枪大作战**：机械水枪 / 超大水炮 + 装备化水弹
- **东南亚庙会**：彩色粉末 + 泰式建筑 + 花环（偏文化侧）
- **彩虹泼墨**：把"水"换成液态颜料，走泼彩艺术方向

## 摘果节

### 定位
- **核心概念**：田园丰收的慢节奏狂欢，以热带水果为视觉核心，主打"劳动成就感 + 丰盛感"。
- **情绪关键词**：丰收 · 香甜 · 阳光 · 童真 · 满载而归 · 比拼
- **节日身份**：秋收 / 丰收季备用节日，可与农业题材拼档。
- **相邻联想**：与感恩节、中秋、丰收节系共用视觉语言，差异点在**热带水果**而非谷物/南瓜。

### 视觉调性
| 维度 | 特征 |
|------|------|
| 主色系 | 暖橙 + 果黄 + 鲜红 + 叶绿（熟透水果的高饱和暖色）|
| 风格关键词 | 田园 / 热带 / 丰盈 / 阳光高光 / 卡通可爱 |
| 表现手法 | 果实堆叠 · 阳光漏光 · 露珠高光 · 藤蔓缠绕 · 奖杯质感 |
| 避坑 | ❌ 出现香蕉（配置明确规避）· ❌ 过于写实的农贸市场风 · ❌ 与万圣节南瓜丰收混淆 |

### 模块拓扑（现有骨架）
| 模块 | 子项 | 文案锚点 |
|------|------|----------|
| 活跃集合子活动 | 欢乐采果节 | "完成每日采果任务，赢取丰厚奖励！" |
| 消耗子活动 | 采果加速 | "消耗加速道具，采集更多水果！" |
| 掉落子活动 | 摘果大赛 | "快去采集更多水果，成为采果冠军吧！" |
| BP 主活动 | 水果聚会 | — |
| 活动道具 A | 水果篮 | 装满热带水果的篮子 |
| 城内装饰建筑 | 椰果冠军奖杯 | "用超大椰子做成冠军奖杯……是荣耀的象征！" |
| 剧情引导 | BP 首进 | 艾文 / 斯科特 / 罗杰 / 艾文 |

### 命名风格样本
- 活动层：**欢乐采果节 · 采果加速 · 摘果大赛 · 水果聚会**（全部带"采 / 摘 / 果"动作或实体）
- 道具：**水果篮**（极简实体名，反差于科技节的修饰+主体）
- 装饰：**椰果冠军奖杯**（物料 + 荣誉概念组合，5 字走情绪升华）
- 命名公式：`[欢乐/丰收/采果] + [实体/活动]`，多用**动作词**（采、摘、聚），比泼水节更软萌。

### 核心元素库
| 类别 | 元素 |
|------|------|
| 主元素 | 热带水果堆（椰子 / 菠萝 / 芒果 / 火龙果 / 草莓）、水果篮、椰子壳奖杯 |
| 辅元素 | 藤蔓、绿叶、小猴子（站在水果山上扔水果）、进度奖杯 |
| BP 演出 | 中央水果小山 + 猴子投掷 + 右侧进度奖杯 |
| 规避 | **香蕉不能出现**（配置明文）、过于朴素的农产品 |

### 美需范式（真实样本）

**BP Banner / 情景**：
```
1、中央是一个水果堆成的小山，一只小猴子站在水果小山上。
2、玩家每消耗一个道具，小猴子就丢一个水果到右边的进度图标（进度图标建议用奖杯）。
3、水果中不能出现香蕉。
```

**活动道具 A · 水果篮**：
```
一个装满热带水果的篮子
```

**城内装饰 · 椰果冠军奖杯**：
```
一个用巨大的椰子壳制作的奖杯，奖杯里装满了各种热带水果
```

**美需写作要点**：
1. 显式列出**规避清单**（香蕉不画，是摘果节配置的强约定）
2. BP 演出带**交互动画**（消耗 → 投掷 → 进度）
3. 实体道具描述可以**极简**（"一篮热带水果"即可），靠画师发挥
4. 装饰物走"超大 + 荣誉"逻辑（巨型椰子 + 奖杯化）

### 可探索方向
- **果园丰收**：果树 + 梯子 + 竹筐，走温暖田园
- **热带雨林**：藤蔓 + 巨型果实 + 鹦鹉，探险感
- **水果怪兽**：拟人化水果角色（菠萝骑士 / 芒果将军），IP 化
- **果酿 / 果汁工坊**：榨汁机 + 果酒桶，工业化丰收
- **采摘竞赛**：竞技化，奖杯主视觉 + 运动员猴子
- **水果市集**：摊位 + 叫卖小猴，生活化
- **糖果化水果**：把水果做成糖霜 / 果冻质感，更童趣

## 篝火节

### 定位
- **核心概念**：部落围火、夜晚歌舞、温暖相聚，走"原始氛围 + 暖色夜景"路线。
- **情绪关键词**：温暖 · 团聚 · 夜晚 · 火光 · 歌舞 · 希望
- **节日身份**：冬季 / 跨年 / 秋末备用节日，和春节"热闹"路线互补（篝火更"野 + 暖"，春节更"红 + 喜"）。
- **相邻联想**：与万圣节夜晚场景可共用暗色基底，但篝火节是**暖黄**而非阴冷。

### 视觉调性
| 维度 | 特征 |
|------|------|
| 主色系 | 火焰橙红 + 木炭棕黑 + 夜空深蓝 + 星点金（暖火 vs 冷夜对比）|
| 风格关键词 | 部落 / 夜晚 / 火光 / 原始 / 温暖粗粝 |
| 表现手法 | 火焰舞动 · 飞舞火星 · 木材质感 · 夜空星点 · 光影投射地面 |
| 避坑 | ❌ 纯灾难/森林火灾感 · ❌ 圣诞壁炉（太家居化）· ❌ 与春节灯笼混淆 · ❌ 火焰做得像烧烤 |


**美需写作要点（篝火节现有美需最稀疏，需补位）**：
1. 篝火的**火焰动态**必填（燃烧循环 / 火星飞散）
2. 装饰物的**地面光照**要注明（暖黄光圈 + 影子投射）
3. BP 演出（现缺失）建议走"玩家投柴 → 火焰变大 → 猴子围火舞蹈"的进度化演出

### 可探索方向
- **部落狂欢夜**：图腾 + 皮鼓 + 羽毛头饰的猴子萨满，民族感
- **星空篝火**：夜空银河 + 火星呼应，偏浪漫
- **守夜人**：冬季雪地 + 篝火，温暖对抗严寒（可和春节做区隔）
- **野外露营**：帐篷 + 篝火 + 小马灯，现代化休闲风
- **火祭 / 神秘仪式**：围成圆圈 + 符文 + 火焰蓝紫化，奇幻感
- **跨年篝火晚会**：倒计时 + 烟花，把跨年节嫁接进来
- **篝火故事会**：光影投射在地面讲故事，情感叙事侧

## 附：三节共用特征

### 共用骨架
所有三个备用节日都锁定这 7 块骨架（无额外套装 / 机甲 / 头像框 / 行军特效）：
1. 子活动-活跃活动集合
2. 子活动-消耗活动
3. 子活动-掉落活动
4. 主活动 BattlePass
5. 活动道具 A（BP 推进道具）
6. 城内装饰建筑 ×1
7. 剧情引导（艾文 / 斯科特 / 罗杰 / 艾文 四条，全员"兴奋"立绘）

### 共用命名拓扑
- **活跃子活**：`[主题情绪词] + [狂欢/大赛/聚会]`
- **消耗子活**：`[主题词] + 加速`（清凉加速 / 采果加速 / 烈焰加速）
- **掉落子活**：`[主题] + [比拼动作]`（盛夏大作战 / 摘果大赛 / 舞动青春）
- **BP**：`[主题] + [聚会/比舞/作战]`
- **道具**：单纯 2-4 字实体（水弹 / 水果篮 / 木材）
- **装饰**：`[情绪] + [实体]`（活力冰泉 / 椰果冠军奖杯 / 希望篝火）

### 启用决策建议
| 触发场景 | 优先备用节日 |
|----------|-------------|
| 盛夏降温 / 东南亚市场导量 | 泼水节 |
| 感恩节 / 丰收季 / 轻松向填档 | 摘果节 |
| 跨年冬档 / 夜晚氛围需求 / 部落题材测试 | 篝火节 |

### 美需补位提示
三节现有美需列**非常稀疏**（尤其篝火节只有两条单行描述），若进入排期，需要在 `6、道具/城内装饰` 和 `4、BP 氛围图` 两处**重点扩写**，建议参考科技节的范式：
- 特征陈列 ≥2 点
- 底座 / 支撑物说明
- 动态 / 特效描述
- 多方向备选
- 避坑清单（尤其摘果节的"不画香蕉"已是范例）
