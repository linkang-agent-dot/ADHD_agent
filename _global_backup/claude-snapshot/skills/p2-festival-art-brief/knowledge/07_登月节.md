## 二、节日定位

- **核心概念**: 仰望星空的浪漫探索节日，"太空登陆 + 童趣复古"。强调"这一小步，是猿族的一大步"的英雄叙事 + 猿族自嘲向幽默。
- **情绪关键词**: 星辉 · 浪漫 · 好奇 · 诗意 · 失重轻盈 · 童趣 · 仰望
- **节日身份**: 单次大型节日（非循环），主打主城皮肤 gacha + 装饰物 1-3 级 + BP
- **与科技节的锚定区分**:
  | 维度 | 登月节 | 科技节 |
  |------|--------|--------|
  | 气质 | 浪漫/诗意/探索 | 硬核/工业/军事 |
  | 主体 | 宇航员、月球、飞船、星空 | 坦克、机甲、霓虹、全息 |
  | 色调 | 深蓝/紫金/银白/月光冷 | 冷蓝/机械银/黑红军事 |
  | 动态 | 漂浮、旋转、闪烁、升空 | 变形、开火、扫描、律动 |
  | 情绪 | 抒情、童趣、自嘲 | 燃、突破、威慑 |
- **与其它节日的锚定关系**:
  - 与春节（行军特效循环）、科技节（主城套装循环）、情人节（机甲循环）区分——登月节当前不是循环节日
  - 与周年庆行军特效、科技节星际飞船主城需做视觉断档

## 四、模块拓扑（2025-2026 标准）

### A. 主城皮肤

2025 年走三皮肤结构（传说 + 传说染色 + 超凡），2026 回到传说 + 超凡 + 璀璨 + 林康方向 + 挖孔方向的多方向并行。

| 类别 | 功能定位 | 美需侧重 |
|------|---------|----------|
| 传说（低级）| 基础形象 | 主体干净、体现核心元素（月宫 / 航天飞船 / 大富翁） |
| 超凡（高级）| 升级版 | 体积更大更华丽、增加装饰、加动效 |
| 璀璨（特殊）| 动态演出 | 玩偶/角色动作循环（如搭积木-推倒-复原-礼花） |
| 染色（2025 特有）| 普通平级换色 | 兔子雕像姿势变化、配色差异（如紫金 vs 蓝棕） |
| 挖孔方向 | 副方向 | 变形飞船（参考 Lastwar）+ 工猴驾驶员 + P2 废土风格化 |

### 装饰物命名公式

- **主名称**: 2-5 字意象词（月亮船 / 登月之猩 / 月球探测车 / 火箭摇摇乐 / 望远镜天文台）
- **描述**: 8-14 字，带意境短句或功能说明：
  - "月亮形状的小船，满载着诗情画意"
  - "这一小步，是猿族的一大步"
  - "探索宇宙奥秘的使者"

### 行军/特效命名公式

- **四字主名**: `[意象词 1] + [意象词 2]`，如 银河之夜 / 彩云追月 / 月色入画 / 星月环绕
- **描述**: 10-14 字，诗意收束，如 "为你点亮一抹银河的星辉" / "腾云壮志，踏月前行"

### 铭牌/头像框命名公式

- 四字主名带"星 / 月 / 探 / 梦"：探月日记 / 星空梦想 / 寄情明月 / 月下荡舟
- 描述偏拟人抒情："所有猿族，都有一个宇航梦" / "将一缕思念寄予明月"

### 活动命名公式

- 四字：太空漫游 / 登月留念 / 失重体验 / 月面探险 / 月球矿坑
- **Bingo 名通用**: 2025 起沿用"XX 行动"格式（拓荒行动 / 科技行动）

## 七、美需描述范式（从真实 2024-2026 抽提）

**结构**: `[整体概念] + [结构分解：主体/底座/装饰] + [动效描述] + [配色/材质判定] + [≥1 方向备选] + [避坑/区分提示]`

**示范 · 主城皮肤低级（多方向备选）· 2024-9**:
```
方向1：飞船
整体：具备猿猴特色的航天飞船，可运用香蕉元素或月亮元素，有一定拼接风格；
需和"去年登月节皮肤、着陆舱建筑、主城火箭皮肤、VIP皮肤"等做出明显区别。

结构：为了增加辨识度和差异感，可考虑环形结构或弯月形状，
     可分拆成 助推动力装置、落地支架、飞船主体、玻璃舱室等；

动效：飞船部分结构支持旋转（绕中心旋转，或顺时针旋转），高级版本向底部和周围喷火雾

方向2：建筑-天文馆
整体：具备猿猴特色的天文馆，可运用香蕉元素、星球球体和轨道元素，有一定拼接风格；
结构：1.以球体为中心的环绕结构，或者类似于星象仪的多轨道结构。
     2.添加巨型天文望远镜。
动效：球体、望远镜旋转（高级）。
```

**示范 · 主城皮肤璀璨（动态编排）· 2026-7**:
```
特殊：
穿着小礼服的玩偶猩猩（玩偶材质，不是像素风）+积木城堡+灯光底座

动态：猩猩搭积木，推倒积木城堡，积木城堡复原，从底座周围喷出礼花。（参考动图2）
```

**示范 · 装饰物三级进化 · 2024-9 登月之猩**:
```
等级1：整体算是一尊雕塑，穿着宇航服的猩猩举着手（1级可空着手），站在月球上；
      月球下方可有个简易基座（或者一级时只有半个球，不要基座），增加地面装饰物的感觉。
等级2：在此基础上宇航员猩猩手里或球面上增加旗子；完整月球，底下的基座提升华丽度和复杂度
等级3：底座周围增加星星装饰、人造卫星装饰；增加环绕的星光特效
```

**示范 · 主城特效 · 2024-9 星月环绕**:
```
1.围绕主城建筑底周的一圈的动态特效，特效主要在地表生成，
  特效圈内部边缘感清晰，不和主城皮肤自带特效冲突。
2.考虑到登月节投放，本次以【太空、宇航、月亮】为主题进行制作，
  比如主城被银河彩带环绕，闪耀较小的星光和弯月，逆时针旋转。
```

**必备要素**:
1. 整体概念一句话点明（主城/装饰是什么）
2. 结构分解 ≥2 层（主体/底座/装饰物）
3. 动态/特效描述（高级版本必填）
4. 与历年登月节、周年庆、科技节、主城火箭皮肤等的区分提示
5. 多方向备选（主城皮肤建议 ≥2 方向）
6. 配色/材质明确（尤其当多皮肤平级时，如 2025 三皮肤差异化）
