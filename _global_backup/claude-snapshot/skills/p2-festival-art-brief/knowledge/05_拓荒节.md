## 二、节日定位

- **核心概念**: 劳动节档期的开拓/建设/探险/垦荒主题。中文侧贴"劳动节"朴实勤劳语感，英文侧对接"Frontier / Pioneer"西部拓荒语感，两条语义一条画风。
- **情绪关键词**: 开拓 · 远征 · 冒险 · 勤劳 · 热血 · 荒野 · 英雄主义 · 日出感
- **节日身份**: 主题型节日（非套装循环），每年换一套大主题（废土工业→西部沙龙→淘金探险→大富翁西部），重视一次性视觉记忆点。
- **与其它节日的锚定关系**: 与科技节并列的"男性向活力节日"——科技节偏冷色未来感，拓荒节偏暖色荒野/劳动感，彼此分工明确。

## 四、模块拓扑（当前标准，2026 定型）

### A. 主城皮肤（每年主产出，通常 2-3 套）

2026 样本已回到"多套并行"结构：
- **gongliang 套（大富翁 × 积木）**: 低级→高级→特殊 三阶，猴子形象+积木城堡+灯光底座。
- **liusiyi 套（西幻剑与龙）**: 基础形态→高级形态→套装部件（雷电龙）→终极闪耀形态，龙飞剑上、雷电爆发。

历年主城命名走"开拓者 / 拓荒之城 → 西部沙龙 / 荒野传奇 → 掘金之旅 / 黄金梦乡 → 蒸汽霸主"路线，统一呈现"主体建筑 + 劳动/探险意象 + 动效升级"三段进化。

### 命名公式

- **主城名**: `[形容词/地域词] + [主体名词]`，四字为主。主体池：开拓者 / 沙龙 / 传奇 / 梦乡 / 霸主 / 之城 / 之旅；形容词池：西部 / 荒野 / 黄金 / 掘金 / 蒸汽 / 械动。
- **描述**: 10-18 字，句式"[能力特征/画面] + [情绪收束]"。**动词池：开拓 / 远征 / 垦荒 / 点燃 / 守望 / 掘进 / 穿越 / 瞄准 / 追逐**。情绪收束偏英雄主义+勤劳励志。
- **机甲染色**: `机甲名（意象短语）`，意象带劳动/守卫情绪（如警戒者、守护者）；道具名"猎豹（警戒者）"、道具描述"使用后可激活猎豹机甲'警戒者'涂装"固定公式。
- **装饰物**: "劳动节节装饰等级 1/2/3" 作 functional name，对外命名（如 2023 施工警示、2024 荒野守望者、2025 淘金营地）用一个共享名复用三级。
- **BP**: "掘地求胜 / 收集钻头，领取拓荒节奖励"多年锁定。

## 七、美需描述范式（从真实 2026 抽提）

**结构**: `[核心主体] + [形态细节] + [底座/支撑判定] + [动态效果] + [配色] + [≥1 方向备选] + [避坑提示]`

**示范 · 主城皮肤（多方案平行）**:
```
方案一：黄金矿工+抓娃娃机
1.建筑主体为一个抓娃娃机，装娃娃的箱体结构较大，底部是摇杆和按键，偏废土、破旧；
2.娃娃机顶上有机械矿工猴子、拴绳子的卷筒，卷筒转动，带动抓钩往下，吊起黄金、石块、钻石等；
升级版：有抓钩往下、吊起黄金的动态，机械矿工猴子变为牛仔装束，娃娃机边缘加灯带。

方案二：移动城市（突出探索、冒险主题）…（废土+蒸汽朋克+拟人）
方案三：西部篷车+射击摊位…（篷车+标靶+左轮枪装饰）
```

**示范 · 主城特效（双方案 · 落地难度预案）**:
```
方案一：几个细长的沙尘暴龙卷风围绕城市转动，风中裹挟金砂，短暂旋转后散去、散落满地财宝；
方案二：如果方案一落地有困难（财宝堆需建模），不要财宝堆，改为龙卷风刮完后天上落下财宝雨（金币/珠宝/宝盒），财宝在地面弹跳后消失。
```

**示范 · 机甲皮肤（柴油朋克方法论）**:
```
整体在巨象机甲的基础上进行修改，可做不大的模型变化
1、整体以铜黄、铁黑等为主要颜色，材质上可以有生锈、油污的表现；
2、躯体上有管道、排气筒等构造；也可以有枪炮等；
3、象牙可以用金属管道来表现，四肢、耳朵等可以铁皮拼接+齿轮为主；
4、可以在背部管道口加一点冒黑烟的特效。
```

**示范 · 行军特效（双段编排）**:
```
鲨鱼
前部：圆圈状海水特效，海中有鲨鱼鳍在转圈，可加小鱼、海星等装饰；
尾部：整体是海水波浪的流光效果；流光中有鲨鱼游动，流光颜色稍浅，能表现鲨鱼轮廓。
```

**示范 · 装饰物（三级进化）**:
```
等级1：普通热气球 · 上方简单气球 + 下方方篮子
等级2：高级版 · 气球变大/加涂鸦、补丁、绳子；下方变大篮子，可加炸弹
等级3：华丽版 · 气球加多装饰、可做异形；下方多个篮子或房屋/简易小船；装饰可结合拓荒节，加冒险、西部牛仔元素
```

**示范 · 联动头像框（双阶段叙事）**:
```
阶段一·凋零：细碎菱形冰晶不断向下飘落，左右有漩涡状寒冷雾气，冰晶下落有重力感但受魔法影响较慢，像雪花般轻盈。
阶段二·凝视：下落的冰晶不再消失，而是在圆环中央迅速聚拢、冻结；半透明幽灵冰骨骷髅，骷髅头刚好佩戴顶部王冠，形成完美闭环。
```

**必备要素**:
1. 特征陈列 ≥2 点（编号或分号）
2. 底座/支撑物处理说明（装饰物、主城皮肤必填 —— 2023 起反复强调"需要装饰底座，避免像一个载具"）
3. 动态/特效描述（升级版、高级版、LV3 必填）
4. 多方向/多方案备选 + 落地难度预案（示意 2025 主城特效方案一/二）
5. 避坑提示（与其他功能建筑、节日皮肤的区分）
6. 配色明确（暖色主调+辅色调）
