> 方案选择依据：双档冷热反差 + 与 2025 黑胶复古金完全拉开 + 与 2026 摇滚头像框不抢戏

## 模块一 · 高级行军特效 · 朱祭夜音

- **中文文案**：`朱祭夜音` | `朱伞鼓动，同赴夜祭！`
- **主色调**：朱红 + 墨黑 + 金箔点缀
- **美需描述**：
  1. 主色调：朱红 + 墨黑为主，金箔作点缀（避开 2025 随乐出征的暖金怀旧调，改走朱红鎏金路线）
  2. 最前端：一面悬浮太鼓（带樱花纹），队伍前进时每一击鼓面炸开一圈金色鼓音波纹
  3. 鼓两侧飘散樱花瓣（朱红/酒红系，非粉色）+ 朱红油纸伞虚影，随队伍节奏起落
  4. （可选）尾部拖出一道若隐若现的能面纹样（半透明光影剪影，不做具象五官）
  5. 鼓面节奏 = 行军速度反馈（速度快则鼓点密、慢则鼓点缓）
- **避坑**：
  - 不做鸟居 / 神社 / 佛像等宗教具象符号
  - 不全场金（金只做点缀，主色压朱红 + 墨黑）
  - 不做春节灯笼 / 福字（那是春节专属）
  - 樱花走朱红 / 酒红系，不走粉色（避免与春节粉调混淆）

---

## 模块二 · 低级行军特效 · 荧光追行

- **中文文案**：`荧光追行` | `挥舞应援棒，节拍不停歇！`
- **主色调**：粉 + 紫 + 荧光蓝
- **美需描述**：
  1. 主色调：粉 + 紫 + 荧光蓝（糖果荧光色），整体轻盈高饱和
  2. 最前端：一支悬浮荧光应援棒（棒头为星星/心形），轻微左右摇摆挥动
  3. 飘散元素：彩纸碎片 + 小星星 + 心形光点，朝两侧飘落
  4. （可选）尾部一串 LED 光条随行，循环闪烁呈现彩虹流光
  5. 应援棒头呼吸发光（周期 1-1.5 秒）
- **避坑**：
  - 不使用真实偶像形象 / 真人剪影
  - 糖果荧光 ≠ 儿童向（避免与科技节冷电音 / 儿童霓虹撞车）
  - 不出现电吉他 / 锁链 / 尖刺（让位给 2026 摇滚头像框）
  - 不做奶油 / 甜品元素（避免与情人节 / 复活节混淆）
  - 应援棒不做成魔法杖（避免奇幻风）

## 两档协同校验

| 校验项 | 高级（祭囃子）| 低级（应援） | 结果 |
|--------|-------------|-------------|------|
| 主色调反差 | 朱红 + 墨黑 | 粉紫荧光蓝 | 冷热完全反差 |
| 前端主体 | 太鼓 | 应援棒 | 器物识别度高 |
| 飘散元素 | 樱花瓣 + 红伞 | 彩纸 + 星星 + 心形 | 风格分明 |
| 调性 | 神秘夜祭 | 狂欢轻盈 | 不同情绪轴 |
| 与 2025 随乐出征差异 | 黑胶 → 太鼓 | 黑胶 → 应援棒 | ≥60% |
| 与 2026 摇滚头像框差异 | 异色路线 | 荧光少女 | 不抢摇滚戏 |
