## 二、节日定位

- **核心概念**: 中式传统节日 · 生肖年度化 · 红金主色基调。与猿族废土世界观碰撞，传统元素需做少量"机械拼装/废土嫁接"处理（2023-2024 强调，2025-2026 弱化，回归纯国风）。
- **情绪关键词**: 喜庆 · 祥瑞 · 团圆 · 吉祥如意 · 红火 · 辞旧迎新 · 生肖守护
- **节日身份**: **行军特效套装循环节日**（2026 定型）——主城皮肤为副，行军特效高级套装（巨像+前部环绕+尾部脚印 三组件）为核心产出
- **与其它节日的锚定关系**: 春节与科技节（主城套装循环）、情人节（机甲皮肤循环）并列为三大"循环节日"。各自占一条独立美术产线。

## 四、模块拓扑（当前标准，2026 定型）

### A. 行军特效套装 LV1-4（循环主产出）

| 层级 | 功能定位 | 美需侧重 |
|------|---------|----------|
| 普通行军皮肤 | 基础款，单独投放（非套装）| 底部流光 + 拖尾装饰元素两部分；给 ≥2 方向备选 |
| 套装 · 组件 1 | 巨像特效（天降/俯冲）| 从军队上方出现的大型生物/图腾，有喷射→消失编排 |
| 套装 · 组件 2 | 前部环绕（半环形破风）| 围绕前进方向的地貌/光效，随移动持续生成-消失 |
| 套装 · 组件 3 | 尾部脚印（经过处痕迹）| 脚印/裂缝/冰晶，由小变大或由大变小循环 |

套装命名规律：**主标题（4字意象）+ 3 个组件各自独立命名**，组件文案强调"力感/速度感/破坏感"。

### B. 主城皮肤（普通 + 超凡双档）

| 等级 | 功能定位 | 美需侧重 |
|------|---------|----------|
| 普通皮肤 | 基础生肖形象（虎/蛇/龙）| 主体造型 + 材质描述 + 可选的生肖替换方案（虎↔麒麟）|
| 超凡皮肤 | 在普通基础上升级 | 新增建筑元素（寺庙/冰晶底座）+ 动态演出（长啸/转圈/振翅）+ 特效（流光/风雪）|

2025 之前没有"超凡"档，2026 起跟齐其他节日的双档结构。

### C. 主城装饰升级 LV1-3（挖矿系 / 装饰等级）

三级进化，每级在前一级基础上 (1) 增加主体复杂度 (2) 增加春节元素装饰数量（灯笼/鞭炮/红包/元宝/绸缎）(3) 加发光特效。

历年主题：舞狮（2023）→ 福瑞灯台（2024）→ 摇钱树（2025）→ 舞龙灯台（2026）。

### D. 主城特效

主城顶部动态演出，多方向备选（2025 给 4 方向、2026 给 2 方向）。常见方向：祥龙腾飞、凤凰涅槃、莲花祥云、剑阵、罗盘星象仪。必须带循环结束后的从头播放。

### E. 头像框 / 铭牌

- 动态头像框：生肖 + 鞭炮/绣球动态（爆炸、眨眼、流苏摆动）
- 铭牌（静态）：生肖年号 + 角隅装饰（红包/铜钱/灯笼/金蛇）
- 聊天气泡（2023 独有）：生肖主题两端呼应设计

### F. 活动道具

- 活动道具 A（绣球）：舞狮用，红色为主色调——2023/2024/2025 持续复用
- 活动道具 B（新春骰子）：红色丝绸/锦缎质感
- 活动道具 C（福运灯笼，2023 独有）：5 色渐变灯笼
- 活动道具 D（烟花礼盒，2023 独有）：装饰物升级材料

### 命名公式

- **主城名 / 行军套装名**: `[生肖形容] + [动作/状态]` 四字为主。动词池：迎春 / 贺岁 / 啸春 / 咆哮 / 腾飞 / 送福 / 凌厉 / 威震。
- **描述**: 10-14 字，两短句结构：`[生肖/元素意象] + [吉祥收束]`。吉祥收束池：岁岁平安 / 瑞气盈门 / 万象一新 / 猿气满满 / 新年好运 / 家家户户。
- **行军组件三件套**: 主标题（4字意象）→ 三个组件分别按"巨像主体/中段破风/尾部脚印"的身份各给一个意象名，从"总 → 分"顺序展开。
- **装饰升级三级同名**: 三级共用同一主题同一名称（如"摇钱树"），靠描述字段增量，不要每级独立命名。

## 七、美需描述范式（从真实 2026 抽提）

**结构**: `[方向名/配色锚] + [整体效果一段式编排] + [组件/细节编号列表] + [多方向备选] + [避坑提示]`

### 示范 · 普通行军皮肤（底部流光 + 拖尾装饰 结构）

```
方案一：花灯【已确定】（方案二在右边→→→）

底部流光：整体是金蓝色渐变流光，前金后蓝，可有轻微的水波+风雾感；
可添加粒子特效；

拖尾装饰元素：大小不一的方形亮金色花灯，从前往后飘一会儿消失；
也可以放莲花形状的花灯，建议不要太多，以方灯为主；
可以在其中添加一些大小不一的萤火光点/雪点；
```

**关键**: 行军皮肤必拆成"底部流光（贴地）+ 拖尾装饰元素（飘散物件）"两层描述，颜色+形状+动效三要素。

### 示范 · 行军特效套装（三组件叠加编排）

```
方向一：熔岩火龙（西方龙）【已确定】
整体效果：部队前方是半环形破风效果，经过处留下熔岩脚印，
火龙从军队上方出现，向前喷火后消失

组件1（巨像特效）：火龙从天而降，翅膀处带火焰；
在军队上方停住，往前下方喷火，喷到地面时碎石飞起；
火龙喷完火消失（可做出化红烟消失的效果）
1、火龙可红黑/红金配色，双翼大躯干小……
2、龙息以火焰特效为主……

组件2（前部环绕）：半环形破风效果，以红色为主，有红黑色尖石包围左右；
随着前进，前部尖石不断从地面钻出，尾部尖石消失；

组件3（尾部脚印）：经过处有圆形熔岩裂缝出现并消失，左右交错；
可添加火焰特效；
```

**关键**: 先给"整体效果"一段做空间叙事，再分组件编号展开；每组件标注身份（巨像/前部环绕/尾部脚印）。

### 示范 · 主城特效（多方向 + 循环编排）

```
方向二：罗盘【已选择】

整体效果：顶部是持续旋转的星象仪，底部有罗盘指针，
指针旋转时极光向内聚集，走完一圈后指针消失，极光向外爆发；

1、底部的罗盘可做成月相变化图案或简单的东南西北标识；
2、极光向内聚集时，可做成漩涡+四周线性流光效果（参考图2）；
   极光向外爆发时，可做成冲击波光效+线性流光（参考图3）；
3、特效整体颜色以蓝紫青为主，可添加星光细节；
```

**关键**: 主城特效一定要有"从无→有→高潮→消散→从头播放"的循环编排。

### 示范 · 主城装饰升级（三级递进模板）

```
1级舞龙灯笼：造型较简单，体型小，没有太多花纹和装饰；不发光或黯淡光。
1级台子：简单的台子，没有多余装饰。

2级舞龙灯笼：相比1级体型增加，鳞片花纹更复杂，色彩更丰富；微微发光。
2级台子：可以增加金色花纹和礼花、鞭炮装饰，体现春节喜庆氛围。

3级舞龙灯笼：姿态可以盘卷向上增加高度，整体更华丽，发亮光。
3级台子：可以做国风海浪纹装饰的台子。
```

**关键**: 每级拆成"主体 + 台子/底座"两项；每级三件事：(1) 体型/姿态变化 (2) 花纹/材质变化 (3) 发光/特效变化。

### 必备要素

1. 方向标注（多方向必须明写"方向一/方向二"，标注"已确定/已选择"）
2. 整体效果一段式空间叙事（先描述从哪到哪发生什么）
3. 组件编号 + 身份标签（巨像/前部环绕/尾部脚印 / 底部流光/拖尾）
4. 配色明确（主色+辅色）
5. 替换方案备选（如虎↔麒麟，西方龙↔东方龙）
6. 参考图编号 + 避坑提示
