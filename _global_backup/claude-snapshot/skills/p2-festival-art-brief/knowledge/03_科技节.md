## 二、节日定位

- **核心概念**: 脱离猿族世界观，正面拥抱真实世界科技符号，"打破次元壁 + 耳目一新又了然于掌"。
- **情绪关键词**: 未来感 · 机械感 · 智能 · 冷光 · 探索 · 突破 · 变形
- **节日身份**: 套装循环节日（主城 LV1-6 换皮），同时带手札装饰循环
- **与其它节日的锚定关系**: 科技节与春节（行军特效套装循环）、情人节（机甲皮肤循环）并列为三大"循环节日"

## 四、模块拓扑（当前标准，2026 定型）

### A. 主城套装 LV1-6（套装循环主产出）

| 等级 | 功能定位 | 美需侧重 |
|------|---------|----------|
| LV1 | 低级城堡（基础形象） | 主体造型干净，投放在皮肤 gacha 日排行榜 |
| LV2 | 组件 1 | 静态/挂特效，不遮主城 |
| LV3 | 组件 2 | 静态/挂特效，与 LV2 形态差异化 |
| LV4 | 光环 / 高级特效 | 城堡下方光圈展示，可给 ≥2 方向 |
| LV5 | 高级城堡 | 带特效+动作，通常是 LV1 的"变形觉醒版" |
| LV6 | 终极形态 | 动态演出（如机器人↔坦克循环变形），静→动升级 |

### B. 主城装饰升级 1-3（挖矿系）

三级进化，每级在前一级基础上 (1) 增加躯干复杂度 (2) 换材质 (3) 加特效。

### C. 手札装饰（组件 1-5）

- 活动道具-普通装饰（主装饰）
- 活动道具-普通装饰子模块 1-5（5 个可独立收集组件，合起来拼出完整景）
- 命名走"源力基台 / 充能支座 / 智控面板 / 投影光仪 / 影像展架"系列

### D. BP 奖励组

- 节日 BP 高级行军特效（像素电玩 / 电路流光等）
- 节日 BP 低级行军特效
- 动态头像框
- 铭牌 & 聊天框

### E. 机甲系列（通常配 2-3 套）

- 机甲染色 × 挖孔活动奖励款（中端，~\$400 累充位）
- 机甲染色 × 累充奖励款（高端款）
- 命名公式：`机甲名(例：猎豹) · [配色短语]`，如"猎豹（流金幻影）""战龟（黎明守护者）"

### F. 节日大富翁装饰物（LV1-3）

2026 起新增的节日大富翁挂件，一套三级。

### G. 活动层

- 主 gacha 活动（抽主城皮肤）
- 兑换商店子活动
- 每日活跃任务
- 累充活动
- 早期（2023/2024）有 BP 能源/科研主题子活动
- 礼包团购（静态头像框等小奖励）

### 命名公式

- **主城名**: `[形容词] + [主体名词]`，四字为主。主体从飞船/堡垒/坦克/大厦/要塞池里选；形容词走"威猛/未来/银河/战地/摩登/星际"系。
- **描述**: 10-14 字，两短句结构：`[能力特征] + [情绪收束]`。动词池：引领 / 主宰 / 穿透 / 横扫 / 点燃 / 驱动。
- **机甲染色**: `机甲名（四字意象）`，意象带猎杀/守护/领主情绪。
- **手札组件**: `[属性词] + [部件名]`（源力基台 / 充能支座 / 智控面板）。

## 七、美需描述范式（从真实 2026 抽提）

**结构**: `[核心主体] + [形态细节] + [底座/支撑判定] + [动态效果] + [配色] + [≥1 方向备选] + [避坑提示]`

**示范 · LV5 高级城堡**:
```
高级版：
1、整体是坦克机器人的效果，双脚是履带，手臂是炮筒；
2、坦克机器人的头部可以用望远镜或驾驶舱代替；

底部台子：可以变化围栏样式等；
（底部台子的必要性，履带=台子）
```

**示范 · LV6 终极形态（动态编排）**:
```
动态效果：
1、开火攻击：以机器人形态，用手炮和主炮向周围开火；
2、变形阶段：坦克机器人停顿一会儿，双脚变回履带，炮塔下降……
3、变形后：炮塔和头部升起，履带变成脚部，按原路变回……
机器人形态——开火——变成坦克——坦克变回机器人
```

**示范 · 装饰物（多方向选择）**:
```
方向一：炮火 · 黑红色调，偏红火焰+黑烟+地裂+碎石
方向二：全息地形图 · 蓝色，旋转扫描+外圈围绕
（方向二👉👉👉 在右边）
```

**必备要素**:
1. 特征陈列 ≥2 点（编号或分号）
2. 底座/支撑物处理说明（装饰物类必填）
3. 动态/特效描述（LV4-6 必填）
4. 多方向备选 + 标注"运营倾向"
5. 避坑提示（列明不要做什么）
