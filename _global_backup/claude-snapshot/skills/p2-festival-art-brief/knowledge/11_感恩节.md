> **改名节点**: 2021-2024 叫"丰收节（感恩节）"——"丰收"语义温和普适，适合北美 / 欧洲 / 亚洲市场共用；2025 改为"黑五 & 感恩节"——引入 Black Friday 商业元素，强调"狂欢购物 + 感恩"双主题，面向英语圈促销语境。
>
> 从 **2025 起，感恩节进入"套装循环节日"矩阵**（主城 LV1-6 每年换一套），与科技节（3 月）、春节（行军特效循环）、情人节（机甲皮肤循环）并列。

## 二、节日定位

- **核心概念**: 秋日收获 + 感恩团聚 + 黑五狂欢，三层叠加。丰收是底色情绪，感恩是关系叙事，黑五是消费爆点。
- **情绪关键词**: 丰收 · 温暖 · 感恩 · 金秋 · 满载 · 狂欢购物 · 折扣 · 犒赏自己
- **节日身份**: 套装循环节日（2025 起）；同时保留传统挖矿/BP/礼包活动带黑五商业化包装
- **与其它节日的锚定关系**:
  - 视觉上与万圣（10 月）相邻但错开：万圣用暗色+南瓜鬼怪，感恩用暖色+南瓜丰收
  - 与春节（1 月）在"团聚 + 丰盛饭桌"语义上呼应但不撞色
  - 与科技节同属"套装循环节日"，但基调完全相反（科技=冷/未来，感恩=暖/自然）

| 维度 | 特征 |

| 避坑 | ❌ 与万圣混淆（不要鬼怪 / 暗巫 / 墓地）· ❌ 真实肉食（火鸡必须是雕塑/卡通/玩偶，不做成真肉）· ❌ 香蕉（2022 明确禁用香蕉做丰收元素）· ❌ 宗教符号 · ❌ 过度黑暗的秋色 |

## 四、模块拓扑（2025 套装循环模板定型）

### A. 主城套装 LV1-6（2025 起 · 套装循环主产出）

以 2025 "星体"套装为首发循环样板：

| 等级 | 功能定位 | 2025 样本 | 美需侧重 |
|------|---------|-----------|----------|
| LV1 | 低级城堡（基础形象）| 初始之星（较小的普通星球）| 主体造型干净，参考 Whiteout Survival / Last War 同类 |
| LV2 | 组件 1 | 星体能量仪（星球机械外壳）| 静态/挂特效，不遮主城 |
| LV3 | 组件 2 | 飞火陨石（围绕星球转动的火流星）| 与 LV2 差异化的辅件 |
| LV4 | 光环 / 底周特效 | 数据之环（宇宙/星空底周）| 城堡下方光圈 |
| LV5 | 高级城堡 | 闪耀之星（完整/火焰星球）| LV1 的"觉醒版"，带特效+动作 |
| LV6 | 终极形态（动态演出）| 撼动星辰（恒星爆炸→星云收缩→重生循环）| 连续动态编排 |

> **注意**: 2025 是"星体"主题，更偏"科技节"气质。未来感恩节套装循环应回归丰收本色（见第八节可探索方向）。

### B. 主城装饰 / 手札装饰（1-3 级进化）

历年传统款：

- **2021**: 感恩大餐桌（桌面食物堆叠成小山，香蕉/酒杯/烛台/罐头/南瓜，不做火鸡）
- **2022**: 丰收守望者（稻草猴+硕果粮仓，三级通过瓜果数量 + 特效控制）
- **2023**: 感恩大餐桌（桌布 + 烛台 + 落叶，三级加特效）
- **2024**: 秋日小憩（葡萄架秋千，藤蔓由半黄变全黄 + 双人秋千 + 摇晃动态）

### C. 手札装饰（主装饰 + 组件 1-5）

2025 起并入套装循环体系，与主城套装命名呼应。

### D. 活动层（黑五商业化）

- **主 gacha**（历年延续）：2022"丰收盛宴"→ 2023"丰收盛宴"→ 2025"海滩狂欢"等不同变体
- **挖矿小游戏**（2024 新增、2025 延续）：场景装饰用机械火鸡、丰饶角、食物筐，石块障碍以感恩节食物造型呈现
- **黑五省省卡**（2025 新增核心）：绑定机甲染色 + 头像框 + 行军特效 的组合销售位
- **特殊折扣礼包**（2023 起）：黑五狂欢 / 黑五折扣 / 感恩大回馈
- **BP（2021-2023）**：感恩厨房 BP · 协助雷米筹备宴会（后期式微）
- **7 日活动 / 消耗活动 / 掉落活动**：丰收加倍 / 美食猎人 / 装饰升星

### E. 机甲染色（2025 起双套）

- **黑五主款**：`猎豹（暗金雷霆）`——贵重低调、雷霆迅猛
- **感恩节款**：`金刚（秋狩者）`——枫叶红 / 橙红 / 麦穗印花 / 火鸡配色（方案二）

### F. 行军特效 / 头像框 / 铭牌（历年传统配件）

- **行军皮肤**: 金穗浓情（稻穗拖尾）/ 向阳而生（葵花+落叶+橡果）
- **铭牌**: 秋日低语（栗子+枫叶）/ 丰饶号角（丰饶角+滚落水果）
- **头像框（2025）**: 豪掷千金（黑五）/ 丰收滋味（感恩）/ 驰骋朝夕（跑跑轮 · 日月交替）
- **行军特效（2025 省省卡配套）**: 狂焰之旅（火焰脚印 + 烧焦大地）

### G. 装饰物 · 秋日集市摊位（2025 三级）

"感恩集市"三级装饰：简单单摊 → 中等单摊+地货 → 豪华主摊+两侧小摊，货物选南瓜 / 麦穗 / 松果 / 火鸡 / 香蕉（摊顶图案），感恩节常用食物多样化。

## 七、美需描述范式

**结构**: `[核心主体] + [场景/结构要素] + [底座/支撑判定] + [动态效果] + [配色] + [≥1 方向备选] + [避坑提示]`

**示范 · 主城皮肤 · 多方向**（2024 "收获时刻"三选一）:
```
主城方案1：蔓越莓果酱工厂
- 主体是制作蔓越莓果酱的工厂，拥有工业流水线特征
- 外观：巨大的蔓越莓桶/漏斗、处理罐子、运输管道、包装流水线
- 升级版：细节复杂度 + 罐子涂装 + 流水线动态

主城方案2：烤鸡店
- 主体以烤炉为原型，炉膛 + 柴火 + 烟囱
- 屋顶巨大火鸡（烤制中）、橱窗 + 发光简笔画火鸡 logo
- 周围堆积蔬果
- 升级版：橱窗外扩顶棚 + 南瓜派/热汤小桌 + 火鸡 logo 拟人化 + 烟囱冒烟/柴火燃烧

主城方案3：树屋+火鸡报时钟
- 双层树屋，上层是火鸡报时钟（表盘 + 活动门板）
- 树叶半黄半绿（黄色多），结满果实，落果堆粮
- 每隔一段时间火鸡机械装置探头
- 升级版：门完全打开火鸡弹射报时（参考图3）
```

**示范 · 装饰物升级系列**（2024 "秋日小憩"三级）:
```
LV1：葡萄架秋千，架体简单，藤蔓叶子半黄半绿，葡萄数量少、有几串青色，木板秋千
LV2：藤蔓覆盖面积增加，全部变黄，葡萄全部紫色、数量多，单人座椅秋千，地上有葡萄筐
```

**示范 · 机甲染色双方向**（2025 "金刚 · 秋狩者"）:
```
方案一（枫叶红）：橙红+黄+白拼接，橙红主色调，白色亮色点缀。
  - 腹部/四肢大色块橙红，肩甲/腿甲/手腕/头部管道亮黄+白
  - 白色印花替换成麦穗/枫叶
  - 需区别于落日橙染色

方案二（火鸡配色）：棕+亮橙+白，参考火鸡常用色。
  - 腹部/四肢大色块棕，肩甲/腿甲等亮橙+白
  - 需区别于本体黑 + 落日橙
```

**示范 · 行军特效**（2025 "枫中漫步"）:
```
底部流光：枫叶红主色调，前金后红渐变（参考龙魂守护者配色）
  整体金红色，红偏枫叶红
普通等级加金色粒子闪光效果
拖尾装饰：亮橙色枫叶隐现，可加松果/枫叶/玉米/栗子等果实类

  特效颜色与拖尾一致
```

**必备要素**:
1. 特征陈列 ≥2 点（编号 / 分号）
2. 底座/支撑物处理说明
3. 动态/特效描述（LV5-6 必填）
4. 多方向备选 + 标注运营倾向（如 2025"推荐该方案"）
5. 避坑提示：明确不要真肉 / 不要香蕉 / 区别于同池染色
