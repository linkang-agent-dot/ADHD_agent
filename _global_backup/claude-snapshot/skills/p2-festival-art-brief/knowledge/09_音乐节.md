## 二、节日定位

- **核心概念**: 以"音乐 / 声音 / 律动"为主轴，释放年轻、自由、狂欢情绪。脱离太空/马戏等具体场景限制，把舞台 × 乐器 × 光效作为通用包装语言。
- **情绪关键词**: 热烈 · 律动 · 炫酷 · 怀旧 · 青春 · 释放 · 节拍
- **节日身份**: 新档期建立节（2025 首年 · 2026 第二年）。尚未进入"循环"模板（如春节行军特效循环、科技节主城套装循环、情人节机甲循环）。
- **与其它节日的锚定关系**: 
  - 与春节 / 情人节 / 科技节并列的大节周期里，音乐节是**新增的"氛围节"**，主打演出感和视听冲击；
  - 与沙滩节 / 拓荒节相比，音乐节更"都市"、更"舞台化"，不走自然景观路线；
  - 不要复用登月节的"天文/太空"元素（2025 已切换）。

## 三、视觉调性

| 维度 | 特征 |

| 风格关键词 | 舞台光效 / 电音律动 / 霓虹聚光 / 音波可视化 / 乐器象形 |
| 表现手法 | 音符飘散 · 音量柱 · 音浪声波 · 旋转唱片 · 射灯顶光 · 五线谱缠绕 · 彩带抛洒 |
| 避坑 | ❌ 不做天文/太空/月球（让位给 7 月登月节）· ❌ 不做马戏杂耍（已退役）· ❌ 避免过于 KTV 土味霓虹灯 · ❌ 不与科技节电音舞台混淆（科技节是冷科技感，音乐节是热舞台感）|

## 四、模块拓扑（2025 年基础，2026 延续+增补）

### A. 主城皮肤（普通 + 超凡）

| 等级 | 功能定位 | 美需侧重 |
|------|---------|----------|
| 普通 | 基础主城皮肤 | 可走"乐器改造建筑""留声机""电音舞台"三方向，运营偏向留声机 |
| 超凡 | 红色英雄皮肤 | 名称/描述强化动感节拍（"跃动节拍"系列）|

2026 年新增"**红色英雄皮肤 + 高级主城皮肤 + 低级主城皮肤**"三档，形成类似科技节套装的等级分层（待补细节）。

### B. 主城外挂特效 / 主城特效

- 电音向：环绕主城的音量柱 / 音浪 / 声波 / 音符，动态弹跳后炸开、飘落音符彩带
- 舞台向：旋转五线谱 + 底部星星旋转 + 爆星星 + 顶光洒下
- 入口：累充活动奖励（对标科技节的累充特效）

### C. 行军特效（高级 / 低级）

音乐节的核心产出模块之一。

- **高级行军特效（BP 榜投放）**: 复古暖色（金色）主色 + 前端旋转黑胶唱片 + 唱片飘出音符/旋律线 + 可选羽毛飘动（2025"随乐出征"）
- **低级行军特效**: 2026 新增档位，待定方向

### D. 动态表情

2026 新增方向：**耍酷系**。样本——猴子丢手雷 → 用枪击暴手雷 → 压低帽子低头耍帅。可延伸：搓碟甩头、吉他劈叉、麦霸怒吼、DJ 手势。

### E. 动态头像框

2026 主打"摇滚乐队"方向：
- 底部两把电吉他
- 框边由锁链和尖刺包围
- 动态：吉他震动弹奏 → 音符向顶部进度条填充 → 进度条满后"ROCK"字样 + 火焰燃烧

2025 有"电音狂潮"动态头像框（开箱活动第一名奖励），主题为电子乐。

### F. 静态头像框 × 2

2025 "悦动音符"（小额付费向，简单设计）。2026 预计保持 2 个静态头像框。

### G. 铭牌 · 聊天框

- 2025 "我为乐狂"：左上角耳机 + 右下角音箱 + 音符
- 2026 延续，需给出新主题（乐队/酒吧等）

### H. 挖矿装饰物（节日装饰物 2 · 挖矿小游戏）

等级 1-3 三级进化，2025 方向：
- **方案一 · DJ 工作台**: 箱子→简陋 DJ 台→未来电音 DJ 台+射灯
- **方案二 · 音乐雕塑**（运营偏向）: 单个音符雕塑→缠音符的五线谱→改雕塑结构+乐器点缀+环绕音量装饰

### I. 节日大富翁装饰物 1（2026 新增）

2026 起新增节日大富翁挂件一套，延续音乐主题。可走"乐器系列"：鼓 / 吉他 / 钢琴 / 提琴三级进化。

## 七、美需描述范式（从真实 2025-2026 抽提）

**结构**: `[核心主体] + [形态细节] + [底座/支撑] + [动态效果] + [配色] + [≥1 方向备选] + [避坑提示]`

**示范 · 普通主城皮肤（三方向备选）**:
```
方案一：乐器改造
1. 钢琴改造而成的音乐厅；
2. 建筑顶部是音乐厅的内部环境，有乐谱架、古典乐器、椅子；
升级版：1. 增加钢琴外壳华丽度；2. 考虑普通版顶上不做音乐厅，升级版再做。

方案二：留声机（运营偏向）
1. 留声机外形的音乐厅，底座改造成建筑，有门可以出入；
2. 唱片缓缓转动，上面装饰伴随旋转的音符；
升级版：1. 唱片装饰音符改为纸片猴；2. 增加喇叭播放动效/飘出音符；
        3. 或把喇叭做成拟花形态，周围缠绕花装饰。

方案三：电音舞台
1. 偏现代电音风格的舞台，多层结构，最顶上是舞台；
2. 舞台上放置乐器、音箱、射灯，巨大麦克风/吉他招牌装饰；
升级版：换搓碟猴 + 全息影像装饰。
```

**示范 · 动态头像框（2026 ROCK 框 · 动态编排）**:
```
主题：底部两把电吉他 + 框边由锁链和尖刺包围

动态：
1. 两把吉他震动弹奏音符；
2. 音符向顶部进度条填充；
3. 进度条满后，上面出现"ROCK"字样，冒出火焰燃烧特效。
```

**示范 · 行军特效（2025 随乐出征）**:
```
1. 主色调：复古、怀旧的暖色（比如金色）；
2. 最前端是旋转的黑胶唱片，随唱片旋转从唱片上飘出音符、旋律线；
3. （可选）添加一些飘动的羽毛。
```

**示范 · 主城特效（双方向）**:
```
方案一：偏电音
1. 环绕主城的电音元素（音量柱、音浪、声波、音符）；
2. 动态：弹跳几次（越跳越高）后炸开，飘落音符和彩带。

方案二：偏舞台/传统音乐
1. 环绕主城的旋转五线谱 + 主城底下旋转星星；
2. 旋转过后爆出星星；
3. （可选）顶上洒下变幻顶光。
```

**必备要素**:
1. 特征陈列 ≥2 点（编号或分号）
2. 底座/台子/支撑说明（装饰物类必填）
3. 动态/特效描述（特效/头像框/行军/主城 LV3+ 必填）
4. 多方向备选 + 标注"运营倾向"（音乐节特别常见双/三方向）
5. 避坑提示（与历史登月节 / 马戏节 / 科技节电音区分）
