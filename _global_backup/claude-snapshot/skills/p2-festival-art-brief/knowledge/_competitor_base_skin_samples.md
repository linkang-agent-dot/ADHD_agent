# 竞品主城皮肤 · 动画描述文风参考（34 条）

> 来源：美术从某 SLG 竞品游戏拔出的 34 条"独特基地皮肤"动画美需描述。
> 用途：P2 节日主城皮肤（科技节/感恩节等）美需写作时对齐的文风基准。
> 归档时间：2026-04-22

---

## 一、文风总结（5 条核心特征）

1. **单段落长句叙事**，不分 bullet、不写"① ② ③"、不列镜头编号。用 `随即 / 随后 / 突然 / 与此同时 / 最终` 把动作串起来。
2. **5 段结构式**固定：设定 → 初始静态 → 触发/变形 → 高潮爆点 → 收尾氛围。每条都能套这个骨架。
3. **粒子词汇细腻**：不写"有光效"，写"拖曳着炙热粒子和余烬 / 噼啪作响的蓝色电能 / 辐射橙色光的轨迹"。
4. **暖冷色调切换**明确写出：`整个环境回归原始的火热色调` / `从暖到冷再回暖` / `金色光芒转为空灵蓝色`。
5. **收尾点评氛围**：每条结尾一句概括整体风格，如"电影级动画""高辛烷值""充满趣味性""神话力量感"。

      {主体}开始{变形过程:展开/蜕变/组装/膨胀}，
      {伴随特效:金色光芒/蓝色电能/橙色余烬}从{部位}{动作:迸发/涌出/螺旋上升}。

### 主城 1 — 橙色机器人（萌系/机械）
> 在一个充满活力的等距游戏环境中，一个可爱的橙色机器人站在绿色圆形停机坪的中央，其数字蓝色面孔最初呈现出淡定平静的表情。机器人的金属身体在准备行动时闪闪发光，周围环绕着各种基地建筑，如黄色直升机和挂着南非国旗的建筑。突然，机器人向前弯腰，从腰部下方的舱口处，一架迷你绿色双螺旋桨飞机被部署到停机坪上。随后切换到纯粹的欢欣鼓舞，机器人重新挺直身体，面部显示屏变换成睁大双眼、张嘴大笑的喜悦表情。它高举机械臂，做出胜利庆祝的姿势，与此同时小飞机腾空而起，急剧向右倾斜飞走，留下一道若隐若现的气流尾迹。这一序列以流畅、充满个性的动画和明亮卡通的视觉风格，展现了一段从期待到成功的生动叙事。

### 主城 2 — 天选之人黄金宝座（建筑/传奇揭幕）
> 在明亮黄色的独特基地皮肤界面内，一把宏伟的黄金宝座配备内置大炮和飘扬的红色地毯，作为视觉焦点呈现，最初呈静态并在其脚下显示天选之人文字，画面随即爆发为动态，宝座扶手炮台上两个刺眼的白色光球点亮，投射出强烈光芒，闪光之后随即出现一道燃烧着橙色火焰的天选之人横幅，沿弧形路径流畅划过画面上方，拖曳着炙热粒子和余烬。整个序列巧妙地传达了一种尊贵解锁和传奇地位的感觉，运用快速闪光和流动光束使静态资产栩栩如生，整体氛围胜利高昂，充满电影感的游戏道具揭示风格。

> 一个名为炸弹的大型圆形机械炸弹角色坐落于充满活力的黄色游戏菜单中心，其粗壮的导火线开始被小火焰点燃。机器人的眼睛突然暴凸，巨大的满齿嘴巴大开，红色感叹号凭空弹出，发出突如其来的警报信号。一条流线型机械臂迅速从侧面伸出，抓起一颗更小的燃烧炸弹并直接扔进张开的大嘴里。大型机器人随即经历激烈的变形：机械眼睛发出险恶的红色光芒，身体因内部压力而膨胀，暗色烟雾从底部渗出。张力在一道目眩神迷的黄色光环笼罩角色之后达到顶峰，随即引爆为吞噬屏幕的巨大翻腾火球。烟雾散尽，中央机器人已然消失，留下空旷的金属底座，四颗导火线噼啪作响的小炸弹围绕四周。序列以最后一场混乱行动作结——两颗迷你炸弹同时引爆，将浓厚的棕色烟雾和火花送入空中。

### 主城 33 — KAHU KANAKA 花生凤凰（神话/火焰变形）
> 以黑暗游戏菜单背景为衬托，一个充满活力的黄色弹出窗口KAHU KANAKA占据屏幕，中心是黑暗金属底座上一个花生形状的物体被熊熊燃烧的橙色和红色火焰吞噬，收到Sarah的礼物！字样短暂闪烁，随着火焰与发光余烬和热气波纹一同增强。一只由纯粹火焰组成的雄壮凤凰从火焰中爆发腾飞，这只炙热之鸟从左侧优雅地展翅滑翔，在明亮的黄色背景上留下光的轨迹，地面上凤凰的清晰影子紧随其空中机动。凤凰飞升到中心，有力地展翅，同时发光的火焰符号PPP在其头顶以光的爆发形式出现。序列以凤凰翱翔的最终动态展示达到高潮，随即溶解为耀眼的火花与热气涟漪，燃烧的花生在底座上持续燃烧，动画循环经历其壮丽的变形。

### 主城 34 — Y ELEVEN CASTLE（魔幻/钟表激活）
> 在充满活力的游戏风格动画中，一座以Y ELEVEN CASTLE为名的奇妙多塔城堡，拥有紫色屋顶和发光的绿色窗户，坐落于一片郁郁葱葱的粉色植被之上。序列以右上方悬浮的发光蓝色光球开始，它迅速扩展并转化为一个巨大的空灵钟盘，古罗马数字在城堡尖顶后方逐渐显现。随着钟声滴答，一个巨大的半透明淡蓝色V符号从前景涌现，以魔幻跳动的光辉覆盖城堡。整个场景中，闪烁的蓝色与紫色光迹在建筑周围舞动，而娇嫩的粉色花瓣懒洋洋地随风飘落，增强了迷人的氛围。固定镜头捕捉着这些神秘元素的节律出现与发光强度，在温暖的金黄色背景下营造出一种强大激活的感觉。

## 五、写 P2 主城皮肤美需时的应用要点

1. **先确定题材分类**（战士 / 机械 / 神话生物 / 萌系 / 建筑 / 魔幻），套对应同类样本。
2. **5 段结构必须齐**，缺哪段补哪段：设定 / 初始静态 / 触发 / 高潮 / 收尾氛围。
3. **粒子与色调切换不能省略**——竞品的质感主要靠这两个东西砸出来。
4. **结尾必须有风格点评句**（电影感 / 高辛烷值 / 传奇 / 神话力量 / 欢乐生机），不写等于残缺。
5. **等距视角 / 固定镜头 / 相机保持静止**这类镜头语言，P2 主城皮肤默认也是等距视角，可以大胆用。
