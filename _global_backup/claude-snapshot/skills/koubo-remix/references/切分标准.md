# 口播视频三段式切分标准（v1.2 定稿·根痛平案沉淀）

> 校准历程：4 整片初稿 → 41 条人工旧切片点评立铁律 → 用户拍板 5 问 → 正确案例库(11A/6B/11C)+4 条过审成品两轮重定义。
> 新产品复用时：三段功能定义通用；分界判据和段长线用该产品的人工正确案例重新校准。

## 铁律

**切点只能落在完整句子边界；句内短语间隙一律不合格。**
判据：切出的每一段，首句能独立开口、尾句说完收口。
（根因：切在句内，同源拼回听不出问题，跨源混拼立刻穿帮——"每粒都浓缩了｜白芍葛根桃仁"事故。）

## 三段定义与分界判据

| 段 | 装什么 | 分界判据 | 段长(根痛平实测) |
|---|---|---|---|
| **A 钩子** | 只装钩子**表演形式**：排比句式("吃它!"×5)、悬念问答("为什么男人不能随便吃")、小剧场、对比/实验演示。单钩或已复合好的开场都合法，粒度混合一个库，不拆子库 | **A→B：钩子表演形式一结束就是 B**——转入"对你说话的连续说服"("你是颈椎经络淤堵…"/"如果你颈椎疼痛…"/"…是不可能自己好的"/"你就记住…")即 B 头。⚠️ 症状共鸣/痛点教育归 B 不归 A | 4~35s，无固定目标 |
| **B 口播** | 完整说服主体：痛点共鸣/教育→产品亮相→机理成分→认证→用法。**B 尾禁 CTA**（防与 C 双重促单） | | 45~85s |
| **C 促单** | 成交话术：厂家/优惠/包邮/点击链接 | **B→C：首句进入成交语境**。三种合法开场：①直接促单 ②人群召唤("…脑子不清醒的朋友，可以去试试") ③异议处理("有人说太贵，价格打下来了")。成交框架内可重申卖点；纯功效开场不合格 | 6~19s |

- 模糊句归前段，保 C 纯净（C 最短，要能接任何 B）。
- A 内点到产品名/认证 = 悬念式钩子，合法。
- ASR 品牌名错拼/繁体不影响判断，只看语义角色。
- A 段台账记录钩子单元数；带桥接句("来我给大家展示一下")的 A 标 bridge=true。

## 边界判定流程

1. 语义定位转折句（读整片转写）。
2. 句边界回退：切点放在转折句之前最近的完整句结束处，绝不句内下刀。
3. **窗口重转写**：切点 ±6s 单独重跑 ASR（vad_filter=False）——整片转写会整段漏人声（实证漏 19 秒）。
4. **空窗抽帧**：整片转写 ≥3s 无人声区间，先抽帧看画面再定归属（可能是字幕演绎段/产品特写/漏听人声，
   三种都实际出现过）；无人声演示归随语境；段尾无人声 >5s 强制复核。
5. 声学落点分级：句边界真实停顿 ≥0.30s=HIGH 自动切 / 0.12~0.30=MID / <0.12=LOW 人耳定。
   停顿必须用梯级静音检测实测（-25/-30/-35dB 级联）——whisper 词时间戳首尾相连，不能当停顿用。
6. 镜头切换点辅助：切点同时落在镜头切换处最佳（如产品特写起点）。

## 切割物理标准

- 重编码（非 -c copy）：任意帧可切、段首即关键帧。
- 停顿中点 → round(t×30) 帧号；A=[0,c1) B=[c1,c2) C=[c2,end)；用 -frames:v 精确帧数保零重叠。
- 统一 30fps / 1080×1920 / H.264 CRF18 / AAC 192k 48kHz（源片常混 25/30/60fps，统一后才能任意拼）。
- 段头尾 30ms 音频淡入淡出。

## 验收五项

1️⃣ 句完整 2️⃣ 无跨段残留（A尾无说服正文/B头无裸列表/B尾无CTA/C头即成交语境） 3️⃣ 语义正确
（防错分类进错库、防截字反转语义——实例："还没试过"截成"有试过"） 4️⃣ 画面不突兀（优先口型闭合处）
5️⃣ 段长合格。**终极验收 = 跨源混拼试听：任取异源 A+B+C，边界处听不出半句冒头。**

## 归档

`切片库/A|B|C/<源片名>__<段>__f<起>-f<止>.mp4` + `ledger.json`
台账字段：segment/source/start/end/frames/file/hook_units/cut_confidence/review/used_in。
防重与组合记录全靠台账，不靠文件名排序。
