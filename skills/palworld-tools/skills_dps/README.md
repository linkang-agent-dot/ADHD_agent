# 帕鲁 技能DPS + 习得等级 数据

给全图鉴HTML的「技能DPS弹窗 / 输出指数排序 / 技能图鉴页签」供数据。

## 文件
- `paldex_skills.js` — 内联进全图鉴：`SKILLS={slug:[中文名,属性,威力,冷却,DPS]}` + `PALSK={帕鲁名:[slug按习得顺序]}`
- `skills_by_slug.json` — 全323技能源(含威力/冷却/属性)
- `pal_skills.json` — 298帕鲁的技能slug列表
- `_fetch_palskills.py` — 爬帕鲁页技能列表(断点续)

## 数据源(paldb.cn 浏览器UA urllib)
1. **技能威力/冷却**：`/active-skills` 页，每个 `<a href="/skills/slug">` 卡片里 `威力: N` `冷却: Ns` `X属性`。DPS=威力÷冷却。
2. **每帕鲁技能**：`/pals/X` 页的 `href="/skills/slug"` 列表。

## 🔑 关键结论(已验证)
- **paldb帕鲁页技能顺序 = 游戏内习得顺序**(棉悠悠对game8 8/8全中)。
- 升级习得固定节点 `[1,7,15,22,30,40,50,70]`(Lv70=1.0新增)，第i个技能→第i个节点；**第9个及以后=技能果实/配种**(无等级)。
- 同时只能装3个主动技能。DPS与等级无关。

## ⚠️ 坑
- 技能属性是"地"非"土"(帕鲁表归一化成土，显示要转)。
- 威力/冷却正则：`威力[:：][\s|]*(\d+)`(冒号后是空格再管道)。
- 缺16个slug(双牙/猛牙/史莱姆重压变体)需单独爬`/skills/slug`补。
