## 2. ROI 价值锚框架（换皮核心尺子）

### 公式
```
售价钻石价值 = 售价(USD) × 500          （$49.99→24995；$99.99→49995）
礼包总钻石价值 = Σ(每个道具钻石价值 × 数量)
ROI = 礼包总钻石价值 ÷ 售价钻石价值
```

### 钻石 ↔ 美刀换算（官方兑换比例 1 USD = 500 钻石）
**确认口径：1 美元 = 500 钻石**（用户确认的官方兑换比例）。`钻石充值` 页签（Pack 1001-1006）佐证：$4.99→2500、$14.99→7500、$29.99→15000、$59.99→30000、$99.99→50000 均为 1:500（$0.99→500 算出 ~505 是小额档取整，首充双倍是额外加成）。
```
美刀价值     = 钻石价值 ÷ 500              （1 钻石 = $0.002）
礼包总美刀价值 = 礼包总钻石价值 ÷ 500
美刀ROI      = 礼包总美刀价值 ÷ 售价(USD)
```
- **美刀ROI ≡ 钻石ROI（同值）**：美刀ROI = (礼包总钻石价值÷500) ÷ 售价 = 礼包总钻石价值 ÷ (售价×500) = 钻石ROI。分子分母同时 ÷500 抵消，比值不受单位影响。两个口径并列只为直观——钻石口径看"装了多少钻"，美刀口径看"装了多少钱"，ROI 数字一致。
- 例：水手招募 $49.99 档 → 礼包总钻石价值 ~451666 → 礼包总美刀价值 ~$903 → 美刀ROI = 903÷49.99 ≈ **18.1x**（与钻石ROI 18.1x 相同）。

### 换皮铁律：ROI 不变
换皮 = 换道具/皮肤/文案，**不改礼包的性价比定位**。替换道具后到 `礼包总览`（或用录入模版）重算 ROI：
- ✅ 落在模板同档区间 → 价值守恒，OK
- ❌ 偏离 → 换错了等值物，回去调数量或换等值道具

> 观测样本：水手招募系列 $49.99 / $99.99 档 ROI 普遍 **17~19x**（同价位同玩法 ROI 高度一致）。各礼包类型 / 价位的标准 ROI 带可从 `礼包总览` 按 `礼包类型`+`售价` 透视得出，换皮时以**被换模板自己的 ROI** 为锚最稳（不必跨类型比）。

## 4. 换皮数值微调规则

承接 `reskin-numerical-framework.md`（P2/X2 框架）的判断标准，X3 适配：

| 维度 | 规则 |
|------|------|
| **变现侧（锁死）** | 礼包 ROI、随机礼包期望值、掉落权重 —— 无数据支撑不动。换皮保持 ROI 同档 |
| **流量侧（可调）** | 投放节奏、累充档位门槛、限购次数、活动时长 —— 按运营节奏可调 |
| **占位行** | Reward 模板常藏 `MinNum=0` 装饰性占位（52003 信物 / 11002 加速等），换皮后**扫全部 MinNum=0 行**清掉，否则面板显 ×0（见 `table-reference.md §4.2`） |
| **等值替换** | 换道具优先选 `道具表` 已估值的等值物；重算 ROI 守在模板区间 |
| **随机礼包** | floor/锚点变动不自动联动期望值；无数据不调 drop 权重 |
