## 坑 / 注意事项

- `111110008`（永久资源增产特权）是**通用道具**，6001/6002 rewards 里可直接保留，无需换
- `111110104`（2026科技节头像框-集卡册-静态）和 `111110105`（2026科技节自选宝箱）是**科技节专属**，分配新 ID 时跳过这段，且换皮时必须替换（复活节对应：`111111014` 头像框、`111110264` 自选宝箱）
- 保底卡包和主题卡包 icon 都固定用 `403303`
- display_key 需在已有节日值之后分配：复活节 6002 已用到 `151105050`，6001 用了 `151105051`，主题卡包从 `151105052` 开始（复活节用到了 151105055）
- **自选卡包的 select_box 里 book_id 出现两次**（4★一次、5★一次），两处都要改，容易漏一个
- **ID 段附近可能被其他道具占用**：复活节 111110310–111110320 已全被城市皮肤/行军表情占用，查找缺失道具时不能只看连续段
- 低星组（欢乐寻蛋1-2★、彩绘时光1-3★、春日花园1-3★）**不需要专属主题卡包**，玩家通过全册保底卡包自然获取这些卡
- 6002 的 rewards 字段含 `11112936` 可粗判❌5★，但此规律不完全可靠，**必须用 6003 col[2] A_INT_star 最终确认**
- **⚠️ 6004 集卡商店每次换节日必须更新**：rows 60040026–60040030 的 `goods_id` 指向当前节日的保底卡包 ID，下一个节日换皮时必须同步修改
