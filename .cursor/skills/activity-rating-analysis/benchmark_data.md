## 各等级活动特征

### S级活动共性

- 付费率 ≥15%
- 超R收入占比在健康区间（50-70%）
- 分层清晰度在理想区间（15-25倍）
- 投放内容：往期特效、多档位设计

### A级活动共性

- 付费率 10-15%
- 各维度无明显短板
- 可能在单一维度有突出表现（如异族大富翁变现力极强）

### C级及以下问题模式

| 活动 | 核心问题 |
|-----|---------|
| MJ8 | 变现力不足 + 过度依赖超R |
| 行军表情 | 品类天花板低，小R几乎不买 |
| 挖矿 | 分层完全失效，超R付费率低于大R |

## 投放内容与评级关系

| 投放类型 | 典型等级 | 说明 |
|---------|---------|-----|
| 往期特效/套装 | S-A | 各层玩家都有需求 |
| 新皮肤 | A | 吸引力中等偏上 |
| 独立玩法+皮肤 | A-B | 取决于玩法质量 |
| 行军特效 | B-C | 超R专属感强，小R参与度低 |
| 养成线 | C-D | 天然面向中小R，超R兴趣低 |
| 表情 | D | 品类天花板明显 |

## 对标使用方式

新活动数据分析时，按以下维度对标：

1. **投放内容类型** → 找同类型活动作为基准
2. **各维度得分** → 判断相对强弱
3. **综合评级** → 确定活动定位
4. **问题诊断** → 参考同类活动的成功/失败模式
