# 评分规则详细说明

## 一、变现力评分（权重30%）

基于付费ARPU评分：

| ARPU区间 | 得分 | 解读 |
|---------|-----|-----|
| ≥15 | 100 | 极强变现 |
| 10-15 | 80 | 强变现 |
| 5-10 | 60 | 中等 |
| 2-5 | 40 | 偏弱 |
| <2 | 20 | 弱 |

## 二、转化力评分（权重25%）

基于整体付费率评分：

| 付费率区间 | 得分 | 解读 |
|-----------|-----|-----|
| ≥15% | 100 | 极高转化 |
| 10-15% | 80 | 高转化 |
| 7-10% | 60 | 中等 |
| 4-7% | 40 | 偏低 |
| <4% | 20 | 低 |

## 三、鲸鱼依赖度评分（权重25%）

基于超R收入占比评分（非线性，50-70%为最健康区间）：

| 超R收入占比 | 得分 | 解读 |
|------------|-----|-----|
| 50-70% | 100 | 最健康：超R主导但不过度依赖 |
| 70-80% | 80 | 略高但可接受 |
| 40-50% | 60 | 偏低，超R贡献不足 |
| 80-90% | 40 | 过度依赖超R |
| <40%或>90% | 20 | 极端，需重点关注 |

### 解读说明

- **过高（>80%）**：收入过度集中在超R，风险大，超R流失=收入崩盘
- **过低（<40%）**：超R贡献不足，可能是活动对高付费玩家吸引力不够
- **理想区间（50-70%）**：超R是主力但有中大R补充，结构稳健

## 四、付费分层清晰度评分（权重20%）

基于"超R付费率÷小R付费率"评分：

| 清晰度倍数 | 得分 | 解读 |
|-----------|-----|-----|
| 15-25倍 | 100 | 完美分层：各R级都有匹配的付费点 |
| 10-15倍 或 25-35倍 | 80 | 分层清晰 |
| 5-10倍 或 35-50倍 | 60 | 分层一般 |
| 50-100倍 | 40 | 过度分化：小R几乎不买 |
| <5倍 或 >100倍 | 20 | 分层失效 |

### 解读说明

- **清晰度过低（<5倍）**：超R不比小R更想买，活动对高付费玩家没有额外吸引力
- **清晰度过高（>100倍）**：小R完全不买，变成"超R专属品"，用户基础薄弱
- **理想区间（15-25倍）**：各层玩家都有参与，但付费能力强的玩家付费意愿明显更高

## 五、综合得分计算

```
综合得分 = 变现力得分×0.30 + 转化力得分×0.25 + 鲸鱼依赖度得分×0.25 + 分层清晰度得分×0.20
```
