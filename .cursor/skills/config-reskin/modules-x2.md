# X2 配置表模块知识库

> **项目**：X2（别名后缀 `_x2_` 或无后缀，走注册表自动解析）
> P2 项目见 [modules-p2.md](./modules-p2.md)
>
> 本文件定义 X2 所有配置表的**大定义**和**子定义**。
> P2 和 X2 的大定义相同，但**列号、列名、子定义有差异**。
>
> ⚠️ X2 所有表的 col[0] 都是 `p2_title`（元数据列，不导出），导致**所有字段列号比 P2 多 1**。

---

## 使用方法

1. **确认项目** → 见 SKILL.md S-1（X2 用本文件，P2 用 modules-p2.md）
2. **接到任务** → 理解任务本质
3. **逐表过触发条件** → 命中即拉进
4. **读子定义** → 确认具体改什么字段
5. **都不命中** → 可能需新模块

## 2112 — 活动主体表 (activity_config)

### 大定义

定义活动的顶层入口：时间窗口、banner、入口图标、内容引用。**所有活动配置的追踪起点。**

**何时需要**：只要有「活动入口」，就需要这张表。

### 重点字段（X2 — 27 列）

| col | 字段名 | 角色 | 换皮处理 | P2 差异 |
|-----|--------|------|---------|---------|
| [0] | `p2_title` | 元数据 | — | P2 无此列 |
| [1] | `A_INT_id` | 活动行 ID | **换** | P2=[0] |
| [3] | `A_STR_constant` | 活动组件类型 | **不换** | P2=[2] |
| [6] | `A_INT_base_activity_id` | 父活动 ID | **换** | P2=[5] |
| [7] | `A_MAP_filter` | 前置条件 | **换**（如新建筑 ID） | P2=[6] |
| [8] | `A_MAP_text` | 活动标题/描述 LC key | **换** | P2=[7] |
| [9] | `A_ARR_activity_components` | 内容引用 task/package | **换** | P2=[8] |
| [10] | `A_MAP_description` | 规则 LC key | 同类**可不换** | P2=[9] |
| [11] | `A_INT_ui_template` | 2119 UI 关联 | 同类**可不换** | P2=[10] |
| [14] | `S_STR_banner_url` | Banner 图片 | **换** | P2=[13] |
| [18] | `A_INT_show_hud` | HUD 入口图标 ID | **换** | P2=[17] |
| [23] | `C_INT_title_icon` | 标题图标 | 按需换 | **P2 无此列** |
| [24] | `S_INT_merge_save` | 合并存档标记 | **不换** | **P2 无此列** |
| [25] | `S_STR_mini_banner_url` | 小 banner 图 | **换** | P2=[22] |
