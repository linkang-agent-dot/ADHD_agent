# P2 养成线与养成线梳理

本文档服务制作人视角，解决三件事：
1. **收入侧**「哪几条线在扛盘子」
2. **深度侧**「每条线值多少美元、R级梯度怎样」
3. **体验侧**「各线的游戏内实态和中R扩线可行性」

具体的配表消耗明细、材料单价、升级跳升等数值细节见 [`p2-numerical-design/养成线深度手册.md`](../p2-numerical-design/养成线深度手册.md)。

## 何时查阅本页

- 评估**养成线收入结构**与**资源分配优先级**时
- 评审**品质梯度、新红/共享槽、价值口径**时
- 判断**中R扩线方向**时
- 需要养成线深度的制作人级汇总数字时

## 一、养成线全景（收入侧）

与 `P2数据现状档案.md` 第四章一致。战略上简记为 **三大支柱 + 中坚 + 辅助**：

| 养成线（收入口径） | 累计占比 | 战略角色 |
|-------------------|:---:|---------|
| SLG | 22.7% | 基本盘，存量期绝对额承压 |
| 英雄 | 19.8% | 第二支柱，全R覆盖；与赛季/活动强耦合 |
| 机甲 | 17.8% | 第三支柱，跌幅深，需诊断「维持 vs 重构」 |
| 混合 / 收藏品 | 11.5% / 11.4% | 中坚，占比相对稳，防「只做头不做腰」 |
| 装备洗练 / 军备技能 / 战车 | 8.2% / 5.1% / 2.5% | 辅助或探索线 |

## 三、制作人关键洞察

### 3.1 战装是唯一「全R段覆盖」的深度线

- 中R入门 ~$105（打造+Lv.5）→ 超R含重铸 ~$9,500/英雄
- 寄生在英雄系统上，不需额外理解成本
- 红色升级（21→30）占不含重铸总深度的 **62.2%**——超过打造+橙升级之和
- 重铸是超R的持续消耗坑：30次/件 ~$5,528（4件套），追求完美可达 $18,000+

### 3.2 机甲芯片统一 $1/个，经验才是大头

- 所有机甲芯片（犀牛到鳄鱼+通用）$100量级=100，单价 $1
- 经验占单台橙机甲的 **52%**，芯片+突破占 48%
- 超R等级完成率 72.66% 高位不动——增长已停滞

### 3.3 英雄升星是橙色英雄最大卡点

- 橙英雄升星 $980（$100=10,000 星点），占英雄线 **44%**
- 升星材料投放节奏直接影响中大R消费体验
- schema6（红英雄）超R完成率仅 8.4%，付费空间最大

### 3.4 收藏品已不是「低成本线」

- 紫升星 $0.40、橙升星 $1.00、红升星 $1.60（较旧数据翻倍）
- 18件混合展览 $8,070，全红 $20,880
- 保养系统创造了超R的**周期性消费**需求

### 3.5 机师碎片分品质差价大

- 紫碎片 $6.67/个 vs 橙碎片 $10/个
- 机师渗透率极低（超R仅17.5%），连超R都没怎么玩进去

## 四、英雄养成线梳理（配置侧）

面向 **斗士（英雄）** 主养成：**升星、等级、技能** 三线 + **红色专属「共享等级」**。

### 4.1 养成维度定义

| 养成维度 | 消耗在表里的落点 | 制作人关心什么 |
|----------|------------------|----------------|
| **升星** | 星星经验（累计至满星） | 品质越高、单点美元越贵，拉大 R 与中R差距 |
| **等级** | 英雄经验（累计至满级） | 相对升星/技能「不贵」，但不能无限加新等级组 |
| **技能** | 升级道具数量（汇总为「奖牌」计数）× 道具单价 | 碎片/道具型消耗是英雄付费深度的主杠杆 |
| **共享等级** | 仅品质=红（5），四共享槽升级道具 | 红英雄的额外深度层，评估新红/新槽时单独算 |

### 4.2 配置索引

| 配置表 | 用途 |
|-------|------|
| `1920_p2_hero_data`（k6英雄，gid=360787860） | 英雄主表：品质、等级组、星级组、技能配置ID |
| `1921` 斗士等级配置 | 等级经验曲线 |
| `1924_p2_hero_skill` | 技能消耗反查：skill_id → group → upgrade_need 汇总 |
| `1951_p2_hero_share_skills` | 红色共享等级（独立于主表） |

### 4.3 价值口径（与数值策划一致）

- **星星经验→美元**：金（橙）$100 = 10,000 点、紫 $100 = 70,000 点
- **英雄等级经验→美元**：$100 = 12,000,000 点
- **技能/共享等级道具→美元**：以 [`data/道具价值_核心列.csv`](../p2-numerical-design/data/道具价值_核心列.csv) 的 `$100量级` 为准

### 4.4 标准汇总表格式

列顺序固定为：**英雄品质｜养成维度｜经验总数｜星星点数｜奖牌｜价值｜养成总价值**

| 紫 | 升星 | $63 | $954 |

| 紫 | 技能 | $450 | $954 |

## 五、各养成线游戏内实态（截图分析，2026-04-01）

> 以下基于实际游戏截图整理。截图来自特定账号，数值仅反映该账号进度。截图存放于同目录各子文件夹。

### 战装（Battle Equipment）

**养成路径**：制造（图纸+材料）→ 升级(1→20) → 重铸（满级后随机词条）
**槽位**：4件套——防护服、驱动器、探测仪、应急包
**付费卡点**：重铸是最深的付费坑（随机词条+锁定机制），高级芯片在 Lv.8/12/16/20 跳升
**中R体验**：入门门槛低（寄生在英雄系统上），有即时反馈（战斗力数字增长），重铸门槛高（超R层）

### 英雄（Hero）

**品质体系**：紫色(史诗) → 橙色(传说) → 红色(超凡)
**紫/橙基础养成**：等级 + 升星（电池，幸运值概率）+ 技能（奖牌）+ 天赋树（三分支各50点）
**红色独有系统**：
- 7星升星（vs紫/橙5颗，深度+40%）
- 同频技能（核心ABCD四槽，独立升级）
- 同频斗士（绑定另一英雄共享属性——套娃系统）
- 天赋同频（DNA链，钻石消耗）
- 专属技能

**关键理解**：红英雄不是「更强版橙英雄」，而是完整套娃系统。超R完成率仅8.4%——系统刚展开，付费空间巨大。

### 收藏品（Collectibles）

**养成路径**：探测（扭蛋/盲盒）→ 升星（品质升星石）→ 展览（6层×3展位=18件）→ 保养（持续消耗维持属性）
**保养系统**：展览属性非永久生效，分50%/80%/120%三档，到期后属性失效——**订阅制设计**
**付费卡点**：探测是主入口（gacha）、升星是长线坑、保养是超R周期性消耗坑

### 机甲（Mecha）

**获取门槛极高**：单个机甲解锁礼包4,999-9,999钻，直接挡住中小R
**养成路径**：获取 → 升级(1→69+) → 技能 → 修理（耐久度）→ 研究
**中R体验**：极高门槛，大多数机甲对中R显示"0%/未获得"

### 机师（Pilot）

**强依赖机甲系统**：先有机甲才能配机师
**技能觉醒需同名机师**（gacha+合成模型）
**超R渗透仅17.5%**——连超R都没怎么玩进去

### 战车（Battle Vehicle）

**获取门槛**：统一9,999钻解锁，但渗透率72.58%（比机甲宽）
**问题**：不是「进不去」而是「进去了但不花钱」——渗透高但付费转化弱

## 六、中R扩线可行性总结

| 养成线 | 中R入门门槛 | 与现有消费线关联性 | 付费感知直观度 | 深度梯度 | 扩线推荐度 |
|--------|:---:|:---:|:---:|------|:---:|
| **战装** | **低** | **强**（绑定英雄） | **高** | $105→$325→$1,500→$9,500 | ★★★ |
| 收藏品 | 中（免费入门） | 弱（独立系统） | 中 | $24/件→$641/件 | ★★ |
| 机甲 | **极高**(5k-10k钻) | 弱 | 高（但进不去） | — | ★ |
| 机师 | 极高（依赖机甲） | 无 | 低 | — | ✗ |
| 战车 | 高(9,999钻) | 弱 | 中 | — | ★ |

**结论**：战装是中R扩线的最佳第一选择——寄生英雄系统、无额外解锁门槛、养成反馈直观、深度梯度覆盖全R段。
