## 复用决策原则

1. **2124 drop**：掉落内容不含节日专属道具 → 复用；含节日道具（如 BP 道具）→ 新建
2. **2121 new_progress/drop_topay**：进度条数值和逻辑不变 → 复用
3. **2137 retake**：回收的道具是通用掉落货币 → 复用；回收的是节日专属道具 → 新建
4. **2122 rank**：score_rule 里的 2011 ID 列表需要指向当前节日 → 新建；排名奖励(2118)不含节日道具 → 可复用
5. **2135→2011→2013**：礼包内容含节日道具 → 全链路新建；纯通用礼包 → 复用
6. **2116 exchange**：兑换奖励含节日道具 → 新建；通用 → 复用
