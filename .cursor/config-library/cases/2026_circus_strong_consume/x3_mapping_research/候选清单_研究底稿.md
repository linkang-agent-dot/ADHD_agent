## 关键实查结论（防丢）
- 深海大富翁 2802：普抽=1057 航海罗盘 / 精抽=1058 海神罗盘 / 神秘宝藏=1206 深海宝箱 / 进度=1204 珍珠贝。
- 深海转盘 1025：消耗 1200 深海藏宝图，单抽1/十连10。
- 深海券礼包锚 13021-024（$4.99/19.99/49.99/99.99）：藏宝图 20/80/200/400 + 钻石 2500/1w/2.5w/5w + 「100VIP点」道具2022 ×25/100/250/500；行内标注 400%（=4x ROI 平铺全档；钻石面值≈售价 1:1，券是"白送"层）。
- Pack 表第41列 GuildGift=201~205 五档，全表 100~605 个礼包在用 → X2 联盟礼物5档的 X3 原生等价机制（配字段，不发道具）。
- X3 排行奖励惯例（深海转盘本服榜 RankCfg2000→30910-917）：节日纪念卡 180080 ×10/7/5… + 3h建造/招募/研究加速三件套 ——主奖走独占外显，非 X2 的养成四件套。
- X3 加速无 15m 档：1m/5m/30m/1h/3h/8h；填充惯例档=1小时（通用11004=470组、建造11014=319、招募11024=338、研究11034=265）。
- X3 英雄无"喂经验"体系：招募=英雄美酒7001（Type13），升星=信物（万能52001/02/03），技能=技能书19001-03。玩家经验1004 流通0（别用），标准玩家成长填充=1008 冒险阅历（1956组）。
- 机甲经验最佳类比=1142 驯养经验（海妖养成线，453组，商店实售 1000点=30钻 → 0.03钻/点）。
- 券价值口径分裂：道具表口径 券≈500钻/张（1042/1096/1128 均500）；skill 设计锚=$0.25(125钻)/张 —— 4 倍差，评估 ROI 用设计锚，前端价值展示%用道具表口径。
- 资源发放惯例量：金币 10w-50w / 硬木板 5k-5w / 啤酒 5w-6w；资源当填充物按惯例量给，别做价值乘法（表内单价×5w 会得出荒谬面值）。
- 旧愚人节马戏三件（1042 马戏团入场券/1043 狂欢币/1044 马戏团邀请函）仍在役（1042=愚人节转盘1006消耗，48组流通）→ 新节日别复用（复用旧代币有存量/口径风险），新建。
- 1200 段 id 占用至 1206（1203/1205 跳号），新代币建议 1207/1208 起。
